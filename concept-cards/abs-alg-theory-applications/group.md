# Quick Definition

A group $(G, \circ)$ is a set $G$ together with a binary operation that is associative, has an identity element, and where every element has an inverse.

# Core Definition

"A group $(G, \circ)$ is a set $G$ together with a law of composition $(a, b) \mapsto a \circ b$ that satisfies the following axioms:
- The law of composition is associative: $(a \circ b) \circ c = a \circ (b \circ c)$
- There exists an element $e \in G$, called the identity element, such that for any element $a \in G$, $e \circ a = a \circ e = a$
- For each element $a \in G$, there exists an inverse element in $G$, denoted by $a^{-1}$, such that $a \circ a^{-1} = a^{-1} \circ a = e$"
(Judson, pp. 46-47).

# Prerequisites

- **Binary operation** — A group requires a binary operation on $G$
- **Set** — A group is built on a set

# Key Properties

1. **Associativity**: $(a \circ b) \circ c = a \circ (b \circ c)$ for all $a, b, c \in G$
2. **Identity**: There exists $e \in G$ with $ea = ae = a$ for all $a$
3. **Inverses**: For each $a \in G$, there exists $a^{-1}$ with $aa^{-1} = a^{-1}a = e$
4. The identity element is unique (Proposition 3.17)
5. Inverses are unique (Proposition 3.18)
6. $(ab)^{-1} = b^{-1}a^{-1}$ (Proposition 3.19)
7. $(a^{-1})^{-1} = a$ (Proposition 3.20)
8. Cancellation laws hold: $ba = ca \Rightarrow b = c$ and $ab = ac \Rightarrow b = c$ (Proposition 3.22)

# Construction / Recognition

## To Verify $(G, \circ)$ is a Group:
1. Check that $\circ$ is a binary operation on $G$ (closure)
2. Verify associativity: $(a \circ b) \circ c = a \circ (b \circ c)$
3. Find an identity element $e$ with $ea = ae = a$ for all $a$
4. For each $a$, find $a^{-1}$ with $aa^{-1} = a^{-1}a = e$

# Context & Application

Groups are the central object of study in abstract algebra. They arise in number theory ($\mathbb{Z}$, $\mathbb{Z}_n$), geometry (symmetry groups), linear algebra ($GL_n(\mathbb{R})$), physics (particle symmetries), chemistry (molecular symmetries), and cryptography.

# Examples

**Example 1** (p. 47): $(\mathbb{Z}, +)$ is a group. Identity is 0, inverse of $n$ is $-n$.

**Example 2** (p. 47): $(\mathbb{Z}_5, +)$ is a group of order 5. See Cayley table in Figure 3.10.

**Example 3** (p. 48): $GL_2(\mathbb{R})$, the set of invertible $2 \times 2$ matrices under multiplication, is a nonabelian group.

**Example 4** (p. 48): The symmetries of an equilateral triangle form the group $S_3$ (or $D_3$), a nonabelian group of order 6.

**Non-Example** (p. 47): $\mathbb{Z}_n$ under multiplication is NOT a group because 0 has no multiplicative inverse.

# Common Errors

- **Error**: Forgetting to check associativity (assuming it from commutativity)
  **Correction**: Commutativity and associativity are independent; both must be verified separately

- **Error**: Checking only left identity/inverse without right
  **Correction**: The axioms require both $ea = a$ and $ae = a$, and both $aa^{-1} = e$ and $a^{-1}a = e$

# Common Confusions

- **Confusion**: Thinking a group must be commutative
  **Clarification**: Groups need not be commutative; those that are commutative are called abelian

- **Confusion**: Confusing a group with a ring
  **Clarification**: A group has one operation; a ring has two operations (addition and multiplication) where the additive structure forms a group
