# Quick Definition

A group $G$ is finitely generated if there exists a finite set $\{g_1, \ldots, g_n\}$ of elements that generates $G$, meaning every element of $G$ can be written as a product of these generators and their inverses.

# Core Definition

"If there is a finite set $\{g_i : i \in I\}$ that generates $G$, then $G$ is **finitely generated**" (Judson, p. 171). By Proposition 13.3, $h \in H$ (the generated subgroup) exactly when $h = g_{i_1}^{\alpha_1} \cdots g_{i_n}^{\alpha_n}$.

# Prerequisites

- **Group** — The structure being generated
- **Subgroup** — The generated subgroup is the smallest subgroup containing the generators

# Key Properties

1. All finite groups are finitely generated
2. $\mathbb{Z} \times \mathbb{Z}_n$ is finitely generated by $\{(1,0), (0,1)\}$
3. $\mathbb{Q}$ under addition is NOT finitely generated
4. In an abelian group, each generator need occur only once in a product

# Examples

**Example 1** (p. 171): $S_3$ is generated by $(12)$ and $(123)$.

**Example 2** (p. 172): $\mathbb{Q}$ under addition is not finitely generated: for any finite set of generators $p_1/q_1, \ldots, p_n/q_n$, a prime $p$ not dividing any $q_i$ means $1/p$ cannot be expressed.
