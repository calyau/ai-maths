# Quick Definition

A cyclic code is a linear code where any cyclic shift of a codeword is also a codeword. Cyclic codes correspond exactly to ideals in the ring $R_n = \mathbb{Z}_2[x]/\langle x^n - 1 \rangle$.

# Core Definition

Let $\phi: \mathbb{Z}_2^k \to \mathbb{Z}_2^n$ be a binary $(n,k)$-block code. Then $\phi$ is a **cyclic code** if for every codeword $(a_1, a_2, \ldots, a_n)$, the cyclically shifted $n$-tuple $(a_n, a_1, a_2, \ldots, a_{n-1})$ is also a codeword (Judson, p. 296).

By Theorem 22.16, a linear code $C$ in $\mathbb{Z}_2^n$ is cyclic if and only if it is an ideal in $R_n = \mathbb{Z}_2[x]/\langle x^n - 1 \rangle$.

# Prerequisites

- **Polynomial code** — Cyclic codes are a special type of polynomial code

# Key Properties

1. Cyclic codes are ideals in $R_n = \mathbb{Z}_2[x]/\langle x^n - 1 \rangle$ (Theorem 22.16)
2. Every ideal in $R_n$ is principal: $C = \langle g(t) \rangle$ where $g(x) \mid (x^n - 1)$
3. The cyclic shift of $f(t)$ corresponds to multiplication by $t$ in $R_n$
4. Cyclic codes are easy to implement using shift registers
5. The number of cyclic codes of length $n$ equals the number of divisors of $x^n - 1$ in $\mathbb{Z}_2[x]$

# Context & Application

Cyclic codes are important in practice because they can be implemented efficiently using shift registers. The connection to ring theory via $R_n$ provides powerful algebraic tools for analyzing error-detection and correction capabilities.

# Examples

**Example 1** (p. 296): The $(6,3)$-code with generator matrix $G_1$ encoding $(a_0, a_1, a_2)$ to $(a_0, a_1, a_2, a_0, a_1, a_2)$ is cyclic. Shifting any codeword produces another codeword.

**Example 2** (p. 298): The factorization $x^7 - 1 = (1+x)(1+x+x^3)(1+x^2+x^3)$ gives a $(7,4)$-cyclic code generated by $g(t) = 1 + t + t^3$.
