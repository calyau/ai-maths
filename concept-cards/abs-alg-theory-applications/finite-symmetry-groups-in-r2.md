# Quick Definition

The only finite symmetry groups in $\mathbb{R}^2$ are the cyclic groups $\mathbb{Z}_n$ (pure rotations) and the dihedral groups $D_n$ (rotations and reflections).

# Core Definition

**Theorem 12.15**: "The only finite symmetry groups in $\mathbb{R}^2$ are $\mathbb{Z}_n$ and $D_n$" (Judson, p. 162). The proof divides into two cases: (1) if all elements have determinant 1, the group is cyclic (generated by the smallest rotation); (2) if some element has determinant $-1$, the group is dihedral.

# Prerequisites

- **Symmetry group** — This classifies finite symmetry groups
- **Orthogonal group** — Finite symmetry groups are subgroups of $O(2)$
- **Dihedral group** — One of the two types

# Key Properties

1. Case 1 (all $\det = 1$): the group is $\mathbb{Z}_n$ generated by the smallest positive rotation
2. Case 2 (some $\det = -1$): the group is $D_n$ with $n$ rotations and $n$ reflections
3. The proof uses $\det: G \to \{-1, 1\}$ as a homomorphism with cyclic kernel

# Context & Application

This classification theorem shows that the symmetry of any bounded figure in the plane must be one of $\mathbb{Z}_n$ or $D_n$. This has applications in crystallography and molecular chemistry.
