# Quick Definition

Every group of prime order $p$ is cyclic, generated by any non-identity element. Consequently, every group of prime order $p$ is isomorphic to $\mathbb{Z}_p$.

# Core Definition

**Corollary 6.12:** "Let $|G| = p$ with $p$ a prime number. Then $G$ is cyclic and any $g \in G$ such that $g \neq e$ is a generator" (Judson, p. 89). The proof follows from Corollary 6.11: the order of $g$ divides $p$, and since $|g| > 1$, we must have $|g| = p$.

# Prerequisites

- **Lagrange's Theorem** — The order of any element divides the group order

# Key Properties

1. $G$ is cyclic
2. Every non-identity element generates $G$
3. The only subgroups are $\{e\}$ and $G$ itself
4. $G \cong \mathbb{Z}_p$ (Corollary 9.9)

# Construction / Recognition

## To Recognize:
1. Verify $|G| = p$ for some prime $p$
2. Conclude $G$ is cyclic and $G \cong \mathbb{Z}_p$

# Context & Application

This corollary provides complete classification: there is exactly one group of order $p$ (up to isomorphism) for each prime $p$. This contrasts with composite orders, where multiple non-isomorphic groups may exist.

# Examples

**Example 1** (p. 89): If $|G| = 29$, then $G \cong \mathbb{Z}_{29}$, since 29 is prime.

# Common Errors

- **Error**: Trying to find non-trivial subgroups of a group of prime order
  **Correction**: The only subgroups are $\{e\}$ and $G$ itself

# Common Confusions

- **Confusion**: Thinking groups of prime-squared order are also necessarily cyclic
  **Clarification**: $\mathbb{Z}_{p^2}$ and $\mathbb{Z}_p \times \mathbb{Z}_p$ are both groups of order $p^2$ but are not isomorphic
