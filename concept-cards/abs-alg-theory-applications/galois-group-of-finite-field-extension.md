# Quick Definition

The Galois group of a finite field extension $GF(p^{nk})/GF(p^n)$ is cyclic of order $k$, generated by the Frobenius automorphism $\sigma(\alpha) = \alpha^{p^n}$.

# Core Definition

**Corollary 23.9.** Let $F$ be a finite field with a finite extension $E$ such that $[E:F] = k$. Then $G(E/F)$ is cyclic of order $k$ (Judson, p. 309).

The generator is $\sigma(\alpha) = \alpha^{p^n}$ where $|F| = p^n$.

# Prerequisites

- **Galois group** — This characterizes the Galois group for finite fields
- **Finite field** — The result applies to finite field extensions
- **Frobenius automorphism** — Generates the cyclic Galois group

# Key Properties

1. $G(E/F)$ is always cyclic for finite field extensions
2. The order equals $[E:F]$
3. The Frobenius map $\sigma(\alpha) = \alpha^{p^n}$ is the canonical generator
4. $\sigma^k = \text{id}$ since $\alpha^{p^{nk}} = \alpha$ for all $\alpha \in E$

# Examples

**Example 1**: $G(GF(2^4)/GF(2))$ is cyclic of order 4, generated by $\alpha \mapsto \alpha^2$.

**Example 2** (p. 309): $G(GF(p^{nk})/GF(p^n)) \cong \mathbb{Z}_k$.
