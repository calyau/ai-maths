# Quick Definition

A finite extension is a field extension $E/F$ where $E$ has finite dimension as a vector space over $F$. Every finite extension is algebraic, but not every algebraic extension is finite.

# Core Definition

If an extension field $E$ of a field $F$ is a finite dimensional vector space over $F$ of dimension $n$, then we say that $E$ is a **finite extension of degree $n$ over $F$** (Judson, p. 278).

# Prerequisites

- **Extension field** — A finite extension is an extension field
- **Degree of field extension** — The degree must be finite

# Key Properties

1. Every finite extension is algebraic (Theorem 21.15)
2. The converse is false: algebraic extensions can be infinite (Remark 21.16)
3. A finite extension is equivalent to being finitely generated by algebraic elements (Theorem 21.22)
4. The tower law applies: $[K:F] = [K:E][E:F]$

# Context & Application

Finite extensions are the well-behaved extensions that most of Galois theory concerns. They are finite-dimensional vector spaces over the base field, allowing linear algebraic techniques.

# Examples

**Example 1** (p. 278): $[\mathbb{C}:\mathbb{R}] = 2$ — a finite extension of degree 2.

**Example 2** (p. 280): $[\mathbb{Q}(\sqrt{3}, \sqrt{5}):\mathbb{Q}] = 4$ — a finite extension of degree 4.

**Example 3**: The field of all algebraic numbers is an algebraic but infinite (not finite) extension of $\mathbb{Q}$.
