# Quick Definition

A field $E$ is an extension field of a field $F$ if $F$ is a subfield of $E$. The field $F$ is called the base field, and we write $F \subset E$.

# Core Definition

A field $E$ is an **extension field** of a field $F$ if $F$ is a subfield of $E$. The field $F$ is called the **base field**. We write $F \subset E$ (Judson, p. 274).

# Prerequisites

- **Field** — One must understand the structure of fields before studying how they extend
- **Subring** — An extension field contains the base field as a subfield (a subring that is itself a field)

# Key Properties

1. Every extension field $E$ of $F$ can be viewed as a vector space over $F$ (p. 278)
2. There always exists an extension of $F$ containing a root of any nonconstant polynomial in $F[x]$ (Theorem 21.5, Kronecker's Theorem)
3. If $\alpha \in E$ is algebraic over $F$ with minimal polynomial of degree $n$, then $F(\alpha) \cong F[x]/\langle p(x) \rangle$ and $[F(\alpha):F] = n$
4. Extension fields can be chained: $F \subset E \subset K$

# Construction / Recognition

## To Construct:
1. Start with a field $F$ and an irreducible polynomial $p(x) \in F[x]$
2. Form the quotient ring $F[x]/\langle p(x) \rangle$
3. This is a field extension of $F$ containing a root of $p(x)$

## To Recognize:
1. Verify $E$ is a field
2. Verify $F \subset E$ as a subfield (the operations of $F$ agree with those of $E$)

# Context & Application

Field extensions are the central object of study in Chapters 21-23. They provide the framework for understanding roots of polynomials, geometric constructions, and Galois theory. The key insight is that field extensions can be studied using linear algebra: $E$ is a vector space over $F$, and $[E:F]$ controls the structure of the extension.

# Examples

**Example 1** (p. 274): $\mathbb{Q}(\sqrt{2}) = \{a + b\sqrt{2} : a, b \in \mathbb{Q}\}$ is an extension field of $\mathbb{Q}$.

**Example 2** (p. 275): $\mathbb{Z}_2[x]/\langle x^2 + x + 1 \rangle$ is a field with four elements and is an extension of $\mathbb{Z}_2$.

**Example 3** (p. 274): $\mathbb{R} \subset \mathbb{C}$ is a field extension with $[\mathbb{C}:\mathbb{R}] = 2$.

# Relationships

## Builds Upon
- **Field** — Extension fields extend the structure of a field

## Enables
- **Algebraic element** — Elements of an extension can be algebraic or transcendental
- **Splitting field** — An extension where a polynomial factors completely
- **Galois group** — Automorphisms of an extension fixing the base field
- **Degree of field extension** — The dimension of $E$ as a vector space over $F$

## Related
- **Simple extension** — An extension generated by a single element
- **Base field** — The smaller field being extended

# Common Errors

- **Error**: Assuming every field has only one extension
  **Correction**: A field generally has many different extensions; which one is appropriate depends on the polynomial being studied

# Common Confusions

- **Confusion**: Confusing a field extension with a ring extension
  **Clarification**: A field extension requires $E$ to be a field, not just a ring containing $F$
