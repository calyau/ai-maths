# Quick Definition

In a cyclic group of order $n$ generated by $a$, the order of the element $b = a^k$ is $n/\gcd(k, n)$.

# Core Definition

**Proposition 4.12**: In a cyclic group of order $n$ with generator $a$, $a^k = e$ if and only if $n$ divides $k$.

**Theorem 4.13**: "Let $G$ be a cyclic group of order $n$ and suppose that $a \in G$ is a generator of the group. If $b = a^k$, then the order of $b$ is $n/d$, where $d = \gcd(k, n)$" (Judson, p. 62).

**Corollary 4.14**: The generators of $\mathbb{Z}_n$ are the integers $r$ with $1 \leq r < n$ and $\gcd(r, n) = 1$.

# Key Properties

1. $|a^k| = n/\gcd(k, n)$ in a cyclic group of order $n$
2. $a^k$ is a generator iff $\gcd(k, n) = 1$ (order = $n$)
3. $a^k = e$ iff $n \mid k$
4. Determines the complete subgroup lattice of cyclic groups

# Construction / Recognition

## To Find the Order of $a^k$ in $\mathbb{Z}_n$:
1. Compute $d = \gcd(k, n)$
2. The order is $n/d$

## To Determine if $a^k$ Generates $\mathbb{Z}_n$:
1. Compute $\gcd(k, n)$
2. If $\gcd(k, n) = 1$, then $a^k$ is a generator

# Context & Application

This formula completely determines the order of every element in a cyclic group and characterizes all generators. It is fundamental for understanding the subgroup structure of cyclic groups.

# Examples

**Example 1** (p. 62): In $\mathbb{Z}_{16}$, the element 9 has order $16/\gcd(9, 16) = 16/1 = 16$, so 9 generates $\mathbb{Z}_{16}$.

**Example 2** (p. 60): In $\mathbb{Z}_6$, $|2| = 6/\gcd(2, 6) = 6/2 = 3$. Indeed $\langle 2 \rangle = \{0, 2, 4\}$ has 3 elements.

- **Error**: Computing order as $k$ instead of $n/\gcd(k, n)$
