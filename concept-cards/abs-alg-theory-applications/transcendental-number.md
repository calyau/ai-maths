# Quick Definition

A transcendental number is a complex number that is transcendental over $\mathbb{Q}$, meaning it is not the root of any nonzero polynomial with rational coefficients.

# Core Definition

A **transcendental number** is an element of $\mathbb{C}$ that is transcendental over $\mathbb{Q}$ (Judson, p. 276).

# Prerequisites

- **Transcendental element** — A transcendental number is a transcendental element over $\mathbb{Q}$

# Key Properties

1. Almost all real numbers are transcendental (the algebraic numbers are countable)
2. $\pi$ and $e$ are transcendental over $\mathbb{Q}$
3. It is unknown whether $\pi + e$ is transcendental
4. Transcendental numbers are not constructible (relevant to squaring the circle)

# Examples

**Example 1** (p. 276): $\pi$ is transcendental over $\mathbb{Q}$ (proved by Lindemann, 1882).

**Example 2** (p. 276): $e$ is transcendental over $\mathbb{Q}$ (proved by Hermite, 1873).
