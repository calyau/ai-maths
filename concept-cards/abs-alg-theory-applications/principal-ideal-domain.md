# Quick Definition

A principal ideal domain (PID) is an integral domain in which every ideal is principal, i.e., generated by a single element.

# Core Definition

"An integral domain in which every ideal is principal is called a principal ideal domain, or PID" (Judson, p. 243).

# Prerequisites

- **Integral domain** — A PID is an integral domain
- **Principal ideal** — Every ideal must be of the form $\langle a \rangle$

# Key Properties

1. Every ideal $I$ is principal: $I = \langle a \rangle$ for some $a$
2. Every PID is a UFD (Theorem 18.15)
3. In a PID, irreducible $\Leftrightarrow$ prime (Corollary 18.13)
4. In a PID, $\langle p \rangle$ is maximal iff $p$ is irreducible (Theorem 18.12)
5. PIDs satisfy the ascending chain condition (Lemma 18.14)
6. $\mathbb{Z}$ and $F[x]$ are PIDs

# Construction / Recognition

## To Verify:
1. Confirm the ring is an integral domain
2. Show every ideal is generated by a single element

# Context & Application

PIDs are central in the hierarchy of integral domains: Euclidean domain $\subset$ PID $\subset$ UFD. In PIDs, the ideal structure is as simple as possible, making factorization theory clean.

# Examples

**Example 1**: $\mathbb{Z}$ is a PID (Theorem 16.25).

**Example 2**: $F[x]$ for any field $F$ is a PID (Theorem 17.20).

**Example 3** (p. 245): $\mathbb{Z}[x]$ is NOT a PID: the ideal $I = \{5f(x) + xg(x)\}$ is not principal.

# Relationships

## Builds Upon
- **Integral domain** — A PID is an integral domain

## Enables
- **Unique factorization** — Every PID is a UFD

## Related
- **Euclidean domain** — Every Euclidean domain is a PID

## Contrasts With
- **UFD** — Every PID is a UFD, but $\mathbb{Z}[x]$ is a UFD that is not a PID

# Common Errors

- **Error**: Assuming every UFD is a PID
  **Correction**: $\mathbb{Z}[x]$ is a UFD but not a PID

# Common Confusions

- **Confusion**: Confusing PID with "principal ideal ring"
  **Clarification**: A PID must be an *integral domain* (commutative, with identity, no zero divisors)
