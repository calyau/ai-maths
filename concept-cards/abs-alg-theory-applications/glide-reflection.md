# Quick Definition

A glide reflection is an isometry that combines a translation with a reflection. In $\mathbb{R}^2$, the four types of isometries are rotations, reflections, translations, and glide reflections.

# Core Definition

"A **glide reflection** is a translation followed by a reflection" (Judson, p. 161). It is one of the four basic types of isometries in $\mathbb{R}^2$.

# Prerequisites

- **Isometry** — A glide reflection is a specific type of isometry

# Key Properties

1. Composition of translation and reflection
2. Has infinite order (like translations)
3. Preserves distances
4. Appears in some wallpaper group classifications
