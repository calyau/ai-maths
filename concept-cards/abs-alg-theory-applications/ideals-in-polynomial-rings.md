# Quick Definition

If $F$ is a field, then every ideal in $F[x]$ is principal. That is, $F[x]$ is a principal ideal domain (PID).

# Core Definition

"If $F$ is a field, then every ideal in $F[x]$ is a principal ideal" (Judson, Theorem 17.20, p. 232). The proof uses the division algorithm: the ideal is generated by a nonzero element of minimal degree.

# Prerequisites

- **Polynomial ring** â€” The result concerns $F[x]$
- **Field** â€” The division algorithm requires a field

# Key Properties

1. Every ideal $I$ in $F[x]$ has the form $\langle p(x) \rangle$ for some $p(x) \in F[x]$
2. The generator is a polynomial of minimal degree in $I$
3. $F[x]$ is a PID
4. This fails for $F[x, y]$ (Example 17.21: the ideal $\langle x, y \rangle$ is not principal)

# Construction / Recognition

## To Find the Generator:
1. Take a nonzero element $p(x)$ of minimal degree in $I$
2. Then $I = \langle p(x) \rangle$

# Context & Application

This theorem is a key step in the chain: $F$ field $\Rightarrow$ $F[x]$ is a Euclidean domain $\Rightarrow$ $F[x]$ is a PID $\Rightarrow$ $F[x]$ is a UFD. It shows that polynomial rings over fields have ideal structures as simple as $\mathbb{Z}$.

# Examples

**Example 1** (p. 232): $\langle x^2 \rangle$ in $F[x]$ consists of all polynomials with no constant or linear term.

**Example 2** (p. 232): The ideal $\langle x, y \rangle$ in $F[x,y]$ is not principal, showing the theorem fails for multivariate polynomials.

# Common Errors

- **Error**: Assuming all polynomial rings are PIDs
  **Correction**: $\mathbb{Z}[x]$ is a UFD but not a PID; $F[x,y]$ is not a PID
