# Quick Definition

A cyclic group is a group $G$ that can be generated by a single element $a$; that is, $G = \langle a \rangle = \{a^k : k \in \mathbb{Z}\}$.

# Core Definition

"If $G$ contains some element $a$ such that $G = \langle a \rangle$, then $G$ is a cyclic group. In this case $a$ is a generator of $G$" (Judson, p. 60).

Key theorems:
- **Theorem 4.9**: Every cyclic group is abelian.
- **Theorem 4.10**: Every subgroup of a cyclic group is cyclic.

# Prerequisites

- **Group** — A cyclic group is a group
- **Cyclic subgroup** — A cyclic group is a cyclic subgroup that equals the whole group
- **Generator** — A cyclic group has at least one generator

# Key Properties

1. Generated by a single element
2. Every cyclic group is abelian (Theorem 4.9)
3. Every subgroup of a cyclic group is cyclic (Theorem 4.10)
4. A cyclic group of order $n$ is isomorphic to $\mathbb{Z}_n$
5. An infinite cyclic group is isomorphic to $\mathbb{Z}$
6. May have more than one generator

# Construction / Recognition

## To Determine if $G$ is Cyclic:
1. Find an element $a \in G$
2. Compute powers $a, a^2, a^3, \ldots$
3. If all elements of $G$ appear, then $G = \langle a \rangle$ is cyclic
4. If no single element generates all of $G$, then $G$ is not cyclic

# Context & Application

Cyclic groups are among the simplest and most fundamental groups. $\mathbb{Z}$ and $\mathbb{Z}_n$ are the prototypical cyclic groups. Every group of prime order is cyclic. Cyclic groups play a fundamental role in the classification of all finite abelian groups.

# Examples

**Example 1** (p. 60): $\mathbb{Z}_6$ is cyclic with generators 1 and 5 (since $\gcd(1,6) = \gcd(5,6) = 1$). The element 2 generates only the subgroup $\{0, 2, 4\}$.

**Example 2** (p. 60): $\mathbb{Z}$ is cyclic with generators 1 and $-1$.

**Example 3** (p. 60): $U(9) = \{1, 2, 4, 5, 7, 8\}$ is cyclic with generator 2.

**Non-Example** (p. 60): $S_3$ is NOT cyclic, even though every subgroup of $S_3$ is cyclic. No single element generates all of $S_3$.

# Common Errors

- **Error**: Checking only one potential generator and concluding the group is not cyclic
  **Correction**: Must check all elements; a group can be cyclic even if most elements are not generators

# Common Confusions

- **Confusion**: Thinking every abelian group is cyclic
  **Clarification**: $\mathbb{Z}_2 \times \mathbb{Z}_2$ is abelian but not cyclic (no element has order 4)

- **Confusion**: Thinking $U(n)$ is always cyclic
  **Clarification**: $U(8) = \{1, 3, 5, 7\}$ is not cyclic since every element has order $\leq 2$
