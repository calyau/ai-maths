# Quick Definition

The Primitive Element Theorem states that every finite separable extension of a field $F$ is a simple extension: there exists a single element $\alpha$ such that $E = F(\alpha)$.

# Core Definition

**Theorem 23.13 (Primitive Element Theorem).** Let $E$ be a finite separable extension of a field $F$. Then there exists an $\alpha \in E$ such that $E = F(\alpha)$ (Judson, p. 311).

Such an element $\alpha$ is called a **primitive element** of the extension.

# Prerequisites

- **Separable extension** — The theorem requires separability
- **Simple extension** — The conclusion is that the extension is simple

# Key Properties

1. Every finite extension of a finite field is simple (Corollary 22.12), without needing separability
2. For infinite fields, separability is needed
3. The proof is constructive for $F(\alpha, \beta)$: find $\gamma = \alpha + a\beta$ for suitable $a \in F$

# Context & Application

The theorem simplifies the study of field extensions by showing that most extensions of interest are generated by a single element. This reduces the study of finite separable extensions to the study of simple extensions, which are controlled by minimal polynomials.

# Examples

**Example 1** (p. 311): $\mathbb{Q}(\sqrt{3}, \sqrt{5}) = \mathbb{Q}(\sqrt{3} + \sqrt{5})$, so $\sqrt{3} + \sqrt{5}$ is a primitive element.

**Example 2** (p. 311): $\mathbb{Q}(\sqrt[3]{5}, \sqrt{5}i) = \mathbb{Q}(\sqrt[6]{5}i)$, so $\sqrt[6]{5}i$ is a primitive element.
