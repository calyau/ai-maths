# Quick Definition

The standard generator matrix $G$ associated with canonical parity-check matrix $H = (A \mid I_m)$ has the form $G = \begin{pmatrix} I_{n-m} \\ A \end{pmatrix}$. Encoding a message $\mathbf{x}$ is done by computing $G\mathbf{x}$.

# Core Definition

"With each canonical parity-check matrix we can associate an $n \times (n-m)$ standard generator matrix $G = \begin{pmatrix} I_{n-m} \\ A \end{pmatrix}$" (Judson, p. 114). The key property: $\mathbf{y}$ is a codeword (in Null($H$)) if and only if $\mathbf{y} = G\mathbf{x}$ for some $\mathbf{x} \in \mathbb{Z}_2^{n-m}$ (Theorem 8.28). Also, $HG = \mathbf{0}$ (Lemma 8.27).

# Prerequisites

- **Canonical Parity-Check Matrix** — $G$ is derived from $H$

# Key Properties

1. $G = \begin{pmatrix} I_{n-m} \\ A \end{pmatrix}$ where $A$ comes from $H = (A \mid I_m)$
2. $HG = \mathbf{0}$ (Lemma 8.27)
3. $G\mathbf{x} = \mathbf{y}$ gives a codeword; different $\mathbf{x}$ give different $\mathbf{y}$ (Theorem 8.26)
4. The first $n - m$ bits of $G\mathbf{x}$ are exactly $\mathbf{x}$ (the information bits)
5. The code generated by $G$ is the same as Null($H$) (Theorem 8.28)

# Construction / Recognition

## To Encode:
1. Given message $\mathbf{x} \in \mathbb{Z}_2^{n-m}$, compute $\mathbf{y} = G\mathbf{x}$
2. The first $n-m$ positions of $\mathbf{y}$ are the information bits
3. The last $m$ positions are the check bits

# Context & Application

The generator matrix provides a systematic and efficient way to encode messages. It is computationally cheaper than solving the system $H\mathbf{y} = \mathbf{0}$ directly.

# Examples

**Example 1** (p. 115-116): With $G = \begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 1 \\ 0 & 1 & 1 \\ 1 & 1 & 0 \\ 1 & 0 & 1 \end{pmatrix}$, encoding $(010)$ gives $G(010)^t = (010110)^t$.

# Common Errors

- **Error**: Confusing the roles of $G$ and $H$
  **Correction**: $G$ is for encoding ($\mathbf{y} = G\mathbf{x}$); $H$ is for checking ($H\mathbf{y} = \mathbf{0}$ for codewords)

# Common Confusions

- **Confusion**: Thinking $G$ and $H$ are inverses
  **Clarification**: They satisfy $HG = \mathbf{0}$, not $HG = I$
