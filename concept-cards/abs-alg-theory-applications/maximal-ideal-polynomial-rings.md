# Quick Definition

In a polynomial ring $F[x]$ over a field $F$, the ideal $\langle p(x) \rangle$ is maximal if and only if $p(x)$ is irreducible over $F$. Consequently, $F[x]/\langle p(x) \rangle$ is a field.

# Core Definition

"Let $F$ be a field and suppose that $p(x) \in F[x]$. Then the ideal generated by $p(x)$ is maximal if and only if $p(x)$ is irreducible" (Judson, Theorem 17.22, p. 232).

# Prerequisites

- **Ideals in polynomial rings** — $F[x]$ is a PID, so all ideals are principal
- **Irreducible polynomial** — The characterization involves irreducibility
- **Maximal ideal** — $R/M$ is a field iff $M$ is maximal

# Key Properties

1. $\langle p(x) \rangle$ is maximal $\Leftrightarrow$ $p(x)$ is irreducible
2. $F[x]/\langle p(x) \rangle$ is a field when $p(x)$ is irreducible
3. This provides a method to construct field extensions

# Construction / Recognition

## To Apply:
1. Test if $p(x)$ is irreducible over $F$
2. If yes, $\langle p(x) \rangle$ is maximal and $F[x]/\langle p(x) \rangle$ is a field

# Context & Application

This theorem is the bridge between irreducible polynomials and field extensions. To extend a field $F$ by a root of an irreducible polynomial $p(x)$, form $F[x]/\langle p(x) \rangle$. This is the fundamental construction in Galois theory.

# Examples

**Example 1**: $\langle x^2 + 1 \rangle$ is maximal in $\mathbb{R}[x]$ since $x^2 + 1$ is irreducible over $\mathbb{R}$. Thus $\mathbb{R}[x]/\langle x^2 + 1 \rangle \cong \mathbb{C}$.

**Example 2**: $\langle x^2 - 2 \rangle$ is maximal in $\mathbb{Q}[x]$ since $x^2 - 2$ is irreducible over $\mathbb{Q}$. Thus $\mathbb{Q}[x]/\langle x^2 - 2 \rangle \cong \mathbb{Q}(\sqrt{2})$.

# Relationships

## Builds Upon
- **Irreducible polynomial** — Irreducibility characterizes maximality
- **Maximal ideal** — The quotient by a maximal ideal is a field

## Enables
- **Field extensions** — $F[x]/\langle p(x) \rangle$ constructs extensions containing roots of $p(x)$

# Common Errors

- **Error**: Assuming $\langle p(x) \rangle$ is maximal in $\mathbb{Z}[x]$ when $p(x)$ is irreducible
  **Correction**: This theorem requires $F$ to be a field; $\mathbb{Z}[x]$ is not a PID

# Common Confusions

- **Confusion**: Thinking irreducibility over $F$ implies irreducibility over extensions of $F$
  **Clarification**: $x^2 + 1$ is irreducible over $\mathbb{R}$ but reducible over $\mathbb{C}$
