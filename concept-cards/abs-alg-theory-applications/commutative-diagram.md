# Quick Definition

A commutative diagram is a diagram of groups and homomorphisms where every path between two groups yields the same composite homomorphism. The First Isomorphism Theorem is expressed by a commutative triangle: $\psi = \eta \circ \phi$.

# Core Definition

"Mathematicians often use diagrams called **commutative diagrams** to describe such theorems. The following diagram 'commutes' since $\psi = \eta\phi$" (Judson, p. 149), referring to the First Isomorphism Theorem.

# Prerequisites

- **Group homomorphism** — Arrows in the diagram represent homomorphisms

# Key Properties

1. A diagram "commutes" if all paths give the same map
2. The First Isomorphism Theorem: $G \xrightarrow{\phi} G/K \xrightarrow{\eta} \psi(G)$ commutes with $\psi: G \to H$
3. Commutative diagrams provide visual representations of algebraic relationships
