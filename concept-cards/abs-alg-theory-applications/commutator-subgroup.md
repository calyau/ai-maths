# Quick Definition

The commutator subgroup $G'$ of a group $G$ is the subgroup generated by all commutators $aba^{-1}b^{-1}$. It is normal in $G$, and $G/G'$ is abelian. Moreover, $G/N$ is abelian if and only if $N$ contains $G'$.

# Core Definition

"Let $G$ be a group and let $G' = \langle aba^{-1}b^{-1} \rangle$; that is, $G'$ is the subgroup of all finite products of elements in $G$ of the form $aba^{-1}b^{-1}$. The subgroup $G'$ is called the **commutator subgroup** of $G$" (Judson, p. 145, Exercise 10.4.14). The commutator subgroup is normal in $G$, and $G/N$ is abelian if and only if $N$ contains $G'$.

# Prerequisites

- **Normal subgroup** — $G'$ is always a normal subgroup
- **Group** — The ambient structure

# Key Properties

1. $G'$ is a normal subgroup of $G$
2. $G/G'$ is abelian
3. For a normal subgroup $N$ of $G$: $G/N$ is abelian if and only if $G' \subseteq N$
4. $G$ is abelian if and only if $G' = \{e\}$
5. The commutator subgroup is the smallest normal subgroup with abelian quotient

# Construction / Recognition

## To Compute the Commutator Subgroup:
1. Compute all commutators $aba^{-1}b^{-1}$ for $a, b \in G$
2. Take the subgroup generated by all such commutators
3. Note: the set of commutators may not itself be a subgroup; one must take the generated subgroup

# Context & Application

The commutator subgroup measures "how far $G$ is from being abelian." It is central to the theory of solvable groups, where the derived series $G \supset G' \supset G'' \supset \cdots$ is formed by iterating the commutator subgroup construction. A group is solvable if and only if this derived series eventually reaches $\{e\}$.

# Examples

**Example 1**: For an abelian group $G$, $G' = \{e\}$ since $aba^{-1}b^{-1} = e$ for all $a, b$.

**Example 2** (p. 199): For a group $G$ of order $pq$, the commutator subgroup $G'$ is contained in any normal subgroup $N$ such that $G/N$ is abelian.

# Relationships

## Builds Upon
- **Normal subgroup** — $G'$ is always normal in $G$

## Enables
- **Solvable group** — Defined via the derived series $G^{(0)} = G, G^{(i+1)} = (G^{(i)})'$
- **Derived series** — The sequence $G \supset G' \supset G'' \supset \cdots$

## Related
- **Factor group** — $G/G'$ is the largest abelian quotient of $G$

# Common Errors

- **Error**: Assuming the set of commutators $\{aba^{-1}b^{-1}\}$ is already a subgroup
  **Correction**: One must take the subgroup *generated by* all commutators; not every product of commutators is a single commutator

# Common Confusions

- **Confusion**: Confusing the commutator $[a,b] = aba^{-1}b^{-1}$ with the group operation
  **Clarification**: The commutator measures the failure of $a$ and $b$ to commute; it equals $e$ precisely when $ab = ba$
