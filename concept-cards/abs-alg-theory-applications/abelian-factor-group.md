# Quick Definition

If $G$ is abelian, then $G/H$ is abelian for any subgroup $H$. More generally, $G/N$ is abelian if and only if $N$ contains the commutator subgroup $G'$.

# Core Definition

**Exercise 10.4.6**: "If $G$ is abelian, prove that $G/H$ must also be abelian." **Exercise 10.4.14(b)**: "$G/N$ is abelian if and only if $N$ contains the commutator subgroup of $G$" (Judson, pp. 144-145).

# Prerequisites

- **Factor group** — The structure being studied
- **Normal subgroup** — Required for factor groups

# Key Properties

1. If $G$ is abelian, every factor group $G/H$ is abelian
2. $G/N$ abelian $\iff$ $G' \leq N$
3. $G/G'$ is the "largest" abelian quotient of $G$
