# Quick Definition

A $p$-subgroup of a group $G$ is a subgroup that is itself a $p$-group, meaning every element has order a power of $p$. Equivalently (for finite groups), a subgroup whose order is a power of $p$.

# Core Definition

"A subgroup of a group $G$ is a **$p$-subgroup** if it is a $p$-group" (Judson, p. 195).

# Prerequisites

- **$p$-group** — A $p$-subgroup is a subgroup that is a $p$-group
- **Subgroup** — Must be a subgroup

# Key Properties

1. A $p$-subgroup has order $p^k$ for some $k \geq 0$
2. Every $p$-subgroup is contained in some Sylow $p$-subgroup

# Relationships

## Builds Upon

## Related

## Contrasts With
