# Quick Definition

A field extension $E/F$ is finite if and only if $E = F(\alpha_1, \ldots, \alpha_n)$ where each $\alpha_i$ is algebraic over $F$. Equivalently, there exists a tower of simple algebraic extensions from $F$ to $E$.

# Core Definition

**Theorem 21.22.** Let $E$ be a field extension of $F$. Then the following statements are equivalent (Judson, p. 280):

1. $E$ is a finite extension of $F$.
2. There exists a finite number of algebraic elements $\alpha_1, \ldots, \alpha_n \in E$ such that $E = F(\alpha_1, \ldots, \alpha_n)$.
3. There exists a tower $E = F(\alpha_1, \ldots, \alpha_n) \supset \cdots \supset F(\alpha_1) \supset F$ where each step is algebraic.

# Prerequisites

- **Finite extension** — The concept being characterized
- **Algebraic element** — The generators must be algebraic

# Key Properties

1. Finite = finitely generated by algebraic elements
2. Each step $F(\alpha_1, \ldots, \alpha_i)/F(\alpha_1, \ldots, \alpha_{i-1})$ is finite
3. The total degree is the product of the step degrees (tower law)

# Examples

**Example 1** (p. 280): $\mathbb{Q}(\sqrt{3}, \sqrt{5}) = \mathbb{Q}(\sqrt{3})(\sqrt{5})$ with $[\mathbb{Q}(\sqrt{3}, \sqrt{5}):\mathbb{Q}] = 4$.
