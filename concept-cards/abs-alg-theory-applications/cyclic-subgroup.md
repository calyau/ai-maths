# Quick Definition

The cyclic subgroup generated by an element $a$ in a group $G$ is $\langle a \rangle = \{a^k : k \in \mathbb{Z}\}$, the smallest subgroup of $G$ containing $a$.

# Core Definition

**Theorem 4.3**: "Let $G$ be a group and $a$ be any element in $G$. Then the set $\langle a \rangle = \{a^k : k \in \mathbb{Z}\}$ is a subgroup of $G$. Furthermore, $\langle a \rangle$ is the smallest subgroup of $G$ that contains $a$" (Judson, p. 59).

For groups written additively: $\langle a \rangle = \{na : n \in \mathbb{Z}\}$ (Remark 4.4).

# Prerequisites

- **Subgroup** — A cyclic subgroup is a subgroup
- **Group** — Defined within a group

# Key Properties

1. Contains all powers of $a$: $\ldots, a^{-2}, a^{-1}, e, a, a^2, \ldots$
2. Is the smallest subgroup containing $a$
3. Identity: $a^0 = e \in \langle a \rangle$
4. Closure: $a^m \cdot a^n = a^{m+n} \in \langle a \rangle$
5. Inverses: $(a^n)^{-1} = a^{-n} \in \langle a \rangle$

# Construction / Recognition

## To Construct $\langle a \rangle$:
1. Compute $a, a^2, a^3, \ldots$ until elements repeat (for finite groups)
2. Include $e = a^0$ and all negative powers $a^{-1}, a^{-2}, \ldots$
3. The resulting set is $\langle a \rangle$

# Context & Application

Cyclic subgroups are the building blocks of cyclic groups. Every element of a group generates a cyclic subgroup. Understanding cyclic subgroups is essential for determining group structure.

# Examples

**Example 1** (p. 59): In $\mathbb{Z}$, $\langle 3 \rangle = 3\mathbb{Z} = \{\ldots, -3, 0, 3, 6, \ldots\}$.

**Example 2** (p. 59): In $\mathbb{Q}^*$, $\langle 2 \rangle = \{2^n : n \in \mathbb{Z}\} = \{\ldots, 1/4, 1/2, 1, 2, 4, \ldots\}$.

**Example 3** (p. 60): In $\mathbb{Z}_6$, $\langle 2 \rangle = \{0, 2, 4\}$.

# Common Errors

- **Error**: Forgetting negative powers when computing $\langle a \rangle$
  **Correction**: $\langle a \rangle$ includes all integer powers, both positive and negative

# Common Confusions

- **Confusion**: Thinking $\langle a \rangle$ always equals the whole group
  **Clarification**: $\langle a \rangle$ may be a proper subgroup; it equals $G$ only when $a$ generates $G$
