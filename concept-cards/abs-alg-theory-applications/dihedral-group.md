# Quick Definition

The $n$th dihedral group $D_n$ is the group of rigid motions (rotations and reflections) of a regular $n$-gon. It is a subgroup of $S_n$ with order $2n$.

# Core Definition

"For $n = 3, 4, \ldots$, we define the $n$th dihedral group to be the group of rigid motions of a regular $n$-gon. We will denote this group by $D_n$" (Judson, p. 78). $D_n$ is a subgroup of $S_n$ of order $2n$ (Theorem 5.20). It consists of $n$ rotations and $n$ reflections.

# Prerequisites

- **Permutation Group** — $D_n$ is a permutation group
- **Symmetric Group** — $D_n$ is a subgroup of $S_n$

# Key Properties

1. $|D_n| = 2n$ (Theorem 5.20)
2. $D_n$ is generated by two elements $r$ (rotation) and $s$ (reflection) satisfying $r^n = 1$, $s^2 = 1$, $srs = r^{-1}$ (Theorem 5.21)
3. $D_n = \{1, r, r^2, \ldots, r^{n-1}, s, rs, r^2s, \ldots, r^{n-1}s\}$
4. $D_n$ is nonabelian for $n \geq 3$
5. The $n$ rotations form a cyclic subgroup $\langle r \rangle$ of order $n$

# Construction / Recognition

## To Construct $D_n$:
1. Let $r$ be the rotation by $360^\circ/n$
2. Let $s$ be any single reflection
3. The group consists of elements $r^k$ and $r^k s$ for $k = 0, 1, \ldots, n-1$
4. Compute products using the relation $srs = r^{-1}$ (equivalently $sr^k = r^{-k}s$)

## To Recognize:
1. Check order is $2n$ for some $n \geq 3$
2. Verify it contains an element of order $n$ and an element of order 2
3. Check the relation $srs = r^{-1}$

# Context & Application

Dihedral groups arise naturally as symmetry groups of regular polygons. They are among the most important families of finite groups and provide key examples of nonabelian groups.

# Examples

**Example 1** (p. 80): $D_4$ (symmetries of a square) has 8 elements. The rotations are $r = (1234)$, $r^2 = (13)(24)$, $r^3 = (1432)$, $r^4 = (1)$. The reflections are $s_1 = (24)$, $s_2 = (13)$, $rs_1 = (12)(34)$, $r^3 s_1 = (14)(23)$.

# Relationships

## Builds Upon
- **Permutation Group** — $D_n$ is a permutation group
- **Symmetric Group** — $D_n \leq S_n$

## Enables
- **Internal Direct Product** — $D_6 \cong \mathbb{Z}_2 \times S_3$ (Chapter 9)

## Related
- **Alternating Group** — Both are important subgroups of $S_n$

# Common Errors

- **Error**: Computing $sr$ as $rs$ (forgetting that $D_n$ is nonabelian)
  **Correction**: Use the relation $sr = r^{-1}s$ (equivalently $srs = r^{-1}$)

# Common Confusions

- **Confusion**: Confusing $|D_n| = 2n$ with $|D_n| = n$ or $|D_n| = n!$
  **Clarification**: $D_n$ has exactly $2n$ elements: $n$ rotations and $n$ reflections
