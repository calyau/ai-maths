# Quick Definition

If x is an element of a group and x^n = e for some positive integer n, then x has finite order, and the smallest such positive integer m with x^m = e is called the order of x. If no such n exists, x has infinite order.

# Core Definition

"If x is an element of a group, and if x^n = e for some positive integer n, then we say x has finite order, and the smallest positive integer m such that x^m = e is called the order of x. Otherwise x has infinite order" (Armstrong, Ch. 4, p. 25).

# Prerequisites

- **Group** — Order is defined for elements of a group
- **Identity Element** — Order is defined relative to the identity
- **Power of an Element** — x^n must be defined

# Key Properties

1. The order of x is the smallest positive m with x^m = e
2. The identity element always has order 1
3. x has order 2 if and only if x = x^{-1} (and x != e)
4. The order of x equals the order of the subgroup <x>
5. In a finite group, every element has finite order
6. Elements of order 2 are called involutions

# Construction / Recognition

## To Find the Order of x:
1. Compute x, x^2, x^3, ... in succession
2. The first positive power that equals e gives the order
3. If working in Z_n with addition, the order of k is n/gcd(k,n)

# Context & Application

The order of an element is crucial for understanding group structure. In D_3, elements have orders 1 (identity), 2 (reflections), or 3 (non-trivial rotations). In Z_6, elements have orders 1, 2, 3, or 6. A group of even order must contain an element of order 2 (Exercise 4.5).

# Examples

**Example 1** (p. 25): In D_3, r and r^2 have order 3; s, rs, r^2s have order 2.

**Example 2** (p. 25): In Z_6, elements 1 and 5 have order 6; elements 2 and 4 have order 3; element 3 has order 2.

**Example 3** (p. 25): In R (additive group of reals), every non-zero element has infinite order.

**Example 4** (p. 25): In the unit circle group C, the element e^{i*theta} has finite order precisely when theta = 2m*pi/n for integers m, n.

# Common Errors

- **Error**: Confusing order of a group with order of an element
  **Correction**: |G| = number of elements in G; order of x = smallest m with x^m = e

- **Error**: In additive groups, forgetting that "x^m" means mx (m copies of x added)
  **Correction**: In additive notation, the order of x is the smallest positive m with mx = 0

# Common Confusions

- **Confusion**: Thinking every element of an infinite group has infinite order
  **Clarification**: The unit circle group C is infinite but contains elements of every finite order
