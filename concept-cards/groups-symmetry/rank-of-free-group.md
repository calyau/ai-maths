# Quick Definition

The rank of a free group is the cardinality of any free set of generators. Theorem 27.3 establishes that this is an isomorphism invariant: F_m is isomorphic to F_n if and only if m = n.

# Core Definition

"Let F_n stand for 'the' group which is freely generated by n elements" (Armstrong, Ch. 27, p. 175). The rank n is well-defined because of two theorems: (27.2) "Abelianising F_n gives Z^n," and (27.3) "If F_m is isomorphic to F_n, then m = n." The proof of Theorem 27.3 uses abelianization: an isomorphism F_m -> F_n induces an isomorphism Z^m -> Z^n, forcing m = n.

# Prerequisites

- **Free group** -- The rank is a property of free groups

# Key Properties

1. Abelianizing F_n gives Z^n (Theorem 27.2)
2. F_m = F_n if and only if m = n (Theorem 27.3)
3. The rank is therefore a complete isomorphism invariant for free groups
4. A bijection between free generating sets induces an isomorphism (Armstrong, p. 175)
5. F_1 = Z (rank 1 is infinite cyclic)
6. For rank >= 2, the free group is non-abelian with every element of infinite order

# Construction / Recognition

## To Determine the Rank
1. Find a free set of generators X for the group
2. The rank is |X|
3. Alternative: abelianize and find the rank of the resulting free abelian group

# Context & Application

The rank invariance theorem (27.3) ensures that the concept of "F_n" is well-defined. Armstrong's proof via abelianization is elegant: it reduces the problem to the easier question of when Z^m = Z^n, which is settled by linear algebra (or the fundamental theorem of finitely generated abelian groups).

# Examples

**Example 1** (p. 175): F_1 = Z has rank 1.

**Example 2** (p. 175): F_2 = F({x, y}) has rank 2; it is non-abelian.

**Example 3** (Exercise 27.9, p. 178): F_2 contains subgroups of every finite rank and of infinite rank.

# Common Errors

- **Error**: Assuming the rank of a subgroup is at most the rank of the ambient free group
  **Correction**: F_2 contains subgroups of every countable rank (Exercise 27.9-27.10)

# Common Confusions

- **Confusion**: Confusing rank of a free group with rank of a free abelian group
  **Clarification**: Both are well-defined invariants, but they apply to different classes of groups; a free group of rank n abelianizes to a free abelian group of rank n
