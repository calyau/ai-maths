# Quick Definition

For $n \ge 3$, the alternating group $A_n$ is generated by the set of all 3-cycles.

# Core Definition

"(6.5) Theorem. For $n \ge 3$ the 3-cycles generate $A_n$" (Armstrong, p. 37). The proof proceeds by writing any element of $A_n$ as a product of an even number of transpositions of the form $(1a)$ (using Theorem 6.2a), collecting them into adjacent pairs, and combining each pair using the identity $(1a)(1b) = (1ba)$ to obtain 3-cycles.

# Prerequisites

- **Alternating group** — The group being generated
- **Transposition** — The proof works by pairing transpositions into 3-cycles

# Key Properties

1. Every 3-cycle is an even permutation (a 3-cycle decomposes into 2 transpositions)
2. Every element of $A_n$ can be written as a product of 3-cycles
3. The key identity is $(1a)(1b) = (1ba)$

# Construction / Recognition

## To Express an Element of $A_n$ as 3-Cycles:
1. Write the element as a product of transpositions of the form $(1a)$ using Theorem 6.2a
2. Since the element is even, there is an even number of such transpositions
3. Pair adjacent transpositions: $(1a)(1b) = (1ba)$
4. The result is a product of 3-cycles

# Context & Application

This theorem provides a minimal generating set for $A_n$ in terms of element types. It is used in Chapter 8 to establish that the rotational symmetry group of the dodecahedron is isomorphic to $A_5$: one shows that every 3-cycle in $S_5$ arises from a rotation.

# Examples

**Example** (p. 37): $(13)(24) = (13)(12)(14)(12) = (123)(124)$.

# Common Errors

- **Error**: Trying to use 3-cycles of the form $(abc)$ with arbitrary $a, b, c$ without reducing to $(1a)(1b)$ form first.
  **Correction**: The proof specifically uses transpositions involving 1, then combines pairs via $(1a)(1b) = (1ba)$.

# Common Confusions

- **Confusion**: Thinking $A_n$ is generated by a single 3-cycle.
  **Clarification**: $A_n$ is generated by the set of all 3-cycles, not by any single one.
