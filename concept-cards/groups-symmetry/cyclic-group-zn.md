# Quick Definition

The cyclic group Z_n is the finite abelian group consisting of the integers {0, 1, 2, ..., n-1} under addition modulo n. It has order n and is generated by the element 1.

# Core Definition

"We have a finite abelian group with n elements which will be denoted by Z_n" (Armstrong, Ch. 3, p. 20). The group operation is addition modulo n, zero is the identity, n - x is the inverse of x (for x != 0), and the group is abelian because x +_n y = y +_n x.

# Prerequisites

- **Group** — Z_n satisfies the group axioms
- **Addition Modulo n** — The group operation
- **Abelian Group** — Z_n is abelian

# Key Properties

1. The set is {0, 1, 2, ..., n-1}
2. The group operation is addition modulo n
3. The identity is 0
4. The inverse of x is n - x (for x != 0)
5. The group is abelian
6. The group has order n
7. Generated by 1 (and by any element coprime to n)
8. Every finite cyclic group of order n is essentially Z_n

# Construction / Recognition

## To Construct Z_n:
1. Take the set {0, 1, 2, ..., n-1}
2. Define the operation x +_n y = (x + y) mod n
3. This gives a group of order n

## To Recognize:
1. Check if the group has order n and is generated by a single element
2. If so, it is isomorphic to Z_n

# Context & Application

Z_n is the prototypical finite cyclic group. It appears throughout the text as a basic building block. The order of each element divides n, and for each divisor d of n, there is exactly one subgroup of order d (Theorem 5.3).

# Examples

**Example 1** (p. 20): Z_6 has elements {0, 1, 2, 3, 4, 5}. The element 1 has order 6, elements 2 and 4 have order 3, and element 3 has order 2 (Ch. 4, p. 25).

**Example 2** (Ch. 5, p. 29): In Z_6, the subgroups generated by each element are: <0> = {0}, <1> = <5> = Z_6, <2> = <4> = {0, 2, 4}, <3> = {0, 3}.

# Common Errors

- **Error**: Confusing Z_n (additive group) with the multiplicative group modulo n
  **Correction**: Z_n uses addition mod n on {0, ..., n-1}; the multiplicative group uses multiplication mod n on {1, ..., n-1} and requires n to be prime

# Common Confusions

- **Confusion**: Thinking Z_6 and D_3 are the same because both have order 6
  **Clarification**: Z_6 is abelian and cyclic; D_3 is non-abelian and has a different structure
