# Quick Definition

The dihedral group D_n is the rotational symmetry group of a regular n-sided plate. It has 2n elements: n rotations about the axis perpendicular to the plate and n rotations through pi about axes in the plane of the plate.

# Core Definition

"The dihedral group D_n is the rotational symmetry group of a flat plate with n equal sides" (Armstrong, Ch. 4, p. 24). Its elements are e, r, r^2, ..., r^{n-1}, s, rs, r^2s, ..., r^{n-1}s, where r is rotation through 2pi/n about the perpendicular axis and s is rotation through pi about an axis in the plane. The relations r^n = e, s^2 = e, and sr = r^{-1}s determine all products.

# Key Properties

1. Has order 2n (contains 2n elements)
2. Generated by two elements r and s with r^n = e, s^2 = e, sr = r^{-1}s
3. Non-abelian for n >= 3
4. Elements of the form r^a are rotations preserving the plate's orientation
5. Elements of the form r^a*s are rotations flipping the plate
6. The relation sr = r^{-1}s is the key computational tool
7. Products are computed using: r^a * r^b = r^{a+_n b}, r^a * (r^b*s) = r^{a+_n b}*s, (r^a*s) * r^b = r^{a+_n(n-b)}*s, (r^a*s) * (r^b*s) = r^{a+_n(n-b)}

# Construction / Recognition

## To Construct D_n:
1. Take a regular n-sided plate
2. Let r = rotation through 2pi/n about the perpendicular axis
3. Let s = rotation through pi about an axis in the plane of the plate
4. List elements: e, r, r^2, ..., r^{n-1}, s, rs, r^2s, ..., r^{n-1}s
5. Compute products using r^n = e, s^2 = e, sr = r^{-1}s

## To Compute a Multiplication Table:
1. Use sr = r^{-1}s = r^{n-1}s to move all s's to the right
2. Reduce powers of r modulo n
3. Fill in the 2n x 2n table using the four product formulas

# Context & Application

The dihedral groups are the first family of non-abelian groups presented in the text. Armstrong uses D_3 (the triangular plate) as the detailed example, working out its complete multiplication table. The dihedral groups illustrate how a group can be described by generators and relations.

# Examples

**Example 1** (p. 22): D_3 has six elements: e, r, r^2, s, rs, r^2s. The relations are r^3 = e, s^2 = e, sr = r^2s.

**Example 2** (p. 23): Computing products in D_3: r(r^2s) = r^3s = es = s, and (r^2s)(rs) = r.

**Example 3** (p. 24): The complete multiplication table of D_3 is given (6 x 6 table). Each element appears exactly once in every row and column.

# Common Errors

- **Error**: Reading rs as "first r, then s"
  **Correction**: rs means "first s, then r" (right-to-left convention)

- **Error**: Forgetting the relation sr = r^{-1}s when computing products
  **Correction**: Always use sr = r^{n-1}s to move s to the right before reducing

# Common Confusions

- **Confusion**: Thinking D_n has n elements
  **Clarification**: D_n has 2n elements; the subscript n refers to the number of sides of the plate, not the order of the group

- **Confusion**: Thinking all elements of D_n have the same type
  **Clarification**: Elements r^a are orientation-preserving rotations; elements r^a*s involve flipping the plate
