# Quick Definition

The additive group of complex numbers, denoted C, is the set of all complex numbers with addition as the group operation. The identity is 0 and the inverse of z is -z.

# Core Definition

"Addition of numbers (real or complex) makes each of the following sets into a group: ... C, the set of complex numbers" (Armstrong, Ch. 3, p. 18). The group is abelian and has infinite order. Z < Q < R < C is the standard chain of subgroups.

# Prerequisites

- **Group** — C is verified to satisfy the group axioms under addition

# Key Properties

1. The group operation is addition of complex numbers
2. The identity element is 0
3. The inverse of z is -z
4. The group is abelian
5. R < C (subgroup relationship)

# Construction / Recognition

## To Verify:
1. The sum of two complex numbers is complex (closure)
2. Addition is associative
3. 0 is the identity
4. -z is the inverse of z

# Context & Application

C under addition is the largest of Armstrong's standard additive number groups. The subgroup of Gaussian integers (generated by {1, i}) is discussed in Chapter 5.

# Examples

**Example 1** (p. 18): C forms a group under addition, completing the chain Z < Q < R < C.

# Common Errors

- **Error**: Confusing the additive group C with the multiplicative group C - {0}
  **Correction**: Under addition, every complex number has an inverse (-z); under multiplication, only non-zero complex numbers form a group

# Common Confusions

- **Confusion**: Confusing C (the additive group of all complex numbers) with the unit circle group (also denoted C by Armstrong)
  **Clarification**: Armstrong uses C for the unit circle group under multiplication (|z| = 1) and C for all complex numbers under addition; context distinguishes them
