# Quick Definition

A group is finitely generated if it has a finite set of generators. Finitely generated abelian groups can be completely classified up to isomorphism.

# Core Definition

A group is **finitely generated** if it has a finite set of generators. Finitely generated abelian groups may be classified: one can draw up a list of "standard" examples, no two isomorphic, so that any finitely generated abelian group is isomorphic to one on the list (Armstrong, p. 126).

# Prerequisites

- **Abelian group** -- the group operation is commutative
- **Direct product** -- the classification theorem expresses f.g. abelian groups as direct products

# Key Properties

1. Every element can be written as $x_1^{n_1} x_2^{n_2} \cdots x_r^{n_r}$ for generators $x_1, \ldots, x_r$ and integers $n_i$
2. Generators may satisfy relations of the form $x_1^{n_1} \cdots x_r^{n_r} = e$
3. A minimal set of generators is one where no proper subset generates the group
4. Changing generators (e.g., replacing $x_1$ by $x_1 x_2^q$) preserves the group

# Context & Application

The classification of finitely generated abelian groups is one of the fundamental structure theorems in algebra. Armstrong notes that $\mathbb{R}$ is not finitely generated (Exercise 21.12), so this classification does not apply to all abelian groups.

# Examples

**Example (iii)** (p. 127): The subgroup of $\mathbb{C} \setminus \{0\}$ generated by $-1$ and $i/2$ is isomorphic to $\mathbb{Z}_2 \times \mathbb{Z}$.

**Example (iv)** (p. 127): $\mathbb{R}$ is not finitely generated, so it does not appear on the classification list.
