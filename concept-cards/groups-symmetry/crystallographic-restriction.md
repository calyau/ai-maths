# Quick Definition

The crystallographic restriction states that the order of a rotation in a wallpaper group can only be 2, 3, 4, or 6. Five-fold and higher-than-six-fold rotational symmetry are incompatible with a periodic lattice.

# Core Definition

"(25.3) Theorem. The order of a rotation in a wallpaper group can only be 2, 3, 4, or 6" (Armstrong, Ch. 25, p. 157). The proof uses the fact that the point group acts on the lattice: if a rotation of order q > 6 existed, then f_A(a) - a would be a lattice vector shorter than a (the shortest lattice vector), a contradiction. For q = 5, the vector f_A^2(a) + a provides the contradiction.

# Prerequisites

- **Lattice** -- The proof depends on the lattice having a shortest non-zero vector
- **Point group** -- The point group contains the rotations under consideration
- **Wallpaper group** -- The setting for the theorem

# Key Properties

1. Allowed rotation orders: 1, 2, 3, 4, 6
2. Forbidden rotation orders: 5, 7, 8, 9, ...
3. The proof is geometric: it uses length comparisons in the lattice
4. Corollary (25.4): The point group is generated by a rotation through one of the angles 0, pi, 2pi/3, pi/2, pi/3 and possibly a reflection

# Construction / Recognition

## Proof Strategy (Armstrong's proof)
1. Let A be the rotation matrix of angle 2pi/q in the point group J
2. Since J acts on L, f_A(a) is in L where a is a shortest non-zero lattice vector
3. For q > 6: the angle 2pi/q < 60 degrees, so ||f_A(a) - a|| < ||a||, contradicting minimality of a
4. For q = 5: the angle between f_A^2(a) and -a is 36 degrees, so ||f_A^2(a) + a|| < ||a||, contradiction
5. Therefore q can only be 2, 3, 4, or 6

# Context & Application

The crystallographic restriction is one of the most important constraints in the wallpaper classification. It explains why we never see 5-fold or 7-fold symmetric repeating patterns in wallpaper, tiling, or crystallography. It limits the point group to one of finitely many subgroups of O_2, which makes the complete classification of wallpaper groups feasible.

**Note:** Quasicrystals (discovered in 1984) exhibit 5-fold diffraction symmetry but are not periodic -- they do not have a wallpaper group structure.

# Common Errors

- **Error**: Applying the crystallographic restriction to non-periodic patterns
  **Correction**: The restriction applies only to wallpaper groups (periodic patterns); non-periodic tilings like Penrose tilings can have 5-fold symmetry

# Common Confusions

- **Confusion**: Thinking the restriction forbids all rotations except 60, 90, 120, 180 degrees
  **Clarification**: The restriction says rotation *orders* in the point group must be 1, 2, 3, 4, or 6. The allowed angles are 2pi/q for these q values, plus their multiples
