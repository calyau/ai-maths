# Quick Definition

D_3 is the dihedral group of order 6, the rotational symmetry group of the equilateral triangular plate. Its six elements are e, r, r^2, s, rs, r^2s, with relations r^3 = e, s^2 = e, sr = r^2s.

# Core Definition

D_3 is the detailed worked example Armstrong uses to illustrate dihedral groups. "When n is three we have a triangular plate. It has six rotational symmetries and, if r and s are the rotations shown in Figure 4.1, they are e, r, r^2, s, rs, r^2s" (Armstrong, Ch. 4, p. 22). The relations r^3 = e, s^2 = e, and sr = r^2s determine all products.

# Key Properties

1. Order 6 (six elements)
2. Non-abelian (e.g., rs != sr)
3. Two elements of order 3: r, r^2
4. Three elements of order 2: s, rs, r^2s
5. Generated by r and s
6. Key relation: sr = r^2s (equivalently, sr = r^{-1}s)
7. Complete multiplication table is 6 x 6

# Construction / Recognition

## To Compute Products:
1. Use r^3 = e, s^2 = e, sr = r^2s
2. Move all s factors to the right
3. Reduce powers of r modulo 3
4. Result is either r^a or r^a*s with 0 <= a <= 2

## Worked Example:
(r^2s)(rs) = r^2(s(rs)) = r^2((sr)s) = r^2((r^2s)s) = r^2(r^2s^2) = r^4s^2 = r*e = r

# Context & Application

D_3 is Armstrong's primary computational example for dihedral groups. It is the smallest non-abelian group (along with S_3, to which it is isomorphic). Its multiplication table appears in full on p. 24.

# Examples

**Example 1** (p. 22): sr = r^2s, verified geometrically and algebraically.

**Example 2** (p. 23): sr^2 = rs, computed algebraically: sr^2 = s(rr) = (sr)r = (r^2s)r = r^2(sr) = r^2(r^2s) = r^4s = rs.

**Example 3** (p. 23): r(r^2s) = r^3s = es = s.

# Common Errors

- **Error**: Computing sr as rs instead of r^2s
  **Correction**: sr = r^2s = r^{-1}s, not rs; the s "moves through" r and inverts it

# Common Confusions

- **Confusion**: Thinking D_3 is the same as Z_6 because both have order 6
  **Clarification**: D_3 is non-abelian with elements of orders 1, 2, 3; Z_6 is abelian and cyclic with elements of orders 1, 2, 3, 6
