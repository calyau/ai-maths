# Quick Definition

The subgroup generated by an element x of a group G, denoted <x>, is the set of all powers of x: {..., x^{-2}, x^{-1}, e, x, x^2, ...}. It is the smallest subgroup of G containing x.

# Core Definition

"Given a group G and an element x of G, the set of all powers of x (i.e., the set of all elements of G of the form x^n for some integer n) is a subgroup of G" (Armstrong, Ch. 5, p. 29). Armstrong verifies: x^m * x^n = x^{m+n} is a power of x (closure), x^0 = e is in the set (identity), and x^{-n} is the inverse of x^n (inverses). "This subgroup is called the subgroup generated by x and written <x>."

# Prerequisites

- **Subgroup** — <x> is a subgroup
- **Power of an Element** — Elements of <x> are powers of x
- **Order of an Element** — Determines the size of <x>

# Key Properties

1. <x> = {x^n : n in Z}
2. If x has infinite order, <x> = {..., x^{-2}, x^{-1}, e, x, x^2, ...} (all distinct)
3. If x has finite order m, <x> = {e, x, x^2, ..., x^{m-1}} has exactly m elements
4. The order of <x> equals the order of x
5. <x> is the smallest subgroup of G containing x
6. <x> is always abelian (cyclic groups are abelian)

# Construction / Recognition

## To Construct <x>:
1. Compute x, x^2, x^3, ... until a repeat occurs (finite order) or establish infinite order
2. If x^m = e, then <x> = {e, x, x^2, ..., x^{m-1}}
3. If no power equals e, then <x> = {..., x^{-2}, x^{-1}, e, x, x^2, ...}

# Context & Application

The subgroup generated by an element is the fundamental building block for cyclic groups and a key tool for analyzing group structure. Every element generates a cyclic subgroup, and listing all such subgroups is a first step in finding all subgroups of a finite group.

# Examples

**Example 1** (p. 29): In Z_6: <0> = {0}, <1> = <5> = Z_6, <2> = <4> = {0, 2, 4}, <3> = {0, 3}.

**Example 2** (p. 29): In D_3: <e> = {e}, <r> = <r^2> = {e, r, r^2}, <s> = {e, s}, <rs> = {e, rs}, <r^2s> = {e, r^2s}.

**Example 3** (p. 29): In Z, the element 1 generates all of Z: <1> = Z. Similarly <-1> = Z.

# Relationships

- **Cyclic Group** — A group G is cyclic if G = <x> for some x

## Related

# Common Errors

- **Error**: Forgetting negative powers when x has infinite order
  **Correction**: <x> includes all x^n for n in Z, including negative n

# Common Confusions

- **Confusion**: Thinking <x> always has the same order as the group
  **Clarification**: <x> may be a proper subgroup; its order equals the order of x, not the order of G
