# Quick Definition

A cyclic group is a group that can be generated by a single element. That is, G is cyclic if there exists an element x in G such that every element of G is a power of x: G = <x>.

# Core Definition

A group G is cyclic if G = <x> for some element x, meaning every element of G can be written as x^n for some integer n. Armstrong introduces this implicitly through the subgroup generated by an element and the observation that Z = <1> and Z_n = <1> (Armstrong, Ch. 5, pp. 29-30). The infinite cyclic group is Z, and the finite cyclic group of order n is Z_n.

# Prerequisites

- **Group** — A cyclic group is a group
- **Subgroup Generated by Element** — A cyclic group equals <x> for some x

# Key Properties

1. Generated by a single element x: G = <x>
2. Every cyclic group is abelian (x^m * x^n = x^{m+n} = x^n * x^m)
3. A finite cyclic group of order n is isomorphic to Z_n
4. An infinite cyclic group is isomorphic to Z
5. Every subgroup of a cyclic group is cyclic (Theorem 5.3)
6. The number of generators of Z_n equals phi(n) (Euler's totient)

# Construction / Recognition

## To Recognize:
1. Find an element x whose powers exhaust the entire group
2. If such x exists, G is cyclic with generator x
3. In Z_n, the element k generates Z_n if and only if gcd(k, n) = 1

## To Construct:
1. For finite cyclic group of order n: use Z_n = {0, 1, ..., n-1} under addition mod n
2. For infinite cyclic group: use Z under addition

# Context & Application

Cyclic groups are the simplest type of group. Every group contains cyclic subgroups (generated by individual elements). Understanding cyclic groups is prerequisite to understanding more complex group structure. Armstrong notes that D_n is NOT cyclic for n >= 3.

# Examples

**Example 1** (p. 29): Z is an infinite cyclic group generated by 1 (or by -1). Since the group operation is addition, "powers" are multiples: the nth "power" of 1 is n.

**Example 2** (p. 29): Z_n is a cyclic group of order n generated by 1.

**Example 3** (p. 29): In Z_6, both 1 and 5 generate the entire group; 2 and 4 generate only the subgroup {0, 2, 4}.

**Example 4** (p. 30): D_n is not cyclic because no single element generates all of D_n.

# Common Errors

- **Error**: Assuming a group of prime order might not be cyclic
  **Correction**: Every group of prime order is cyclic (follows from Lagrange's theorem)

# Common Confusions

- **Confusion**: Thinking "cyclic" means the group has a circular structure
  **Clarification**: "Cyclic" means generated by one element; the term comes from the cyclic repetition of powers

- **Confusion**: Thinking abelian implies cyclic
  **Clarification**: Every cyclic group is abelian, but not every abelian group is cyclic (e.g., Z_2 x Z_2 is abelian but not cyclic)
