# Quick Definition

The subgroup generated by a non-empty subset X of a group G is the collection of all words in the elements of X. It is the smallest subgroup of G containing X.

# Core Definition

"Let X be a non-empty subset of a group G. ... The collection of all [words in the elements of X] is a subgroup of G. ... This subgroup is called the subgroup generated by X" (Armstrong, Ch. 5, p. 30). Armstrong verifies: the product of two words is a word (closure), the identity is the word x^0 (identity), and the inverse of a word x_1^{m_1}...x_k^{m_k} is x_k^{-m_k}...x_1^{-m_1} (inverses).

# Prerequisites

- **Subgroup** — The generated set is a subgroup
- **Word in Generators** — Elements are words in X

# Key Properties

1. <X> consists of all words x_1^{m_1} x_2^{m_2} ... x_k^{m_k} where x_i are in X and m_i are integers
2. <X> is a subgroup of G (verified by checking closure, identity, inverses)
3. <X> is the smallest subgroup containing X
4. If X = {x}, this reduces to the cyclic subgroup <x>
5. If <X> = G, then X is a set of generators for G

# Construction / Recognition

## To Construct <X>:
1. Start with elements of X and their inverses
2. Form all possible products (words) of these elements
3. The resulting collection is <X>

# Context & Application

This generalizes the subgroup generated by a single element to arbitrary subsets. It is the formal foundation for the concept of generators: X generates G means <X> = G.

# Examples

**Example 1** (p. 30): {r, s} generates D_n: <{r, s}> = D_n.

**Example 2** (p. 30): {rs, s} also generates D_n because r = (rs)s, so every word in r, s can be rewritten as a word in rs, s.

**Example 3** (p. 30): In (C, +), the subgroup generated by {1, i} is the Gaussian integers {a + ib : a, b in Z}.

# Common Errors

- **Error**: Forgetting that words can use elements of X multiple times and in any order
  **Correction**: Words are arbitrary products x_1^{m_1}...x_k^{m_k} where x_i can repeat

# Common Confusions

- **Confusion**: Thinking <X> consists only of elements of X themselves
  **Clarification**: <X> includes all products and powers of elements of X, which may yield many elements not in X
