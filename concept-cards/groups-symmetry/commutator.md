# Quick Definition

A commutator in a group $G$ is an element of the form $xyx^{-1}y^{-1}$ for $x, y \in G$. The commutator equals the identity precisely when $x$ and $y$ commute.

# Core Definition

An element of $G$ of the form $xyx^{-1}y^{-1}$ is called a **commutator** (Armstrong, Ch. 15, p. 90). Since $x$ and $y$ commute precisely when the commutator $xyx^{-1}y^{-1}$ is the identity, the set of commutators measures how far $G$ is from being abelian.

# Prerequisites

- **Group** — Commutators use the group operation and inverses

# Key Properties

1. $xyx^{-1}y^{-1} = e$ iff $xy = yx$
2. Every commutator is a product of group elements and their inverses
3. The set of all commutators generates the commutator subgroup $[G,G]$
4. Any conjugate of a commutator is again a commutator: $g(xyx^{-1}y^{-1})g^{-1} = (gxg^{-1})(gyg^{-1})(gxg^{-1})^{-1}(gyg^{-1})^{-1}$

# Construction / Recognition

## To Construct:
1. Choose $x, y \in G$
2. Compute $xyx^{-1}y^{-1}$

# Context & Application

Commutators are the building blocks for the commutator subgroup $[G,G]$. Armstrong uses them to define abelianisation and to compute commutator subgroups of specific groups ($S_n$, $D_n$, $Q$).

# Examples

**Example 1** (p. 90): In $S_n$, every commutator is an even permutation (both $xyx^{-1}y^{-1}$ and the identity are even).

**Example 2** (p. 91): In $D_n$, $rsr^{-1}s^{-1} = rrss = r^2$, showing that $r^2$ is a commutator.

# Relationships

## Enables
- **Commutator subgroup** — Generated by all commutators

## Related
- **Abelian group** — In an abelian group, all commutators are trivial

# Common Errors

- **Error**: Assuming every element of $[G,G]$ is a single commutator
  **Correction**: Elements of $[G,G]$ are products of commutators; not every element of $[G,G]$ need be a commutator itself.

# Common Confusions

- **Confusion**: Confusing the commutator $xyx^{-1}y^{-1}$ with the conjugate $xyx^{-1}$
  **Clarification**: The commutator involves $y^{-1}$ as well. $xyx^{-1}$ is a conjugate of $y$; $xyx^{-1}y^{-1}$ measures the failure of $x$ and $y$ to commute.
