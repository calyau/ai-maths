# Quick Definition

$R_n$ is the collection of all integers $m$ satisfying $1 \le m \le n-1$ with $\gcd(m, n) = 1$, forming a group under multiplication modulo $n$.

# Core Definition

Let $n$ be a positive integer. The collection $R_n$ consists of all integers $m$ satisfying $1 \le m \le n-1$ for which the highest common factor of $m$ and $n$ is 1. Armstrong proves that multiplication modulo $n$ makes $R_n$ into a group (Ch. 11, p. 65). The group is abelian. Its order is $\varphi(n)$, where $\varphi$ is Euler's phi-function.

**Closure:** If $m_1, m_2 \in R_n$, then $\gcd(m_1 m_2, n) = 1$, so $m_1 m_2 \pmod{n} \in R_n$. **Identity:** The integer 1. **Inverses:** If $m \in R_n$, there exist integers $x, y$ with $xm + yn = 1$; reading modulo $n$ gives the inverse $x \pmod{n}$.

# Prerequisites

- **Group** — $R_n$ is verified to satisfy the group axioms
- **Modular arithmetic** — Multiplication is performed modulo $n$

# Key Properties

1. $R_n$ is abelian
2. $|R_n| = \varphi(n)$ (Euler's phi-function)
3. $R_p$ is cyclic when $p$ is prime (Exercise 11.12)
4. $R_n$ is not always cyclic (e.g., $R_{16} \cong \mathbb{Z}_4 \times \mathbb{Z}_2$)

# Construction / Recognition

## To Construct:
1. Fix a positive integer $n$
2. List all integers $m$ with $1 \le m \le n-1$ and $\gcd(m, n) = 1$
3. Define multiplication as $m_1 \cdot m_2 = m_1 m_2 \pmod{n}$

# Context & Application

Armstrong introduces $R_n$ to answer a question from Chapter 3 about when modular multiplication produces a group. The group $R_n$ is the concrete example on which Euler's theorem and Fermat's little theorem are demonstrated as applications of Lagrange's theorem.

# Examples

**Example 1** (p. 65): $R_9 = \{1, 2, 4, 5, 7, 8\}$ with $\varphi(9) = 6$. Since $2^2 = 4$, $2^3 = 8$, $2^4 \equiv 7$, $2^5 \equiv 5 \pmod{9}$, the group $R_9$ is cyclic generated by 2.

**Example 2** (p. 65): $R_{16} = \{1, 3, 5, 7, 9, 11, 13, 15\}$ with $\varphi(16) = 8$. With $H = \langle 3 \rangle = \{1, 3, 9, 11\}$ and $K = \langle 15 \rangle = \{1, 15\}$, we get $R_{16} \cong H \times K \cong \mathbb{Z}_4 \times \mathbb{Z}_2$.

# Common Errors

- **Error**: Including 0 or $n$ in $R_n$
  **Correction**: $R_n$ consists only of integers $1 \le m \le n-1$ coprime to $n$

# Common Confusions

- **Confusion**: Thinking $R_n$ is always cyclic
  **Clarification**: $R_n$ is cyclic when $n$ is 1, 2, 4, $p^k$, or $2p^k$ for odd primes $p$, but not in general (e.g., $R_{16} \cong \mathbb{Z}_4 \times \mathbb{Z}_2$)
