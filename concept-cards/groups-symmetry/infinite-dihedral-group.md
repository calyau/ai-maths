# Quick Definition

The infinite dihedral group D_infinity is the group of distance-preserving functions on the real line that send the integers to themselves. It is generated by a translation t and a reflection s, with relations s^2 = e and st = t^{-1}s, paralleling the finite dihedral groups.

# Core Definition

Armstrong defines D_infinity as the group G of functions from the real line to itself that preserve distance and send the integers among themselves. Its elements are "...t^{-2}, t^{-1}, e, t, t^2, ... and ...t^{-2}s, t^{-1}s, s, ts, t^2s, ..." where t is translation right by one unit (t(x) = x + 1) and s is reflection in the origin (s(x) = -x). The relation st = t^{-1}s holds, "and the only difference [from D_n] is that the rotation r of order n has been replaced by the translation t of infinite order. For this reason we call G the infinite dihedral group and denote it by D_infinity" (Armstrong, Ch. 5, p. 31).

# Key Properties

1. Elements: ..., t^{-2}, t^{-1}, e, t, t^2, ... and ..., t^{-2}s, t^{-1}s, s, ts, t^2s, ...
2. t has infinite order (translation, never returns to identity)
3. s has order 2 (s^2 = e)
4. Key relation: st = t^{-1}s (same form as sr = r^{-1}s in D_n)
5. Generated by {t, s} or equivalently by {ts, s} (two reflections)
6. Non-abelian
7. Elements t^n are translations; elements t^n*s are reflections

# Construction / Recognition

## To Construct:
1. Let t(x) = x + 1 (translation right by 1)
2. Let s(x) = -x (reflection in the origin)
3. Verify st = t^{-1}s: st(x) = s(x+1) = -x-1 and t^{-1}s(x) = t^{-1}(-x) = -x-1
4. List elements as t^n and t^n*s for n in Z

# Context & Application

D_infinity extends the dihedral group concept to infinite order. It shows how the same algebraic structure (generators with relation st = t^{-1}s) can produce either finite or infinite groups depending on whether the "rotation" element has finite or infinite order.

# Examples

**Example 1** (p. 31): t(x) = x + 1 is translation right by 1; t^{-2}(x) = x - 2 is translation left by 2.

**Example 2** (p. 31): ts(x) = t(-x) = -x + 1, so ts is reflection in the point 1/2.

**Example 3** (p. 31): st(x) = s(x+1) = -x-1 and t^{-1}s(x) = t^{-1}(-x) = -x-1, confirming st = t^{-1}s.

# Common Errors

- **Error**: Assuming D_infinity has a rotation element of finite order
  **Correction**: t has infinite order; that is precisely the distinction from finite D_n

# Common Confusions

- **Confusion**: Thinking D_infinity acts on a polygon
  **Clarification**: D_infinity acts on the real line with marked integer points, not on a polygon
