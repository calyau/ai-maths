# Quick Definition

A glide reflection is a reflection in a line followed by a non-zero translation parallel to that line. It is an opposite isometry with no fixed points and infinite order.

# Core Definition

"A reflection in a line followed by a translation parallel to the same line is called a glide reflection" (Armstrong, Ch. 24, p. 146). In ordered pair notation, a glide reflection along a line m has the form (2a + b, B) where B is the matrix of reflection in the parallel line through the origin, a is perpendicular to m pointing from the origin, f_B(b) = b (so b is parallel to m), and b is non-zero.

# Prerequisites

- **Reflection** -- A glide reflection combines a reflection with a parallel translation
- **Translation** -- The translation component is parallel to the mirror line
- **Opposite isometry** -- A glide reflection is an opposite isometry

# Key Properties

1. Composition of a reflection and a translation parallel to the reflection axis
2. Has no fixed points (unlike a pure reflection)
3. Has infinite order
4. The square of a glide reflection is a translation (by 2b along the axis)
5. In ordered pair notation: (2a + b, B) where f_B(b) = b and b is non-zero

# Construction / Recognition

## To Construct a Glide Reflection
1. Choose a line m (the glide axis)
2. Choose a non-zero translation vector b parallel to m
3. Reflect in m, then translate by b

## To Recognize from Ordered Pair (v, B)
1. Verify det(B) = -1 (opposite isometry)
2. Compute f_B(v)
3. If f_B(v) is not equal to -v, it is a glide reflection
4. Decompose: w = v - f_B(v), 2a = (v.w/||w||^2)w, b = v - 2a

# Context & Application

Glide reflections play a crucial role in the classification of wallpaper groups. Several wallpaper groups (pg, p2mg, p2gg, p4gm) are distinguished from their counterparts by the presence of glide reflections rather than pure reflections. The notation "g" in wallpaper group names stands for glide reflection.

# Examples

**Example 1** (p. 147): The isometry h(x,y) = (-(x + sqrt(3)y)/2, 4 + (y - sqrt(3)x)/2) is a glide reflection along the line sqrt(3)x + y = 2 with glide vector b = (-sqrt(3), 3).

**Example 2** (Ch. 25, p. 155): The wallpaper group pg is generated by translations and glide reflections along parallel horizontal lines with glide length a/2.

# Relationships

## Builds Upon
- **Opposite isometry** -- A glide reflection is an opposite isometry

## Related
- **Wallpaper group** -- Glide reflections distinguish pg from pm, p2gg from p2mm, etc.

## Contrasts With
- **Reflection** -- A reflection has a fixed line; a glide reflection has no fixed points
- **Translation** -- A translation preserves orientation; a glide reflection reverses it

# Common Errors

- **Error**: Thinking a glide reflection can be decomposed as reflection then translation in any order
  **Correction**: Since the translation is parallel to the mirror, the two operations commute in this case, so the order does not matter for the same axis

- **Error**: Forgetting that the square of a glide reflection is a translation
  **Correction**: Applying the glide twice: the reflection cancels out and the translations add, giving translation by twice the glide vector

# Common Confusions

- **Confusion**: Confusing glide reflections with the composition of a reflection and an arbitrary translation
  **Clarification**: The translation component must be parallel to the mirror line; a perpendicular component would give a reflection in a different line
