# Quick Definition

A finite subgroup of $O_2$ is either cyclic or dihedral. This is the two-dimensional precursor to the classification of finite rotation groups in three dimensions.

# Core Definition

**(19.1) Theorem.** A finite subgroup of $O_2$ is either cyclic or dihedral (Armstrong, p. 111).

# Prerequisites

- **Group action** -- the proof analyses how the group acts on the plane

# Key Properties

1. If $G \le O_2$ is finite and $G \subseteq SO_2$, then $G$ is cyclic
2. If $G \le O_2$ is finite and $G \not\subseteq SO_2$, then $H = G \cap SO_2$ is cyclic of index 2 in $G$, and $G$ is dihedral
3. $SO_2$ consists of rotations; elements of $O_2 \setminus SO_2$ are reflections

# Construction / Recognition

## Proof Sketch (p. 111):
1. If $G \subseteq SO_2$: choose $A_\varphi \in G$ with $\varphi$ positive and minimal. Any $A_\theta \in G$ satisfies $\theta = k\varphi$ (by a division-algorithm argument), so $G = \langle A_\varphi \rangle$ is cyclic.
2. If $G \not\subseteq SO_2$: set $H = G \cap SO_2$, which has index 2. By part 1, $H$ is cyclic, generated by $A$. Choose $B \in G \setminus H$; then $B^2 = I$ (reflection). If $A = I$, then $G \cong \mathbb{Z}_2$. Otherwise the $2n$ elements $I, A, \ldots, A^{n-1}, B, AB, \ldots, A^{n-1}B$ satisfy $A^n = I$, $B^2 = I$, $BA = A^{-1}B$, giving $G \cong D_n$.

# Context & Application

This theorem is a stepping stone to the classification of finite subgroups of $SO_3$ (Theorem 19.2). The argument for the cyclic case (minimal rotation generates the group) is a model for the more complex three-dimensional analysis.

# Examples

Cyclic groups $\mathbb{Z}_n$ (rotational symmetry of a regular $n$-gon) and dihedral groups $D_n$ (full symmetry of a regular $n$-gon) exhaust all possibilities.
