# Quick Definition

The additive group of rationals, denoted Q, is the set of all rational numbers with addition as the group operation. The identity is 0 and the inverse of x is -x.

# Core Definition

"Addition of numbers (real or complex) makes each of the following sets into a group: ... Q, the set of rational numbers" (Armstrong, Ch. 3, p. 18). The group is abelian and has infinite order. It is not cyclic (Exercise 5.11).

# Prerequisites

- **Group** — Q is verified to satisfy the group axioms under addition

# Key Properties

1. The group operation is addition
2. The identity element is 0
3. The inverse of x is -x
4. The group is abelian
5. Z is a subgroup of Q
6. Q is a subgroup of R
7. Q is not cyclic and cannot be generated by finitely many elements (Exercise 5.11)

# Construction / Recognition

## To Verify:
1. The sum of two rationals is rational (closure)
2. Addition is associative
3. 0 is rational and acts as identity
4. For every rational x, -x is rational

# Context & Application

Q under addition is one of Armstrong's standard examples of an infinite abelian group. It sits between Z and R in the chain of subgroups Z < Q < R < C.

# Examples

**Example 1** (p. 18): Q forms a group under addition with identity 0 and inverse -x.

**Example 2** (Exercise 5.11, p. 33): Q is not cyclic, and cannot even be generated by finitely many elements.

# Common Errors

- **Error**: Assuming Q is cyclic because Z is cyclic
  **Correction**: Q is not cyclic; no single rational number generates all of Q by repeated addition

# Common Confusions

- **Confusion**: Confusing (Q, +) with the multiplicative structure Q - {0}
  **Clarification**: Q under addition and Q - {0} under multiplication are different groups
