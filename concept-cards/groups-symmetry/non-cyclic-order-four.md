# Quick Definition

There are exactly two groups of order 4 up to isomorphism: the cyclic group $\mathbb{Z}_4$ and Klein's group $\mathbb{Z}_2 \times \mathbb{Z}_2$.

# Core Definition

Exercise 10.12 (p. 63): "If $G$ is a group of order 4 which is not cyclic, prove that $G$ is isomorphic to Klein's group." The proof idea: if $G$ is not cyclic, no element has order 4, so every non-identity element has order 2. Using three such elements, the group structure is forced to match $\mathbb{Z}_2 \times \mathbb{Z}_2$.

# Prerequisites

- **Direct product** — Klein's group is $\mathbb{Z}_2 \times \mathbb{Z}_2$
- **Klein's group** — One of the two groups of order 4

# Key Properties

1. There are exactly two groups of order 4 (up to isomorphism)
2. $\mathbb{Z}_4$: cyclic, has an element of order 4
3. $\mathbb{Z}_2 \times \mathbb{Z}_2$ (Klein's group): non-cyclic, every non-identity element has order 2
4. Both are abelian

# Construction / Recognition

## To Classify a Group of Order 4:
1. Check if it has an element of order 4
2. If yes: it is isomorphic to $\mathbb{Z}_4$
3. If no: every non-identity element has order 2, and the group is isomorphic to $\mathbb{Z}_2 \times \mathbb{Z}_2$

# Context & Application

This is the simplest non-trivial classification result. It shows that the direct product construction, combined with the cyclic group classification, gives a complete picture at order 4. The classification at higher orders becomes much more complex.

# Examples

**Example**: $\{1, i, -1, -i\}$ under complex multiplication is isomorphic to $\mathbb{Z}_4$ (generated by $i$).

**Example**: The subgroup $\{\varepsilon, (12)(34), (13)(24), (14)(23)\}$ of $S_4$ is isomorphic to Klein's group.

# Common Errors

- **Error**: Thinking there might be a third group of order 4.
  **Correction**: There are exactly two. By Lagrange's theorem, elements have order 1, 2, or 4. If any element has order 4, the group is cyclic. Otherwise all non-identity elements have order 2, forcing Klein's group.

# Common Confusions

- **Confusion**: Thinking both groups of order 4 are cyclic.
  **Clarification**: $\mathbb{Z}_4$ is cyclic; $\mathbb{Z}_2 \times \mathbb{Z}_2$ is not.
