# Quick Definition

A group of order 6 is isomorphic to either $\mathbb{Z}_6$ (cyclic) or $D_3$ (dihedral). There are no other possibilities.

# Core Definition

**(13.2) Theorem.** A group of order 6 is either isomorphic to $\mathbb{Z}_6$ or isomorphic to $D_3$ (Armstrong, p. 76).

**Proof outline.** Let $G$ have six elements. By Cauchy's theorem, choose $x$ of order 3 and $y$ of order 2. The cosets $\langle x \rangle$, $\langle x \rangle y$ give six elements $e, x, x^2, y, xy, x^2 y$. Now $yx$ must be one of these. If $yx = xy$, then $G \cong \langle x \rangle \times \langle y \rangle \cong \mathbb{Z}_3 \times \mathbb{Z}_2 \cong \mathbb{Z}_6$. Otherwise $yx = x^2 y$, giving $G \cong D_3$.

# Prerequisites

- **Cauchy's theorem** — Guarantees elements of order 2 and 3
- **Dihedral group** — One of the two outcomes
- **Cyclic group** — The other outcome
- **Direct product** — $\mathbb{Z}_6 \cong \mathbb{Z}_3 \times \mathbb{Z}_2$

# Key Properties

1. There are exactly two groups of order 6 up to isomorphism
2. $\mathbb{Z}_6$ is abelian; $D_3 \cong S_3$ is non-abelian
3. The key distinction is whether the elements of order 2 and 3 commute
4. Armstrong notes that any group of order $2p$ (odd prime $p$) is cyclic or dihedral (proved in Ch. 15)

# Construction / Recognition

## To Classify a Group of Order 6:
1. Find an element $x$ of order 3 (exists by Cauchy)
2. Find an element $y$ of order 2 (exists by Cauchy)
3. Check whether $xy = yx$
4. If yes: $G \cong \mathbb{Z}_6$
5. If no: $G \cong D_3$

# Context & Application

This is the first application of Cauchy's theorem in the text. It demonstrates the power of knowing that elements of specified prime orders exist. Combined with Lagrange's theorem constraints, this classifies all groups of order 6.

# Examples

**Example 1** (p. 76): $\mathbb{Z}_6$ is cyclic, generated by any element of order 6. The group is abelian.

**Example 2** (p. 76): $D_3 \cong S_3$ satisfies $yx = x^2 y$ (equivalently, $sr = r^2 s$ in dihedral notation).

# Common Errors

- **Error**: Forgetting to rule out the case $yx = xy$ when it holds
  **Correction**: If $x$ and $y$ commute, then $G$ is abelian and isomorphic to $\mathbb{Z}_6$, not $D_3$

# Common Confusions

- **Confusion**: Thinking there might be more groups of order 6
  **Clarification**: Armstrong's proof is exhaustive. The only possibilities for $yx$ that avoid contradiction are $xy$ (giving $\mathbb{Z}_6$) and $x^2 y$ (giving $D_3$).
