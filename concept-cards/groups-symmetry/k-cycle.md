# Quick Definition

A $k$-cycle is a cyclic permutation of length $k$ — it cyclically permutes exactly $k$ elements and fixes the rest.

# Core Definition

"The number $k$ is its length and a cyclic permutation of length $k$ is called a **$k$-cycle**" (Armstrong, p. 34). A $k$-cycle $(a_1 a_2 \ldots a_k)$ has order $k$ and can be decomposed as the product of $k - 1$ transpositions: $(a_1 a_k) \ldots (a_1 a_3)(a_1 a_2)$.

# Prerequisites

- **Cyclic permutation** — A $k$-cycle is a cyclic permutation classified by its length

# Key Properties

1. A $k$-cycle moves exactly $k$ elements
2. A $k$-cycle has order $k$
3. A $k$-cycle decomposes into $k - 1$ transpositions
4. A $k$-cycle is even when $k$ is odd, and odd when $k$ is even (p. 36)
5. A 2-cycle is called a transposition
6. An $n$-cycle together with a transposition generates all of $S_n$ (Theorem 6.3)

# Construction / Recognition

## To Construct:
1. Choose $k$ distinct elements from $\{1, \ldots, n\}$
2. Arrange them in a cycle: $(a_1 a_2 \ldots a_k)$

## To Recognize:
1. Write the permutation in disjoint cycle form
2. If it consists of a single cycle of length $k$, it is a $k$-cycle

# Context & Application

The length of cycles appearing in a permutation's disjoint cycle decomposition determines key properties: the order of the permutation (lcm of cycle lengths) and the sign (even iff the total number of even-length cycles is even). The concept of $k$-cycle provides the vocabulary for discussing cycle type.

# Examples

**Example** (p. 34): $(12)$ is a 2-cycle (transposition), $(123)$ is a 3-cycle, $(2856)$ is a 4-cycle.

**Example** (p. 36): The 3-cycle $(132)$ has sign $+1$ because its length 3 is odd, making it an even permutation.

# Common Errors

- **Error**: Thinking a $k$-cycle requires $k$ transpositions to decompose.
  **Correction**: A $k$-cycle decomposes into $k - 1$ transpositions, not $k$.

# Common Confusions

- **Confusion**: Confusing the length of a cycle with its sign/parity.
  **Clarification**: A $k$-cycle is even precisely when $k$ is odd. The parity of the cycle is opposite to the parity of its length.
