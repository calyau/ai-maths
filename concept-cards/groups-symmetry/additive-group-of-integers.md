# Quick Definition

The additive group of integers, denoted Z, is the set of all integers {..., -2, -1, 0, 1, 2, ...} with addition as the group operation. It is an infinite cyclic group generated by 1 (or by -1).

# Core Definition

"Addition of numbers (real or complex) makes each of the following sets into a group: Z, the set of integers" (Armstrong, Ch. 3, p. 18). Zero is the identity element and -x is the inverse of x. The group is abelian because x + y = y + x.

# Prerequisites

- **Group** — Z is verified to be a group

# Key Properties

1. The set is {..., -2, -1, 0, 1, 2, ...}
2. The group operation is addition
3. The identity element is 0
4. The inverse of x is -x
5. The group is abelian (addition is commutative)
6. The group has infinite order
7. Every non-zero element has infinite order (adding x to itself never gives 0)
8. Z is a subgroup of Q, which is a subgroup of R, which is a subgroup of C

# Construction / Recognition

## To Verify:
1. The sum of two integers is an integer (closure)
2. Addition is associative
3. 0 is an integer and acts as identity
4. For every integer x, -x is also an integer (inverse exists)

# Context & Application

Z is the most fundamental infinite group and serves as the prototype for cyclic groups. Armstrong notes that Z is a subgroup of R (Ch. 3, p. 19), introducing the subgroup concept informally before Chapter 5. Z is generated by 1 (since every integer is a sum of copies of 1 or -1).

# Examples

**Example 1** (p. 18): Z forms a group under addition with identity 0 and inverse -x for each x.

**Example 2** (p. 19): Adding two integers always produces an integer, but adding two odd integers produces an even integer, so the odd integers do not form a group under addition.

**Example 3** (Ch. 5, p. 27): The number 1 generates Z, and -1 also generates Z, so Z is an infinite cyclic group.

## Enables
- **Cyclic Group** — Z is the infinite cyclic group
- **Subgroup** — Z < Q < R < C provides the first chain of subgroups
- **Addition Modulo n** — Z_n arises from reducing Z modulo n

## Contrasts With
- **Multiplicative Group** — The integers do not form a group under multiplication (2 has no multiplicative inverse in Z)

# Common Errors

- **Error**: Trying to form a group from the odd integers under addition
  **Correction**: The sum of two odd integers is even, violating closure

# Common Confusions

- **Confusion**: Confusing the additive group Z with the multiplicative structure of integers
  **Clarification**: Under multiplication, the integers do not form a group because most integers lack multiplicative inverses in Z
