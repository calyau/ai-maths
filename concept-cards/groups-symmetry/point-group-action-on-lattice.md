# Quick Definition

The point group J of a wallpaper group G acts on the lattice L: for each M in J and each lattice vector x in L, the vector f_M(x) also belongs to L. This action is proved using the normality of the translation subgroup.

# Core Definition

"(25.2) Theorem. The point group J acts on the lattice L" (Armstrong, Ch. 25, p. 156). The proof computes: if M in J comes from g = (v, M) in G and x is in L (corresponding to translation (x, I) in H), then g(x, I)g^{-1} = (f_M(x), I) lies in H because H is normal in G. Therefore f_M(x) is in L.

# Prerequisites

- **Point group** -- J is the point group acting
- **Lattice** -- L is the set being acted upon
- **Normal subgroup** -- H is normal in G, which is key to the proof

# Key Properties

1. For each M in J and x in L, f_M(x) is in L
2. The proof uses normality: g * tau * g^{-1} is a translation when tau is a translation
3. This action constrains which point groups are compatible with which lattice types
4. The crystallographic restriction is a consequence of this action (Theorem 25.3)
5. Not all elements of G preserve the lattice, but the point group does

# Construction / Recognition

## Proof (Armstrong, p. 156)
1. Let M in J, so M = pi(g) for some g = (v, M) in G
2. Let x in L, corresponding to translation tau = (x, I) in H
3. Compute g * tau * g^{-1} = (f_M(x), I)
4. Since H is normal in G, g * tau * g^{-1} is in H
5. Therefore (f_M(x), I) is a translation in G, so f_M(x) is in L

# Context & Application

This theorem is the crucial link between the point group and the lattice. It provides the mechanism by which the point group constrains the lattice geometry. In particular, the crystallographic restriction (Theorem 25.3) follows directly: if a rotation by 2pi/q is in J, then it must map the shortest lattice vector a to another lattice vector, which constrains q.

Armstrong notes (p. 155) that not all elements of G necessarily preserve the lattice, but the point group always does.

# Examples

**Example 1** (p. 155): In the wallpaper group generated by tau(x,y) = (x+1,y) and the glide h(x,y) = (-x, y+1), the lattice is spanned by (1,0) and (0,2). The point group {I, B} preserves this lattice even though h itself does not map all lattice points to lattice points.

# Common Errors

- **Error**: Assuming every element of G preserves the lattice
  **Correction**: Only the point group (via the orthogonal part) is guaranteed to preserve L; the full isometry (including translation) may not

# Common Confusions

- **Confusion**: Thinking the lattice determines the point group uniquely
  **Clarification**: The lattice type constrains which point groups are possible, but different wallpaper groups with the same lattice can have different point groups
