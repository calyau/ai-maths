# Quick Definition

The dicyclic group of order $4m$ is generated by elements $x, y$ with $x^{2m} = e$, $y^2 = x^m$, and $yxy^{-1} = x^{-1}$. The quaternion group $Q$ is the special case $m = 2$.

# Core Definition

Let $m \ge 2$. The **dicyclic group** of order $4m$ has $4m$ elements $e, x, \ldots, x^{2m-1}, y, xy, \ldots, x^{2m-1}y$ with multiplication rules:
$$x^a x^b = x^{a+b}, \quad x^a(x^b y) = x^{a+b}y, \quad (x^a y)x^b = x^{a-b}y, \quad (x^a y)(x^b y) = x^{a-b+m}$$
where powers of $x$ are read modulo $2m$ (Armstrong, Exercise 20.15, p. 125).

# Prerequisites

- **Classification of groups of order 12** -- the dicyclic group of order 12 arises in this classification

# Key Properties

1. Order is $4m$
2. Contains a cyclic normal subgroup $\langle x \rangle$ of order $2m$ and index 2
3. $y^2 = x^m \ne e$ (so $y$ has order 4)
4. $y$ inverts $x$: $yxy^{-1} = x^{-1}$
5. When $m = 2$, this is the quaternion group $Q$
6. The dicyclic group of order 12 ($m = 3$) is one of the five groups of order 12

# Construction / Recognition

## To Construct:
1. Start with a cyclic group $\langle x \rangle$ of order $2m$
2. Adjoin an element $y$ with $y^2 = x^m$ and $yxy^{-1} = x^{-1}$
3. The $4m$ elements $\{x^a y^b \mid 0 \le a < 2m, b \in \{0, 1\}\}$ form the group

## How It Arises in Order 12 (p. 122):
When $H = \langle x \rangle \cong \mathbb{Z}_3$ is normal, $K = \langle y \rangle \cong \mathbb{Z}_4$, and $yxy^{-1} = x^2$, the element $z = xy^2$ has order 6 and the group has the dicyclic presentation with $m = 3$.

# Context & Application

The dicyclic groups generalise the quaternion group to arbitrary order $4m$. They provide examples of non-abelian groups that are neither dihedral nor direct products.

## Contrasts With
- **Dihedral group** -- in a dihedral group $y^2 = e$; in a dicyclic group $y^2 = x^m \ne e$

# Common Confusions

- **Confusion**: Confusing dicyclic groups with dihedral groups.
  **Clarification**: The key difference is that in a dicyclic group, $y^2 = x^m$ (non-trivial), while in a dihedral group $y^2 = e$ (trivial). This means $y$ has order 4 in the dicyclic group but order 2 in the dihedral group.
