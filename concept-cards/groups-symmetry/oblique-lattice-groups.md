# Quick Definition

An oblique lattice admits exactly two wallpaper groups: p1 (translations only) and p2 (translations plus half-turns). These are the simplest wallpaper groups.

# Core Definition

"Case (a). The lattice of G is oblique. Then the only orthogonal transformations which preserve L are the identity and rotation through pi about the origin. Therefore, the point group of G is a subgroup of {+/- I}" (Armstrong, Ch. 26, p. 163).

**p1**: J = {I}. "G is the simplest of all wallpaper groups; that generated by two independent translations. Its elements have the form (ma + nb, I)" (Armstrong, p. 163).

**p2**: J = {+/- I}. G contains half-turns. Elements not in H have the form (ma + nb, -I), which are half-turns about the points (m/2)a + (n/2)b (Armstrong, p. 163).

# Prerequisites

- **Wallpaper group** -- p1 and p2 are wallpaper groups
- **Five lattice types** -- These arise from the oblique lattice

# Key Properties

1. p1 has trivial point group; only translations
2. p2 has point group Z_2; contains half-turns about points (m/2)a + (n/2)b
3. Both arise from the most general (oblique) lattice, so they also appear for other lattice types
4. p1 is the only wallpaper group isomorphic to Z x Z
5. p2 is the only wallpaper group with point group Z_2 that contains rotations

# Construction / Recognition

## p1
- Elements: (ma + nb, I) for m, n in Z
- Contains only translations
- No rotations, reflections, or glides

## p2
- Translation coset: (ma + nb, I)
- Half-turn coset: (ma + nb, -I) -- half-turns about (m/2)a + (n/2)b
- Half-turn centres form a sublattice of density 4 times the translation lattice

# Context & Application

p1 and p2 are the "base cases" of the classification. Since the oblique lattice has the least symmetry, any additional symmetry (reflections, higher rotations) forces a more symmetric lattice type.

# Examples

**Example 1** (p. 163): p1 consists of elements (ma + nb, I) for integer m, n -- a pure translation group isomorphic to Z x Z.

**Example 2** (p. 163): p2 adds half-turns; the centres of half-turns are at all points (m/2)a + (n/2)b, forming four types of centres per fundamental domain.

# Common Errors

- **Error**: Thinking p2 has only one half-turn centre per fundamental domain
  **Correction**: There are four distinct half-turn centres per fundamental domain: at vertices, edge midpoints of the parallelogram

# Common Confusions

- **Confusion**: Thinking p1 requires an oblique lattice
  **Clarification**: p1 can occur with any lattice type; the oblique lattice merely forces the point group to be trivial or {+/- I}
