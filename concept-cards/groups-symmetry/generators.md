# Quick Definition

A set of generators for a group G is a subset X of G such that every element of G can be expressed as a word in the elements of X (i.e., a product of elements of X and their inverses). If such X exists, we say X generates G.

# Core Definition

"If [the subgroup generated by X] fills out all of G we say that X is a set of generators for G, or that the elements of X together generate G" (Armstrong, Ch. 5, p. 30). Armstrong also notes: "Suppose X is a set of generators for G, and let Y be another subset of G. Then if Y contains X it is also a set of generators for G. More generally, if every element of X can be written as a word in the elements of Y, then Y is a set of generators for G."

# Prerequisites

- **Group** — Generators generate a group
- **Subgroup Generated by Set** — The subgroup generated by X
- **Word in Generators** — Elements are expressed as words

# Key Properties

1. X generates G if every element of G is a word in elements of X
2. A set containing a generating set is also a generating set
3. If each element of X is a word in elements of Y, then Y generates G too
4. Generators are not unique: a group may have many different generating sets
5. A cyclic group has a single generator; D_n requires at least two

# Construction / Recognition

## To Verify X Generates G:
1. Form all possible words in elements of X and their inverses
2. Check that every element of G can be obtained
3. For D_n: verify that r and s generate all 2n elements

# Context & Application

Armstrong introduces generators through dihedral groups, where r and s generate D_n. The concept is fundamental for describing groups compactly: instead of listing all elements, we can specify generators and relations.

# Examples

**Example 1** (p. 30): r and s generate D_n, since every element has the form r^a or r^a*s.

**Example 2** (p. 30): rs and s also generate D_n, because r = (rs)s, so any word in r, s can be converted to a word in rs, s.

**Example 3** (p. 30): In the additive group C, the subgroup generated by {1, i} is the Gaussian integers a + ib with a, b in Z.

# Common Errors

- **Error**: Assuming a generating set is unique
  **Correction**: Groups typically have many different generating sets

- **Error**: Forgetting to include inverses when forming words
  **Correction**: Words include negative powers (inverses) of generators

# Common Confusions

- **Confusion**: Thinking the generators must be "special" or canonical elements
  **Clarification**: Any subset whose words exhaust G is a generating set; the choice is often a matter of convenience
