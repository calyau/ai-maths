# Quick Definition

The braid group $B_3$ is the group whose elements are equivalence classes of braids on three strings, where two braids are equivalent if one can be continuously deformed into the other (keeping endpoints fixed and strings non-intersecting).

# Core Definition

The equivalence classes of braids under continuous deformation form a group under the multiplication defined by stacking. The equivalence class $\mathcal{R}(e)$ of the trivial braid is the identity element, and $\mathcal{R}(b^{-1})$ is the inverse of $\mathcal{R}(b)$. This is the **braid group** $B_3$ on three strings. There is a corresponding group $B_n$ for each positive integer $n$ (Armstrong, Ch. 12, p. 72).

# Prerequisites

- **Braid** — The elements of $B_3$ are equivalence classes of braids
- **Equivalence relation** — Continuous deformation defines the equivalence
- **Equivalence class** — Group elements are equivalence classes

# Key Properties

1. $B_3$ is infinite (Exercise 12.11)
2. $B_3$ is not abelian (Exercise 12.11)
3. $B_3$ is generated by two braids $b_1, b_2$ (Exercise 12.11)
4. There is a surjective function from $B_3$ to $S_3$ that preserves multiplication (Exercise 12.12)

# Construction / Recognition

## To Construct:
1. Consider all braids on three strings
2. Identify braids that are related by continuous deformation (isotopy)
3. The resulting equivalence classes form the group $B_3$
4. Multiplication: stack one braid atop another
5. Identity: the class of the trivial (straight) braid
6. Inverse: the class of the reflected braid

# Context & Application

Armstrong introduces $B_3$ as an example where equivalence relations are essential to defining the group. The elements are not individual braids but classes of braids. This is analogous to how $\mathbb{Z}_n$ can be constructed as equivalence classes of integers (Exercise 12.6). The surjection $B_3 \to S_3$ (Exercise 12.12) foreshadows the homomorphism theory of Chapter 16.

# Common Errors

- **Error**: Treating individual braids as elements of $B_3$
  **Correction**: Elements of $B_3$ are equivalence classes; many different braids represent the same element

# Common Confusions

- **Confusion**: Thinking $B_3$ is finite because $S_3$ is finite
  **Clarification**: $B_3$ is infinite. The surjection $B_3 \to S_3$ has an infinite kernel — many different equivalence classes of braids map to the same permutation.
