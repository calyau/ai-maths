# Quick Definition

A wallpaper group is a subgroup of the Euclidean group E_2 whose translation subgroup is generated by two independent translations and whose point group is finite. There are exactly seventeen wallpaper groups up to isomorphism.

# Core Definition

"A subgroup of E_2 is a wallpaper group if its translation subgroup is generated by two independent translations and its point group is finite" (Armstrong, Ch. 25, p. 153). The translation subgroup H = G intersect T captures the periodicity of the pattern, while the point group J = pi(G) captures its rotational and reflective symmetry. The point group is isomorphic to G/H by the First Isomorphism Theorem (Armstrong, p. 153).

# Prerequisites

- **Euclidean group** -- Wallpaper groups are subgroups of E_2
- **Translation subgroup** -- Must be generated by two independent translations
- **Point group** -- Must be finite

# Key Properties

1. The translation subgroup H is generated by two linearly independent translations
2. The point group J is finite and isomorphic to G/H
3. The point group J acts on the lattice L (Theorem 25.2)
4. The point group is a subgroup of O_2, hence either cyclic or dihedral (by Theorem 19.1)
5. There are exactly 17 wallpaper groups up to isomorphism
6. An isomorphism between wallpaper groups preserves the type of each isometry (Theorem 25.5)

# Construction / Recognition

## To Determine if a Subgroup G of E_2 is a Wallpaper Group
1. Identify the translation subgroup H = G intersect T
2. Check that H is generated by two independent translations
3. Compute the point group J = pi(G) where pi(v, M) = M
4. Verify J is finite

# Context & Application

Wallpaper groups formalize the symmetry of two-dimensional repeating patterns. Armstrong introduces them through the concrete example of hexagonal patterns (Figure 25.1). The term "plane crystallographic group" reflects the connection to three-dimensional crystallography, where analogous lattice structures model atomic arrangements in crystals. Armstrong notes there are 219 crystallographic groups in three dimensions (Ch. 26).

# Examples

**Example 1** (p. 152): The symmetry group of an unshaded hexagonal pattern has translation subgroup generated by tau_1, tau_2 and point group D_6.

**Example 2** (p. 152): Shading parts of each hexagon reduces the point group from D_6 to a subgroup of order 2, while keeping the same translation subgroup.

**Example 3** (p. 155): The group generated by translation tau(x,y) = (x+1,y) and glide reflection h(x,y) = (-x, y+1) is a wallpaper group whose point group has no copy inside G.

# Common Errors

- **Error**: Assuming the point group J always has a copy inside G
  **Correction**: Armstrong gives an explicit example (p. 155) where G contains only translations and glide reflections, so J = {I, B} but B is not realized by any element of G

# Common Confusions

- **Confusion**: Thinking two patterns with the same point group must have the same wallpaper group
  **Clarification**: Different wallpaper groups can share the same point group (e.g., pm and pg both have point group Z_2)
