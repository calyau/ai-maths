# Quick Definition

The even integers 2Z = {..., -4, -2, 0, 2, 4, ...} form a subgroup of Z under addition. More generally, for any positive integer n, the set nZ of all integer multiples of n is a subgroup of Z.

# Core Definition

"The even integers, which we denote by 2Z, form a subgroup of the additive group of integers. For any positive integer n, the set nZ of all integer multiples of n is a subgroup of Z" (Armstrong, Ch. 5, p. 28).

# Prerequisites

- **Subgroup** — 2Z is a subgroup
- **Additive Group of Integers** — 2Z is a subgroup of Z

# Key Properties

1. nZ = {..., -2n, -n, 0, n, 2n, ...}
2. Sum of two multiples of n is a multiple of n (closure)
3. 0 is a multiple of n (identity)
4. The negative of a multiple of n is a multiple of n (inverses)
5. nZ is a cyclic group generated by n (or -n)
6. Every subgroup of Z has this form (Theorem 5.3)

# Construction / Recognition

## To Construct nZ:
1. Take the set of all integers divisible by n
2. Verify closure: if a = kn and b = ln, then a + b = (k+l)n
3. Identity: 0 = 0*n
4. Inverses: if a = kn, then -a = (-k)n

# Context & Application

The subgroups nZ of Z are the complete list of subgroups of Z (by Theorem 5.3). They form a decreasing chain: Z > 2Z > 4Z > 8Z > ... and the subgroup lattice of Z is determined by divisibility.

# Examples

**Example 1** (p. 28): 2Z = {..., -4, -2, 0, 2, 4, ...} is the subgroup of even integers.

**Example 2** (p. 28): 3Z = {..., -6, -3, 0, 3, 6, ...} is the subgroup of multiples of 3.

**Example 3** (Exercise 5.12): The subgroup generated by two integers a and b is dZ where d = gcd(a, b).

# Common Errors

- **Error**: Thinking the odd integers form a subgroup of Z
  **Correction**: The sum of two odd integers is even, so the odd integers are not closed under addition

# Common Confusions

- **Confusion**: Thinking nZ and Z_n are the same thing
  **Clarification**: nZ is an infinite subgroup of Z; Z_n is a finite group of order n
