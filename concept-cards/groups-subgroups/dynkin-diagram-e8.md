# Quick Definition

E_8 is an exceptional Dynkin diagram with 8 nodes: a chain of 7 nodes with one branch from the 5th node. It has 240 roots, and P = Q (so P/Q is trivial).

# Core Definition

**E_8** is the exceptional Dynkin diagram with 8 nodes: a chain of 7 with a branch from the 5th (Milne, §1j, p. 304).

# Prerequisites

- **Dynkin diagram** — E_8 is one of the exceptional types

# Key Properties

1. Rank: 8; Number of roots: 240
2. P/Q = 1 (trivial) — the simply connected and adjoint groups coincide (from table on p. 376)
3. Simply-laced (all roots same length)
4. The largest exceptional root system
