# Quick Definition

The Weyl group W(G,T) = N_G(T)/C_G(T) of a split reductive group (G,T) is the quotient of the normalizer of T by its centralizer. It is a finite group that agrees with the Weyl group of the root datum.

# Core Definition

For a torus T in a reductive group G, the **Weyl group** W(G,T) = N_G(T)/C_G(T) is the quotient of the normalizer by the centralizer. When T is a split maximal torus, C_G(T) = T, so W(G,T) = N_G(T)/T. It is a constant finite étale group when T is split (Milne, Theorem 2.7, p. 343).

# Prerequisites

- **Split reductive group** — W(G,T) is defined for split reductive groups
- **Split maximal torus** — T must be a maximal torus

# Key Properties

1. W(G,T) acts on X*(T) and stabilizes R (Lemma 2.8)
2. W(G,T) is generated by the reflections s_α (Corollary 3.43, p. 363)
3. Agrees with the Weyl group of the root datum Ψ(G,T)
4. W = N_G(T)/T when T is maximal (since C_G(T) = T)

# Examples

**Example 1** (p. 342): For GL_n, W ≅ S_n, represented by permutation matrices.
