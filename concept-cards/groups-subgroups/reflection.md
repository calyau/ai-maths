# Quick Definition

A reflection of a vector space V is an endomorphism that fixes a hyperplane pointwise and acts as −1 on a complementary line.

# Core Definition

A **reflection** of a vector space V is an endomorphism of V that fixes the vectors in a hyperplane and acts as −1 on a complementary line. A **reflection with vector α** is an endomorphism s of V such that s(α) = −α and the set of vectors fixed by s is a hyperplane H. Then V = H ⊕ ⟨α⟩ with s acting as 1 ⊕ (−1), and s² = 1 (Milne, §1a, p. 296).

If α^∨ ∈ V^∨ satisfies ⟨α, α^∨⟩ = 2, then s_α: x ↦ x − ⟨x, α^∨⟩α is a reflection with vector α, and every reflection with vector α is of this form for a unique α^∨.

# Prerequisites

- **Vector space** — Reflections are linear endomorphisms of V

# Key Properties

1. A reflection s satisfies s² = 1 (it is an involution)
2. Given a finite spanning set R and nonzero α ∈ V, there is at most one reflection with vector α mapping R into R (Lemma 1.1)
3. For a real vector space with inner product, s_α(x) = x − 2((x,α)/(α,α))α is the unique orthogonal reflection with vector α (Lemma 1.2)

# Context & Application

Reflections are the fundamental symmetries from which root systems and Weyl groups are built. Each root α in a root system determines a unique reflection s_α that preserves the root system.

# Examples

**Example 1** (p. 296): For the root system A_n with R = {e_i − e_j}, the reflection s_{e_i − e_j} acts by switching the i-th and j-th coordinates.
