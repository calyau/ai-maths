# Quick Definition

G_2 is an exceptional Dynkin diagram with 2 nodes connected by a triple edge (with arrow). It is the smallest exceptional root system, with 12 roots and Weyl group D_6 (dihedral of order 12).

# Core Definition

**G_2** is the exceptional Dynkin diagram: ○≡>○ (2 nodes, triple edge with arrow toward shorter root). The angle between simple roots is 5π/6, and the length ratio is √3 (Milne, §1f, pp. 299–300).

# Prerequisites

- **Dynkin diagram** — G_2 is one of the indecomposable types

# Key Properties

1. Rank: 2; Number of roots: 12
2. Weyl group: D_6 (dihedral of order 12)
3. P/Q = 1 (trivial) (from table on p. 376)
4. Non-simply-laced (two root lengths, ratio √3)
5. Cartan matrix: [[2,−1],[−3,2]]
