# Quick Definition

A root system is indecomposable (or irreducible) if it cannot be written as a direct sum of two nonempty root systems. Equivalently, its Coxeter graph (or Dynkin diagram) is connected.

# Core Definition

A root system (V, R) is **indecomposable** (or **irreducible**) if it cannot be written as a direct sum of nonempty root systems (Milne, p. 297). Equivalently, R cannot be partitioned into R₁ ⊔ R₂ with every root in R₁ orthogonal to every root in R₂. This is equivalent to the Coxeter graph being connected (Proposition 1.16).

Every root system decomposes uniquely as a direct sum of indecomposable root systems (Proposition 1.7).

# Key Properties

1. Indecomposable iff Dynkin diagram connected (Proposition 1.16)
2. Indecomposable root system ↔ simple Lie algebra (Proposition 2.32)
3. Classification: A_n, B_n, C_n, D_n, E_6, E_7, E_8, F_4, G_2
