# Quick Definition

A root datum is toral if R is empty. A split reductive group is a torus if and only if its root datum is toral.

# Core Definition

A root datum is **toral** if R is empty (Milne, Definition 2.17, p. 347). A split reductive group is a torus if and only if its root datum is toral (Proposition 2.18).

# Prerequisites

- **Root datum** — A toral root datum is a degenerate case

# Key Properties

1. If R = ∅, then Z(G) = T (formula 172 gives X*(Z(G)) = X)
2. The adjoint representation is trivial: 𝔤 = 𝔤₀
