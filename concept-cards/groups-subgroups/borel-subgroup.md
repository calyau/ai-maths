# Quick Definition

A Borel subgroup of a reductive group G is a smooth subgroup B such that B_{k^{al}} is a maximal smooth connected solvable subgroup of G_{k^{al}}. All Borel subgroups are conjugate, and G/B is a projective variety.

# Core Definition

A **Borel subgroup** of a reductive group G is a smooth subgroup B such that B_{k^{al}} is a maximal smooth connected solvable subgroup of G_{k^{al}} (Milne, Definition 3.20, p. 355).

Key properties (Theorem 3.21): If B is a Borel subgroup of G, then G/B is projective. Any two Borel subgroups are conjugate by an element of G(k^{al}).

For a split reductive group (G,T), the Borel subgroups containing T are in bijection with the bases of the root system R.

# Prerequisites

- **Reductive algebraic group** — Borel subgroups exist in reductive groups
- **Borel fixed point theorem** — Used to prove conjugacy

# Key Properties

1. G/B is projective (Theorem 3.21)
2. All Borel subgroups are conjugate (Theorem 3.21)
3. N_G(B) = B (normalizer theorem, Theorem 3.42)
4. G = ∪_{g∈G(k^{al})} gBg⁻¹ (Theorem 3.24)
5. For (G,T) split: Borel subgroups containing T correspond to bases of R (Theorem 4.9)
6. B is generated by T and the root groups U_α for α ∈ R⁺

# Construction / Recognition

## To Find Borel Subgroups:
1. For GL_V: stabilizers of maximal flags (e.g., upper triangular matrices)
2. For SO_n, Sp_{2n}: stabilizers of maximal totally isotropic flags
3. In general: choose a base S for R, then B = ⟨T, U_α | α ∈ R⁺⟩

# Examples

**Example 1** (p. 355): 𝕋₂ = {((*,*),(0,*))} is a Borel subgroup of GL₂.

**Example 2** (p. 358): For GL_V, the Borel subgroups are exactly the stabilizers of maximal flags. The standard Borel subgroup is the upper triangular matrices.

# Common Confusions

- **Confusion**: A Borel subgroup is just any solvable subgroup
  **Clarification**: It must be maximal among connected solvable subgroups
