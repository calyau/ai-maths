# Quick Definition

A root datum is a quadruple Ψ = (X, R, X^∨, R^∨) where X and X^∨ are dual free ℤ-modules, R ⊂ X and R^∨ ⊂ X^∨ are finite subsets in bijection α ↔ α^∨, satisfying ⟨α, α^∨⟩ = 2 and the reflections s_α preserving R. Root data classify split reductive groups.

# Core Definition

A **root datum** is a quadruple Ψ = (X, R, X^∨, R^∨) where (Milne, Definition 2.1, p. 340):

- X, X^∨ are free ℤ-modules of finite rank in duality by ⟨ , ⟩: X × X^∨ → ℤ
- R, R^∨ are finite subsets of X and X^∨ in bijection α ↔ α^∨

satisfying:
- **(rd1)** ⟨α, α^∨⟩ = 2 for all α ∈ R
- **(rd2)** s_α(R) ⊂ R for all α ∈ R, where s_α(x) = x − ⟨x, α^∨⟩α
- **(rd3)** The Weyl group W(Ψ) (generated by the s_α) is finite

The elements of R and R^∨ are called **roots** and **coroots**. The group W(Ψ) is the **Weyl group** of the root datum.

# Prerequisites

- **Root system** — A root datum enriches a root system with lattice data
- **Character group** — X = X*(T) in the algebraic group setting
- **Cocharacter group** — X^∨ = X_*(T) in the algebraic group setting
- **Split reductive group** — Root data classify these groups

# Key Properties

1. s_α² = 1 (reflections are involutions, from rd1)
2. W(Ψ) acts faithfully on R and is finite (Proposition 5.10)
3. For (G,T) split reductive: Ψ(G,T) = (X*(T), R, X_*(T), R^∨) (Theorem 2.10)
4. A root datum is **reduced** if ±α are the only root multiples of α (Definition, p. 348)
5. A root datum is **semisimple** if R generates a subgroup of finite index in X (Definition 2.15)
6. A root datum is **toral** if R is empty (Definition 2.17)

# Construction / Recognition

## To Construct from (G,T):
1. Set X = X*(T), X^∨ = X_*(T) with natural pairing
2. Decompose 𝔤 under Ad(T): roots R are the nontrivial characters
3. For each α ∈ R, find the coroot α^∨ via the rank-1 subgroup G_α (Proposition 2.9)

# Context & Application

Root data are the fundamental classifying invariants for split reductive groups. The root datum is field-independent: the same root datum classifies a split reductive group over any field. This is the key result enabling classification.

# Examples

**Example 1** (p. 341): SL₂ has root datum (ℤ, {2,−2}, ℤ, {1,−1}) with canonical pairing.

**Example 2** (p. 341): PGL₂ has root datum (ℤ, {1,−1}, ℤ, {2,−2}).

**Example 3** (p. 342): GL_n has root datum (ℤⁿ, {e_i−e_j}, ℤⁿ, {e_i−e_j}) with standard pairing.

**Example 4** (p. 342): 𝔾_m has root datum (ℤ, ∅, ℤ, ∅) (toral).

# Relationships

## Extends
- **Root system** — A root datum includes a root system plus lattice data

## Enables
- **Existence theorem** — Every reduced root datum arises from a split reductive group
- **Isomorphism theorem** — Every isomorphism of root data comes from an isomorphism of groups

## Contrasts With
- **Root system** — A root system lives in a vector space; a root datum specifies integral lattices X, X^∨

# Common Confusions

- **Confusion**: Root datum = root system
  **Clarification**: A root datum (X, R, X^∨, R^∨) includes the lattices X and X^∨, not just the roots in a vector space. Different lattices (between Q and P) give non-isomorphic root data with the same root system.
