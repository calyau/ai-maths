# Quick Definition

F_4 is an exceptional Dynkin diagram with 4 nodes: a chain with a double edge (with arrow) between the 2nd and 3rd nodes. It has 48 roots, and P = Q.

# Core Definition

**F_4** is the exceptional Dynkin diagram: ○—○⇒○—○ (4 nodes, double edge between nodes 2 and 3 with arrow toward node 3). It has two root lengths (Milne, §1j, p. 304).

# Prerequisites

- **Dynkin diagram** — F_4 is one of the exceptional types

# Key Properties

1. Rank: 4; Number of roots: 48
2. P/Q = 1 (trivial) (from table on p. 376)
3. Non-simply-laced (two root lengths)
