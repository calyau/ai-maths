# Quick Definition

A base (or system of simple roots) for a root system R is a subset S of R that is a basis for V and such that every root is an integer linear combination of elements of S with coefficients all of the same sign.

# Core Definition

Let (V, R) be a root system. A subset S of R is a **base** for R if it is a basis for V and if each root can be written β = ∑_{α∈S} m_α α with the m_α integers of the same sign (i.e., either all m_α ≥ 0 or all m_α ≤ 0). The elements of a base are called the **simple roots** (Milne, §1e, p. 298).

# Prerequisites

- **Root system** — A base is a distinguished subset of R
- **Weyl chamber** — Each chamber determines a unique base

# Key Properties

1. Bases exist for any root system (Proposition 1.10)
2. The Weyl group acts simply transitively on the set of bases (from 1.8 and 1.10)
3. W is generated by {s_α | α ∈ S} and W · S = R (Proposition 1.11)
4. A base S determines positive roots R⁺ and negative roots R⁻ with R = R⁺ ⊔ R⁻
5. For an indecomposable root system, there is a unique highest root α̃ = ∑ n_α α with n_α ≥ m_α for all other positive roots (Proposition 1.12)

# Construction / Recognition

## To Construct:
1. Choose t in a Weyl chamber (i.e., ⟨t, α^∨⟩ ≠ 0 for all α ∈ R)
2. Define R⁺ = {α ∈ R | (α, t) > 0}
3. Find the indecomposable elements of R⁺ (those not expressible as a sum of two elements of R⁺)
4. These indecomposable elements form the base S

# Examples

**Example 1** (p. 298, Example 1.13): For A_n with R = {e_i − e_j}, choosing t = ne₁ + ... + e_n − (n/2)(e₁ + ... + e_{n+1}) gives R⁺ = {e_i − e_j | i > j}. The base is S = {e₁ − e₂, ..., e_n − e_{n+1}} and the highest root is α̃ = e₁ − e_{n+1} = α₁ + ... + αₙ.
