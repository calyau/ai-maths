# Quick Definition

An algebraic variety X is complete if, for all varieties T, the projection X × T → T is closed. Projective varieties are complete. An affine variety is complete iff it is a finite set.

# Core Definition

An algebraic variety X is **complete** if, for all algebraic varieties T, the projection map X × T → T is closed (Milne, 3.3, p. 351). Every projective variety is complete. An affine variety is complete if and only if it has dimension zero (i.e., is a finite set).

# Prerequisites

This is a foundational concept in algebraic geometry with no prerequisites within this source beyond basic algebraic geometry.

# Key Properties

1. Projective ⇒ complete
2. Complete + affine ⇒ finite set
3. Image of a complete variety under a regular map is closed and complete
4. Key in the Borel fixed point theorem: closed orbits on a complete variety must be points
