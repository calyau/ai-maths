# Quick Definition

The Weyl group W(R) of a root system (V, R) is the finite subgroup of GL(V) generated by the reflections s_α for α ∈ R. It acts simply transitively on the set of Weyl chambers and on the set of bases for R.

# Core Definition

Let (V, R) be a root system. The **Weyl group** W = W(R) is the subgroup of GL(V) generated by the reflections s_α for α ∈ R. Because R spans V, the group W acts faithfully on R, and so is finite (Milne, §1c, p. 297).

For a split reductive group (G,T), the Weyl group W(G,T) = N_G(T)/C_G(T) is the quotient of the normalizer of T by the centralizer of T. When T is split, this is a constant finite group that agrees with the Weyl group of the root datum (Milne, Ch. V, p. 343).

# Prerequisites

- **Root system** — The Weyl group is defined from the reflections of a root system
- **Reflection** — The generators s_α of the Weyl group

# Key Properties

1. W is a finite group (since it acts faithfully on the finite set R)
2. W acts simply transitively on the set of Weyl chambers (Proposition 1.8)
3. W acts simply transitively on the set of bases for R (follows from 1.8 and 1.10)
4. W is generated by {s_α | α ∈ S} for any base S, and W · S = R (Proposition 1.11)
5. n(wα, wβ) = n(α, β) for all w ∈ W (the Cartan integers are W-invariant)

# Construction / Recognition

## To Compute:
1. Choose a base S = {α₁, ..., αₙ} for R
2. The Weyl group is generated by the simple reflections s_{α₁}, ..., s_{αₙ}
3. The order of W can be determined from the Dynkin diagram type

## Common Weyl Groups:
- A_n: W ≅ S_{n+1} (symmetric group), order (n+1)!
- B_n, C_n: W ≅ (ℤ/2)ⁿ ⋊ S_n (hyperoctahedral group), order 2ⁿn!
- D_n: W ≅ (ℤ/2)^{n-1} ⋊ S_n, order 2^{n-1}n!

# Context & Application

The Weyl group is central to the structure theory of semisimple Lie algebras and algebraic groups. It controls the classification of Borel and parabolic subgroups (via Weyl chambers and bases), appears in the Bruhat decomposition G = BWB, and governs the representation theory through its action on weights.

# Examples

**Example 1** (p. 298, Example 1.13): For the root system A_n with roots {e_i − e_j}, the Weyl group is naturally identified with S_{n+1}, generated by the transpositions s_{α_i} that swap entries i and i+1.

**Example 2** (p. 300): For the rank 2 root systems: W(A₁ × A₁) = D₂, W(A₂) = D₃ ≅ S₃, W(B₂) = D₄ (dihedral of order 8), W(G₂) = D₆ (dihedral of order 12).

# Common Errors

- **Error**: Attempting to compute W using all reflections s_α for α ∈ R
  **Correction**: It suffices to use only the simple reflections s_α for α ∈ S (a base)
