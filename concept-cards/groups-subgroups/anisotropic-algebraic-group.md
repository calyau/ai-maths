# Quick Definition

An affine algebraic group G over ℝ is anisotropic (or compact) if G(ℝ) is compact and G(ℝ) is dense in G for the Zariski topology.

# Core Definition

An affine algebraic group G over ℝ is **anisotropic** (or **compact**) if: (a) G(ℝ) is compact, and (b) G(ℝ) is dense in G for the Zariski topology. Condition (b) is equivalent to: every connected component of G (Zariski topology) contains a real point; in particular, (b) holds if G is connected (Milne, Definition 3.3, p. 331).

# Prerequisites

- **Reductive algebraic group** — Anisotropic groups arise as real forms of reductive groups

# Key Properties

1. There is a one-to-one correspondence between maximal compact subgroups of G(ℂ) and anisotropic real forms of G (Corollary 3.6)
2. A subgroup of an anisotropic group is anisotropic (Aside 3.8)
3. Maximal compact subgroups of complex algebraic groups are conjugate (Aside 3.8)

# Relationships

## Contrasts With
- **Split reductive group** — A split group has a split maximal torus; an anisotropic group has no non-trivial split torus
