# Quick Definition

A root datum is reduced if the only multiples of a root α that are also roots are α and −α. Reduced root data classify split reductive groups.

# Core Definition

A root datum is **reduced** if the only multiples of a root α that can also be a root are α and −α (Milne, p. 348). The existence and isomorphism theorems (2.21, 2.22) establish a bijection between reduced root data and split reductive groups.

# Prerequisites

- **Root datum** — A reduced root datum is a special kind

# Key Properties

1. Every root datum arising from a split reductive group is reduced
2. Reduced root data are classified by root systems + lattice Q ⊂ X ⊂ P + toral part X₀
