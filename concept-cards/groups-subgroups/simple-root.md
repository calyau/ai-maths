# Quick Definition

A simple root is an element of a chosen base S for a root system R. Every positive root is a non-negative integer linear combination of simple roots.

# Core Definition

The elements of a base S for a root system R are called **simple roots** (for that base). They form a basis for V, and every root β can be written as β = ∑_{α∈S} m_α α with either all m_α ≥ 0 (positive root) or all m_α ≤ 0 (negative root) (Milne, §1e, p. 298).

A simple root α is called **special** if its coefficient n_α in the highest root α̃ = ∑ n_α α equals 1 (Milne, p. 299).

# Prerequisites

- **Base of root system** — Simple roots are the elements of a base

# Key Properties

1. Simple roots are linearly independent and span V
2. The Weyl group is generated by reflections in the simple roots (Proposition 1.11)
3. Every root is in the W-orbit of a simple root (Proposition 1.11: W · S = R)
4. For distinct simple roots α, β: n(α,β) ≤ 0 (the angle between them is obtuse or right)

# Examples

**Example 1** (p. 298): For A_n, the simple roots are α_i = e_i − e_{i+1} for 1 ≤ i ≤ n.
