# Quick Definition

The Clifford algebra C(V,q) of a quadratic space (V,q) is the quotient of the tensor algebra T(V) by the ideal generated by elements x tensor x - q(x). It is a 2^n-dimensional algebra that encodes the quadratic form and serves as the ambient algebra for the spin group.

# Core Definition

The **Clifford algebra** C(V,q) is the quotient of the tensor algebra T(V) of V by the two-sided ideal I(q) generated by the elements x tensor x - q(x) for x in V (Milne, Definition 18.14). The canonical map rho: V -> C(V,q) satisfies rho(x)^2 = q(x) for all x in V.

# Prerequisites

- **Quadratic form** -- The Clifford algebra is built from a quadratic space
- **Tensor algebra** -- C(V,q) is a quotient of T(V)

# Key Properties

1. dim C(V,q) = 2^n where n = dim V (Theorem 18.18c)
2. C(V,q) has a Z/2Z-grading (superalgebra structure): C = C_0 + C_1
3. The map rho: V -> C(V,q) is injective (Corollary 18.19)
4. For an orthogonal basis e_1,...,e_n with q(e_i) = c_i, C(V,q) is isomorphic to C(c_1,...,c_n) (Theorem 18.18a)
5. C(V,q) has an involution * with (x_1...x_r)* = x_r...x_1 for x_i in V
6. rho(x)rho(y) + rho(y)rho(x) = 2phi(x,y) for x,y in V
7. If n is even, the centre is k; if n is odd, it is degree 2 over k (Proposition 18.21)
8. Universal property: any k-linear map r: V -> D with r(x)^2 = q(x) factors uniquely through C(V,q) (Proposition 18.17)

# Construction / Recognition

## To Construct:
1. Start with a quadratic space (V,q) of dimension n
2. Choose an orthogonal basis e_1,...,e_n with q(e_i) = c_i
3. C(V,q) is the algebra generated by e_1,...,e_n with relations e_i^2 = c_i and e_j*e_i = -e_i*e_j for i != j
4. As a vector space, it has basis {e_{i_1}...e_{i_r} | 1 <= i_1 < ... < i_r <= n}

# Context & Application

The Clifford algebra is the key technical tool for constructing spin groups. The spin group Spin(q) lives inside the even part C_0(V,q) of the Clifford algebra. Historically, Cartan discovered representations of the orthogonal Lie algebra that do not come from the orthogonal group but do come from the spin group -- these are the spinor representations.

# Examples

**Example 1** (p. 209): If V is one-dimensional with basis e and q(e) = c, then C(V,q) = k[e]/(e^2 - c), which is C(c).

**Example 2** (p. 209): If q = 0, then C(V,q) is the exterior algebra on V.

# Common Errors

- **Error**: Forgetting the sign rule for the super tensor product when decomposing C(V_1 + V_2, q_1 + q_2)
  **Correction**: C(V,q) = C(V_1,q_1) hat-tensor C(V_2,q_2), using the super (graded) tensor product

# Common Confusions

- **Confusion**: Confusing the Clifford algebra with the exterior algebra
  **Clarification**: The exterior algebra is the special case q = 0; for nonzero q, the algebra is noncommutative with x^2 = q(x) rather than x^2 = 0
