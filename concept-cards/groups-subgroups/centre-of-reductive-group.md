# Quick Definition

The centre Z(G) of a split reductive group (G,T) equals ∩_{α∈R} Ker(α) ⊂ T, and has character group X*(Z(G)) = X*(T)/ℤR. For the simply connected group, X*(Z) ≅ P/Q.

# Core Definition

Every maximal torus T contains the centre Z(G) (Proposition 2.14a). The kernel of Ad: T → GL_𝔤 is Z(G) (Proposition 2.14b), so Z(G) = ∩_{α∈R} Ker(α). Applying X* gives X*(Z(G)) = X*(T)/{subgroup generated by R} (Milne, formula 172, p. 347).

# Prerequisites

- **Split reductive group** — Z(G) is computed from the root datum
- **Root datum** — The centre is determined by the root datum

# Key Properties

1. Z(G) ⊂ T for any maximal torus T
2. X*(Z(G)) = X/ℤR
3. G is semisimple iff Z(G) is finite iff root datum is semisimple
4. G is a torus iff Z(G) = T iff R = ∅

# Examples

**Example 1** (p. 347): Z(GL_n) = {scalar matrices}, Z(SL₂) = μ₂, Z(PGL₂) = 1.
