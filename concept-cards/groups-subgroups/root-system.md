# Quick Definition

A root system is a finite set R of nonzero vectors in a vector space V satisfying symmetry and integrality conditions that encode the structure of semisimple Lie algebras and algebraic groups.

# Core Definition

A subset R of a vector space V over a field F (of characteristic zero) is a **root system** in V if it satisfies three axioms (Milne, Definition 1.3, p. 296):

- **RS1**: R is finite, spans V, and does not contain 0;
- **RS2**: for each α ∈ R, there exists a unique reflection s_α with vector α such that s_α(R) ⊂ R;
- **RS3**: for all α, β ∈ R, s_α(β) − β is an integer multiple of α.

Equivalently, for each α ∈ R, there exists a unique α^∨ ∈ V^∨ such that ⟨α, α^∨⟩ = 2, ⟨R, α^∨⟩ ⊂ ℤ, and the reflection s_α: x ↦ x − ⟨x, α^∨⟩α maps R into R.

The elements of R are called **roots**. The dimension of V is the **rank** of the root system. A root system is **reduced** if, for each root α, the only multiples of α that are also roots are ±α.

# Prerequisites

- **Reflection** — Root systems are defined in terms of reflections preserving the set R
- **Vector space** — R lives in a finite-dimensional vector space V
- **Dual vector space** — The coroots α^∨ live in V^∨

# Key Properties

1. For each root α, −α is also a root (since s_α(α) = −α)
2. To give a root system over F is the same as giving one over ℚ (Proposition 1.5, p. 297)
3. Every root system decomposes uniquely as a direct sum of indecomposable root systems (Proposition 1.7, p. 297)
4. In a reduced root system, the only scalar multiples of a root α that are also roots are ±α
5. The Weyl group W(R) is a finite group that acts faithfully on R
6. There exists a W-invariant inner product on V (Proposition 1.9, p. 298)

# Construction / Recognition

## To Construct:
1. Start with a finite-dimensional vector space V
2. Choose a finite spanning set R of nonzero vectors in V
3. For each α ∈ R, find α^∨ ∈ V^∨ with ⟨α, α^∨⟩ = 2
4. Verify ⟨β, α^∨⟩ ∈ ℤ for all α, β ∈ R (integrality)
5. Verify s_α(R) ⊂ R for all α ∈ R (invariance under reflections)

## To Recognize:
1. Check R is finite, spans V, and 0 ∉ R
2. For each α, check existence of s_α preserving R
3. Check integrality condition RS3

# Context & Application

Root systems are the combinatorial backbone of the theory of semisimple Lie algebras and algebraic groups. Every split semisimple Lie algebra determines a root system via the eigenvalues of a Cartan subalgebra acting on the Lie algebra, and the root system determines the Lie algebra up to isomorphism. Root systems are classified by Dynkin diagrams, providing a complete classification of semisimple Lie algebras (in characteristic zero).

# Examples

**Example 1** (p. 296, Example 1.4): The root system of type A_n. Let V be the hyperplane in F^{n+1} of tuples (x_i) with ∑x_i = 0. Then R = {e_i − e_j | i ≠ j, 1 ≤ i,j ≤ n+1}, where s_{α_{ij}} acts by switching the i-th and j-th entries. This is the root system of sl_{n+1}.

**Example 2** (p. 299): The four reduced root systems of rank 2 are A₁ × A₁, A₂, B₂, and G₂, with 4, 6, 8, and 12 roots respectively.

# Common Errors

- **Error**: Forgetting to check that 0 ∉ R
  **Correction**: Axiom RS1 explicitly excludes 0 from R

- **Error**: Confusing root systems with root data
  **Correction**: A root system lives in a vector space; a root datum additionally specifies lattices X and X^∨ with a perfect pairing

# Common Confusions

- **Confusion**: Believing every root system is reduced
  **Clarification**: In a non-reduced root system, α/2 can be a root when α is; Milne's convention is that "root system" means "reduced root system" unless stated otherwise

- **Confusion**: Thinking the root system depends on the choice of base field F
  **Clarification**: A root system over F is the same as one over ℚ (Proposition 1.5)
