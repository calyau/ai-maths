# Quick Definition

E_6 is an exceptional Dynkin diagram with 6 nodes: a chain of 5 nodes with one branch from the 3rd node. It has 72 roots, Weyl group of order 51840, and P/Q ≅ ℤ/3ℤ.

# Core Definition

**E_6** is the exceptional Dynkin diagram consisting of a chain of 5 nodes with a single branch from the 3rd node (Milne, §1j, p. 304). It is one of the five exceptional types (E_6, E_7, E_8, F_4, G_2).

# Prerequisites

- **Dynkin diagram** — E_6 is one of the exceptional indecomposable types

# Key Properties

1. Rank: 6; Number of roots: 72
2. P/Q ≅ ℤ/3ℤ (from table on p. 376)
3. All roots have the same length (simply-laced)
4. Has a non-trivial diagram automorphism of order 2

# Context & Application

E_6 appears in various areas of mathematics and physics. The simply connected group of type E_6 has centre ℤ/3ℤ.
