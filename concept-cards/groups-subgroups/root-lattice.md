# Quick Definition

The root lattice Q = Q(R) is the ℤ-submodule of V generated by the roots. Every base for R forms a ℤ-basis for Q.

# Core Definition

Let (V, R) be a root system. The **root lattice** Q = Q(R) is the ℤ-submodule of V generated by the roots: Q(R) = ℤR = {∑_{α∈R} m_α α | m_α ∈ ℤ}. Every base S for R forms a basis for Q (Milne, 1.20, p. 303).

# Prerequisites

- **Root system** — Q is spanned by the roots
- **Base of root system** — Any base gives a ℤ-basis for Q

# Key Properties

1. Q ⊂ P (the weight lattice), since ⟨R, α^∨⟩ ⊂ ℤ for all α
2. P/Q is finite (both lattices span the same ℚ-vector space)
3. For a base S = {α₁, ..., αₙ}, Q = ℤα₁ ⊕ ... ⊕ ℤαₙ
4. The quotient P/Q classifies the possible centres of semisimple groups with given root system

# Examples

**Example 1** (p. 315): For A_n, Q(R) = {m₁ε₁ + ... + m_{n+1}ε_{n+1} | m_i ∈ ℤ, ∑m_i = 0} and P/Q ≅ ℤ/(n+1)ℤ.

# Relationships

## Contrasts With
- **Weight lattice** — Q ⊂ P; the quotient P/Q measures how many groups share the same Lie algebra
