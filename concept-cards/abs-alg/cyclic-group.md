# Quick Definition
A cyclic group is a group generated by a single element: $H = \langle x \rangle = \{x^n \mid n \in \mathbb{Z}\}$. Every cyclic group is abelian. Up to isomorphism, the cyclic group of order n is $\mathbb{Z}/n\mathbb{Z}$.

# Core Definition
A group H is *cyclic* if $H = \langle x \rangle = \{x^n \mid n \in \mathbb{Z}\}$ for some $x \in H$. Then $|H| = |x|$: if $|x| = n < \infty$ then $H = \{1, x, x^2, \ldots, x^{n-1}\}$; if $|x| = \infty$ then all powers are distinct. Any two cyclic groups of the same order are isomorphic (Theorem 4). $Z_n$ denotes the cyclic group of order n; $Z_n \cong \mathbb{Z}/n\mathbb{Z}$ (Dummit & Foote, pp. 54-58).

# Prerequisites
- **Subgroup** — cyclic subgroups are subgroups
- **Order of an element** — $|H| = |x|$

# Key Properties
1. Cyclic groups are abelian
2. $Z_n \cong \mathbb{Z}/n\mathbb{Z}$ and the infinite cyclic group $\cong \mathbb{Z}$
3. Every subgroup of a cyclic group is cyclic (Theorem 7)
4. If $|H| = n$, there is a unique subgroup of order d for each divisor d of n: $\langle x^{n/d} \rangle$
5. $\langle x^a \rangle = H$ iff $(a, n) = 1$; the number of generators is $\varphi(n)$
6. If $|x| = n$ then $|x^a| = n/(n,a)$ (Proposition 5)

# Construction / Recognition
## To Construct/Create:
1. Take any element x of a group G
2. The cyclic subgroup $\langle x \rangle = \{x^n \mid n \in \mathbb{Z}\}$

## To Identify/Recognize:
1. A group is cyclic iff it can be generated by one element
2. Every group of prime order is cyclic (Corollary 10)

# Context & Application
Cyclic groups are the simplest groups. Every finite abelian group decomposes as a direct product of cyclic groups. The subgroup structure of cyclic groups is completely determined by divisibility of n.

# Examples
**Example 1** (p. 54): The rotations in $D_{2n}$ form a cyclic group $\langle r \rangle$ of order n.
**Example 2** (p. 54): $\mathbb{Z} = \langle 1 \rangle = \langle -1 \rangle$ is an infinite cyclic group.
**Example 3** (p. 58): Subgroups of $\mathbb{Z}/12\mathbb{Z}$ correspond to divisors of 12: orders 1, 2, 3, 4, 6, 12.

# Common Errors
- **Error**: Assuming all abelian groups are cyclic. **Correction**: $V_4 = \mathbb{Z}_2 \times \mathbb{Z}_2$ is abelian but not cyclic (no element of order 4).

# Common Confusions
- **Confusion**: Thinking a cyclic group has only one generator. **Clarification**: A cyclic group of order n has exactly $\varphi(n)$ generators.
