# Quick Definition
If I is an ideal of R, then $R[x]/I[x] \cong (R/I)[x]$. If I is prime, then $I[x]$ is prime in $R[x]$.

# Core Definition
(Proposition 2) Let I be an ideal of R and $(I) = I[x]$ the ideal of $R[x]$ generated by I. Then $R[x]/(I) \cong (R/I)[x]$. If I is a prime ideal of R then $(I)$ is a prime ideal of $R[x]$ (Dummit & Foote, pp. 296-297).

# Prerequisites
- **Polynomial ring** — The result is about $R[x]$
- **Ideal** — I is an ideal of R
- **Quotient ring** — The result involves quotients

# Key Properties
1. The map $R[x] \to (R/I)[x]$ reducing coefficients mod I is a ring homomorphism
2. Its kernel is $I[x]$ (polynomials with all coefficients in I)
3. If I is prime, $R/I$ is an integral domain, so $(R/I)[x]$ is an integral domain, so $I[x]$ is prime
4. If I is maximal in R, the ideal $(I, x)$ is maximal in $R[x]$ (p. 297)

# Examples
**Example** (p. 297): $\mathbb{Z}[x]/p\mathbb{Z}[x] \cong \mathbb{Z}/p\mathbb{Z}[x]$. For prime p, $p\mathbb{Z}[x]$ is a prime ideal of $\mathbb{Z}[x]$.
