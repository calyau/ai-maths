# Quick Definition
The Primitive Element Theorem states that every finite separable extension K/F is simple, i.e., $K = F(\alpha)$ for some single element $\alpha \in K$.

# Core Definition
(Theorem, p. 595) If K/F is a finite separable extension, then K = F($\alpha$) for some $\alpha \in K$ (called a primitive element). In particular, every finite extension of a field of characteristic 0 is simple. Over infinite fields, the primitive element can be taken as a linear combination of the generators (Dummit & Foote, pp. 595-597).

# Prerequisites
- **separable-extension** — The extension must be separable
- **field-extension** — Applies to finite extensions

# Key Properties
1. Finite separable extensions are generated by a single element
2. Over infinite fields: if $K = F(\alpha, \beta)$ with $\beta$ separable, then $K = F(\alpha + c\beta)$ for all but finitely many $c \in F$
3. Finite fields: every extension is simple (finite fields are perfect)
4. Counterexample: $\mathbb{F}_p(x, y)$ over $\mathbb{F}_p(x^p, y^p)$ is not simple (inseparable)
