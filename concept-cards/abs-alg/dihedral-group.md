# Quick Definition
The dihedral group $D_{2n}$ is the group of symmetries of a regular n-gon. It has order 2n and is generated by a rotation r of order n and a reflection s of order 2, with the relation $rs = sr^{-1}$.

# Core Definition
For $n \geq 3$, the *dihedral group of order 2n*, denoted $D_{2n}$, is the set of symmetries of a regular n-gon under composition. It has presentation $D_{2n} = \langle r, s \mid r^n = s^2 = 1, rs = sr^{-1} \rangle$. Every element can be written uniquely as $s^k r^i$ with $k \in \{0,1\}$ and $0 \leq i \leq n-1$. The n rotations are $1, r, r^2, \ldots, r^{n-1}$ and the n reflections are $s, sr, sr^2, \ldots, sr^{n-1}$ (Dummit & Foote, pp. 23-28).

# Prerequisites
- **Group** — $D_{2n}$ is a group
- **Order of an element** — generators have specific orders

# Key Properties
1. $|D_{2n}| = 2n$
2. $|r| = n$, $|s| = 2$
3. Key relation: $r^i s = sr^{-i}$ for all i
4. Every element has unique form $s^k r^i$, $k \in \{0,1\}$, $0 \leq i \leq n-1$
5. $D_{2n}$ is non-abelian for $n \geq 3$
6. Every element not a power of r has order 2
7. Center: $Z(D_{2n}) = 1$ if n is odd; $Z(D_{2n}) = \{1, r^{n/2}\}$ if n is even

# Construction / Recognition
## To Construct/Create:
1. Label vertices of a regular n-gon $1, 2, \ldots, n$
2. Let r = clockwise rotation by $2\pi/n$, s = reflection through vertex 1
3. All symmetries are generated by r and s

## To Identify/Recognize:
1. Non-abelian group of order 2n generated by two elements of order 2 and n
2. Contains exactly n elements of order 2 (reflections) when n is odd

# Context & Application
Dihedral groups are used extensively as examples throughout the text. They illustrate generators and relations, non-abelian behavior, and serve as a testing ground for theorems. They also connect group theory to geometry.

# Examples
**Example 1** (p. 27): In $D_{12}$, $(sr^9)(sr^6) = s(r^9s)r^6 = s(sr^{-9})r^6 = s^2r^{-3} = r^9$.
**Example 2** (p. 25): $D_6$ has 6 elements: $\{1, r, r^2, s, sr, sr^2\}$ and $D_6 \cong S_3$.

# Common Errors
- **Error**: Writing the subscript as the number of vertices rather than the group order. **Correction**: $D_{2n}$ denotes the group of order 2n (not $D_n$), following the convention where the subscript is the order.

# Common Confusions
- **Confusion**: Thinking $rs = sr$ since both have finite order. **Clarification**: The key relation is $rs = sr^{-1}$, not $rs = sr$. The dihedral group is non-abelian.
