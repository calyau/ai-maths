# Quick Definition
A hypersurface in A^n is the algebraic set Z(f) of a single nonconstant polynomial f. Every algebraic set is a finite intersection of hypersurfaces.

# Core Definition
The algebraic set Z(f) of a nonconstant polynomial f is called a **hypersurface** in A^n (p. 672). Since k[x_1,...,x_n] is Noetherian, every algebraic set can be written as an intersection of finitely many hypersurfaces.

# Prerequisites
- **affine-algebraic-set** — Hypersurfaces are algebraic sets defined by one equation

# Key Properties
1. Defined by a single polynomial equation
2. Every algebraic set is a finite intersection of hypersurfaces
3. Conic sections are hypersurfaces in A^2

# Examples
**Example** (p. 672): The unit circle Z(x^2+y^2-1), parabola Z(y-x^2), and hyperbola Z(xy-1) are hypersurfaces in R^2.
