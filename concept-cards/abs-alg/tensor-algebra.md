# Quick Definition
The tensor algebra $\mathcal{T}(M) = \bigoplus_{k=0}^{\infty} \mathcal{T}^k(M)$ is the direct sum of all k-fold tensor powers of M, with multiplication given by concatenation of tensors. It is the universal R-algebra generated by M.

# Core Definition
For an R-module M, define $\mathcal{T}^k(M) = M^{\otimes k}$ (k copies) and $\mathcal{T}^0(M) = R$. The tensor algebra is $\mathcal{T}(M) = \bigoplus_{k=0}^{\infty} \mathcal{T}^k(M)$ with multiplication $(m_1 \otimes \cdots \otimes m_i)(m_1' \otimes \cdots \otimes m_j') = m_1 \otimes \cdots \otimes m_i \otimes m_1' \otimes \cdots \otimes m_j'$. Universal property: any R-module map $M \to A$ (an R-algebra) extends uniquely to an R-algebra homomorphism $\mathcal{T}(M) \to A$ (Dummit & Foote, Theorem 31, p. 443).

# Prerequisites
- **tensor-product** — Built from tensor products

# Key Properties
1. $\mathcal{T}(M)$ is a graded ring with $\mathcal{T}^i(M)\mathcal{T}^j(M) \subseteq \mathcal{T}^{i+j}(M)$
2. $\dim_F \mathcal{T}^k(V) = n^k$ for an n-dimensional space V
3. $\mathcal{T}(V)$ is the noncommutative polynomial algebra in the basis elements of V
4. Universal property: any module map from M to an algebra extends uniquely
