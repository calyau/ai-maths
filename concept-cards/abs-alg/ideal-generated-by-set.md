# Quick Definition
The ideal $(A)$ generated by a subset A of a ring R is the smallest ideal of R containing A; equivalently, the intersection of all ideals containing A.

# Core Definition
Let A be any subset of the ring R. $(A)$ denotes the smallest ideal of R containing A. If R is commutative with identity, then $(A) = RA = \{r_1a_1 + \cdots + r_na_n \mid r_i \in R, a_i \in A\}$. An ideal generated by a single element is *principal*; by a finite set is *finitely generated* (Dummit & Foote, pp. 251-252).

# Prerequisites
- **Ideal** — The generated object is an ideal
- **Ring with identity** — Simplifies the description of $(A)$

# Key Properties
1. $(A) = \bigcap_{I \supseteq A} I$ (intersection of all ideals containing A)
2. For commutative R with 1: $(a_1, \ldots, a_n) = Ra_1 + \cdots + Ra_n$
3. $(a) = Ra$ for principal ideals in commutative rings
4. $(A)$ may not be principal even when A is finite: $(2, x)$ in $\mathbb{Z}[x]$

# Examples
**Example 1** (p. 252): In $\mathbb{Z}$, $(n) = n\mathbb{Z}$ and $(n, m) = \gcd(n, m)\mathbb{Z}$.

**Example 2** (p. 252): In $\mathbb{Z}[x]$, $(2, x) = \{2p(x) + xq(x) \mid p, q \in \mathbb{Z}[x]\}$ is not principal.
