# Quick Definition
A subset V of affine n-space A^n is an affine algebraic set if V is the set of common zeros of some collection of polynomials, i.e., V = Z(S) for some S in k[x_1,...,x_n].

# Core Definition
Let k be a field and A^n the affine n-space over k. A subset V of A^n is called an **affine algebraic set** (or just an algebraic set) if V is the set of common zeros of some set S of polynomials: V = Z(S) = {(a_1,...,a_n) in A^n | f(a_1,...,a_n) = 0 for all f in S} (Definition, p. 672). Since k[x_1,...,x_n] is Noetherian, every algebraic set is the intersection of finitely many hypersurfaces.

# Prerequisites
- **polynomial-ring** — Algebraic sets are defined by polynomial equations
- **ideal** — Z(S) = Z(I) where I = (S) is the ideal generated by S
- **noetherian-ring** — Ensures finitely many defining equations suffice

# Key Properties
1. Z is contravariant: if S ⊆ T then Z(T) ⊆ Z(S)
2. Z(S) = Z(I) where I = (S) is the ideal generated by S
3. Intersections and finite unions of algebraic sets are algebraic sets
4. Z(0) = A^n and Z(1) = ∅
5. Every algebraic set is an intersection of finitely many hypersurfaces (by Hilbert Basis Theorem)
6. The union Z(I) ∪ Z(J) = Z(IJ)

# Construction / Recognition
## To Construct:
1. Choose polynomials f_1,...,f_m in k[x_1,...,x_n]
2. The zero set Z(f_1,...,f_m) is an affine algebraic set

## To Recognize:
1. Check if the set is the common zero locus of polynomial equations

# Context & Application
Affine algebraic sets are the fundamental geometric objects in algebraic geometry. The basic idea is to equate geometric questions with algebraic questions involving ideals in polynomial rings. This translation, made precise by the Nullstellensatz, is one of the central themes of modern algebra (p. 671).

# Examples
**Example 1** (p. 672): In A^1, the algebraic sets are ∅, any finite set, and k itself.

**Example 2** (p. 672): Points in A^n are algebraic sets: {(a_1,...,a_n)} = Z(x_1-a_1,...,x_n-a_n).

**Example 3** (p. 672): Conic sections in R^2 are algebraic sets: the unit circle is Z(x^2+y^2-1), the parabola y=x^2 is Z(y-x^2).

# Common Errors
- **Error**: Assuming different ideals always give different algebraic sets. **Correction**: Z(I) = Z(I^k) for all k ≥ 1, so many ideals can define the same set.

# Common Confusions
- **Confusion**: Confusing algebraic sets with varieties. **Clarification**: Every variety is an algebraic set, but not conversely — algebraic sets may be reducible (e.g., the union of coordinate axes).
