# Quick Definition
A cyclic module is an R-module M generated by a single element, i.e., $M = Rm = \{rm \mid r \in R\}$ for some $m \in M$.

# Core Definition
An R-module M is called cyclic if $M = Rm$ for some element $m \in M$. If R is commutative with 1, then $Rm \cong R/\text{Ann}(m)$ where $\text{Ann}(m) = \{r \in R \mid rm = 0\}$ is the annihilator of m (Dummit & Foote, p. 351).

# Prerequisites
- **left-module** — Cyclic modules are a special type of module

# Key Properties
1. A cyclic R-module is isomorphic to $R/I$ for some left ideal I
2. Every quotient of a cyclic module is cyclic
3. When $R = \mathbb{Z}$, cyclic modules are cyclic groups

# Context & Application
Cyclic modules are the simplest type of module. The structure theorem for modules over PIDs decomposes finitely generated modules into direct sums of cyclic modules.

# Examples
**Example** (p. 351): $\mathbb{Z}/n\mathbb{Z}$ is a cyclic $\mathbb{Z}$-module generated by the element 1.
