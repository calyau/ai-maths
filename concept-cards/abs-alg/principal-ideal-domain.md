# Quick Definition
A Principal Ideal Domain (PID) is an integral domain in which every ideal is principal (generated by a single element).

# Core Definition
A *Principal Ideal Domain* (P.I.D.) is an integral domain in which every ideal is principal (Dummit & Foote, p. 279).

# Prerequisites
- **Integral domain** — A PID is a special integral domain
- **Principal ideal** — Every ideal must be principal

# Key Properties
1. Every Euclidean Domain is a PID (Proposition 1, p. 274)
2. Every PID is a UFD (Theorem 14, p. 286)
3. In a PID, every nonzero prime ideal is maximal (Proposition 7, p. 283)
4. In a PID, irreducible = prime (Proposition 11, p. 284)
5. GCDs exist and can be written as linear combinations (Proposition 6, p. 282)
6. R[x] is a PID iff R is a field (Corollary 8, p. 284)
7. Ascending chains of ideals stabilize (used in proof of Theorem 14, p. 287)

# Construction / Recognition
## To Verify:
1. Confirm R is an integral domain
2. Show every ideal is generated by a single element

## To Show R is NOT a PID:
1. Find an ideal that is not principal (e.g., $(2, x)$ in $\mathbb{Z}[x]$)
2. Find an irreducible element that is not prime

# Context & Application
PIDs sit between Euclidean Domains and UFDs in the hierarchy. They retain all the factorization properties of Euclidean Domains but may lack a concrete algorithm for computing GCDs.

# Examples
**Example 1** (p. 279): $\mathbb{Z}$ is a PID (and a Euclidean Domain).

**Example 2** (p. 279): $F[x]$ for a field F is a PID.

**Example 3** (p. 279): $\mathbb{Z}[x]$ is NOT a PID (since $(2, x)$ is not principal).

**Example 4** (pp. 280-281): $\mathbb{Z}[(1+\sqrt{-19})/2]$ is a PID but NOT a Euclidean Domain.

**Example 5** (p. 279): $\mathbb{Z}[\sqrt{-5}]$ is NOT a PID (since $(3, 2+\sqrt{-5})$ is not principal).

# Relationships

## Builds Upon
- **integral-domain** — With the additional condition that all ideals are principal

## Enables
- **unique-factorization-domain** — Every PID is a UFD (Theorem 14)

## Contrasts With
- **euclidean-domain** — Every Euclidean Domain is a PID, but not conversely
- **unique-factorization-domain** — Every PID is a UFD, but not conversely ($\mathbb{Z}[x]$)

# Common Errors
- **Error**: Assuming every PID is a Euclidean Domain
  **Correction**: $\mathbb{Z}[(1+\sqrt{-19})/2]$ is a PID but not Euclidean

# Common Confusions
- **Confusion**: Confusing "every ideal is principal" with "every ideal is generated by a prime"
  **Clarification**: In a PID, every ideal $(a)$ is principal, but a need not be prime or irreducible
