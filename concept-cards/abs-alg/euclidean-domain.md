# Quick Definition
A Euclidean Domain is an integral domain R with a norm function $N: R \to \mathbb{Z}^+ \cup \{0\}$ such that for any $a, b \in R$ with $b \neq 0$, there exist $q, r \in R$ with $a = qb + r$ where $r = 0$ or $N(r) < N(b)$.

# Core Definition
The integral domain R is said to be a *Euclidean Domain* (or possess a *Division Algorithm*) if there is a norm N on R such that for any two elements a and b of R with $b \neq 0$ there exist elements q and r in R with $a = qb + r$ with $r = 0$ or $N(r) < N(b)$ (Dummit & Foote, p. 271).

# Prerequisites
- **Integral domain** — A Euclidean Domain must first be an integral domain

# Key Properties
1. Every Euclidean Domain is a PID (Proposition 1, p. 274)
2. Every ideal is principal, generated by any nonzero element of minimum norm (Proposition 1)
3. The Euclidean Algorithm computes greatest common divisors (Theorem 4, p. 276)
4. The g.c.d. can be written as a linear combination: $d = ax + by$ (Theorem 4(2))
5. Quotient and remainder need not be unique (p. 272)
6. In the hierarchy: fields $\subset$ Euclidean Domains $\subset$ PIDs $\subset$ UFDs $\subset$ integral domains (p. 290)

# Construction / Recognition
## To Verify:
1. Confirm R is an integral domain
2. Find a norm $N: R \to \mathbb{Z}^+ \cup \{0\}$ with $N(0) = 0$
3. Verify: for all $a, b \in R$ with $b \neq 0$, $\exists q, r$ with $a = qb + r$ and ($r = 0$ or $N(r) < N(b)$)

## To Show R is NOT Euclidean:
1. Find a non-principal ideal (then R is not even a PID, hence not Euclidean)
2. Or show R has no universal side divisors (Proposition 5, p. 275)

# Context & Application
Euclidean Domains are computationally the most tractable class of rings for factorization. The Euclidean Algorithm provides an efficient method for computing GCDs, which is logarithmic in the size of the inputs for $\mathbb{Z}$ (p. 278).

# Examples
**Example 1** (p. 272): $\mathbb{Z}$ is a Euclidean Domain with norm $N(a) = |a|$.

**Example 2** (p. 272): $F[x]$ for a field F is a Euclidean Domain with norm = degree.

**Example 3** (p. 273): The Gaussian integers $\mathbb{Z}[i]$ are a Euclidean Domain with $N(a+bi) = a^2 + b^2$.

**Example 4** (p. 275): $\mathbb{Z}[x]$ is NOT a Euclidean Domain since $(2, x)$ is not principal.

**Example 5** (p. 275): $\mathbb{Z}[(1+\sqrt{-19})/2]$ is a PID but NOT a Euclidean Domain.

## Contrasts With
- **principal-ideal-domain** — A PID need not be Euclidean ($\mathbb{Z}[(1+\sqrt{-19})/2]$)
- **unique-factorization-domain** — A UFD need not be a PID ($\mathbb{Z}[x]$)

# Common Errors
- **Error**: Assuming the Euclidean norm is unique
  **Correction**: A ring can be Euclidean with respect to one norm but not another

# Common Confusions
- **Confusion**: Thinking R[x] is Euclidean for any integral domain R
  **Clarification**: R[x] is Euclidean iff R is a field (Corollary 8, p. 284)
