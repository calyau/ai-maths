# Quick Definition
An R-module M is finitely generated if there exist elements $m_1, \ldots, m_n \in M$ such that every element of M can be written as $r_1 m_1 + \cdots + r_n m_n$ for some $r_i \in R$.

# Core Definition
An R-module M is finitely generated if there is a finite set $\{m_1, \ldots, m_n\} \subseteq M$ such that $M = Rm_1 + Rm_2 + \cdots + Rm_n$, i.e., every element of M is an R-linear combination of the $m_i$. Equivalently, M is a quotient of the free module $R^n$ for some n (Dummit & Foote, p. 351).

# Prerequisites
- **left-module** — Finitely generated modules are a type of module

# Key Properties
1. Every quotient of a finitely generated module is finitely generated
2. If both M/N and N are finitely generated, then M is finitely generated
3. A finitely generated module over a PID has a particularly nice structure (invariant factors or elementary divisors)
4. A finitely generated module that is generated by n elements has every quotient generated by at most n elements

# Context & Application
Finite generation is a crucial finiteness condition in module theory. The structure theorem for finitely generated modules over PIDs is one of the main results of Chapter 12.
