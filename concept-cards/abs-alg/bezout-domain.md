# Quick Definition
A Bezout Domain is an integral domain in which every ideal generated by two elements is principal: $(a, b) = (d)$ for some d.

# Core Definition
An integral domain R in which every ideal generated by two elements is principal is called a *Bezout Domain* (Exercise 7, Dummit & Foote, p. 282; also mentioned p. 276).

# Prerequisites
- **Integral domain** — Must be an integral domain
- **GCD in ring** — GCDs exist and are linear combinations

# Key Properties
1. R is Bezout iff every pair $a, b$ has a GCD expressible as $d = ax + by$ (Exercise 7(a))
2. Every finitely generated ideal is principal (Exercise 7(b))
3. A Bezout Domain need not be a PID (may have non-finitely-generated ideals)
4. $R$ is a PID iff R is a UFD and a Bezout Domain (Exercise 11, p. 296)

# Examples
**Example**: $\mathbb{Z} + x\mathbb{Q}[x]$ is a Bezout Domain that is not a PID (Exercise 12, p. 302).

# Relationships

## Related
- **principal-ideal-domain** — Every PID is Bezout; a Bezout + UFD = PID
- **unique-factorization-domain** — PID = UFD + Bezout

## Contrasts With
- **principal-ideal-domain** — A Bezout Domain may have non-principal ideals
