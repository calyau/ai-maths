# Quick Definition
A simple extension is a field extension $K = F(\alpha)$ generated by a single element $\alpha$ over F. By the Primitive Element Theorem, every finite separable extension is simple.

# Core Definition
A field extension K/F is simple if $K = F(\alpha)$ for some $\alpha \in K$. If $\alpha$ is algebraic over F with minimal polynomial $m(x)$ of degree n, then $F(\alpha) \cong F[x]/(m(x))$ and $[F(\alpha):F] = n$. If $\alpha$ is transcendental, $F(\alpha) \cong F(x)$ (rational function field) (Dummit & Foote, pp. 514-516, 595).

# Key Properties
1. $F(\alpha) = \{f(\alpha)/g(\alpha) \mid f, g \in F[x], g(\alpha) \neq 0\}$
2. If $\alpha$ is algebraic: $F(\alpha) = F[\alpha] = \{f(\alpha) \mid f \in F[x]\}$
3. Primitive Element Theorem: finite separable $\Rightarrow$ simple
