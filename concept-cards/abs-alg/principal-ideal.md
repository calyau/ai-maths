# Quick Definition
A principal ideal is an ideal generated by a single element. In a commutative ring R with identity, the principal ideal generated by $a$ is $(a) = \{ra \mid r \in R\}$.

# Core Definition
An ideal generated by a single element is called a *principal ideal* (Dummit & Foote, p. 251). When R is a commutative ring with identity and $a \in R$, the principal ideal $(a) = Ra = \{ra \mid r \in R\}$. The element $b \in R$ belongs to $(a)$ if and only if $a$ divides $b$.

# Prerequisites
- **Ideal** — A principal ideal is a special type of ideal
- **Ring with identity** — Ensures $(a) = Ra$

# Key Properties
1. $0 = (0)$ and $R = (1)$ are both principal (p. 252)
2. $b \in (a)$ iff $a \mid b$; $(a) \subseteq (b)$ iff $b \mid a$ (p. 252)
3. $(a) = (b)$ iff $a$ and $b$ are associates (differ by a unit) (Proposition 3, p. 277)
4. Every ideal of $\mathbb{Z}$ is principal: $n\mathbb{Z} = (n)$ (p. 252)
5. The ideal $(2, x)$ in $\mathbb{Z}[x]$ is *not* principal (p. 252)

# Construction / Recognition
## To Construct:
1. Choose an element $a \in R$
2. $(a) = \{ra \mid r \in R\}$ (in a commutative ring with identity)

## To Recognize:
1. Check if the ideal can be written as $(a)$ for some single element a

# Context & Application
Principal ideals connect the language of ideals with divisibility. In a PID, every ideal is principal, which guarantees unique factorization-like properties.

# Examples
**Example 1** (p. 252): In $\mathbb{Z}$, every ideal is principal: $n\mathbb{Z} = (n)$.

**Example 2** (p. 252): In $\mathbb{Z}[x]$, the ideal $(2, x)$ is not principal.

**Example 3** (p. 252): In $\mathbb{Q}[x]$, the ideal $(2, x)$ is principal since $2$ is a unit, giving $(2, x) = (1) = \mathbb{Q}[x]$.

# Common Errors
- **Error**: Assuming $(a)$ has the same meaning in different rings
  **Correction**: $(2, x)$ in $\mathbb{Z}[x]$ is not principal, but $(2, x) = (1)$ in $\mathbb{Q}[x]$

# Common Confusions
- **Confusion**: Thinking that if $(a) = (b)$ then $a = b$
  **Clarification**: $(a) = (b)$ iff $a = ub$ for some unit $u$
