# Quick Definition
A ringed space is a topological space equipped with a sheaf of rings. Spec R with its structure sheaf is an affine scheme — the basic building block of modern algebraic geometry.

# Core Definition
A **ringed space** is a pair (X, O) where X is a topological space and O is a sheaf of rings on X. An **affine scheme** is (Spec R, O) for a commutative ring R. The pair (Spec R, O) provides the foundation for Grothendieck's approach to algebraic geometry, which generalizes classical affine algebraic geometry to arbitrary commutative rings (p. 742).

# Key Properties
1. Affine schemes are the basic objects in modern algebraic geometry
2. Generalizes classical algebraic geometry beyond algebraically closed fields
3. Scheme morphisms correspond to ring homomorphisms (contravariantly)
