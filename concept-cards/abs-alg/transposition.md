# Quick Definition
A transposition is a 2-cycle $(i\ j)$ that swaps two elements and fixes all others. Every permutation can be written as a product of transpositions, and $S_n$ is generated by the set of all transpositions.

# Core Definition
A *transposition* is a 2-cycle. Any m-cycle can be written as $m-1$ transpositions: $(a_1\ a_m)(a_1\ a_{m-1})\cdots(a_1\ a_2)$. Since every permutation is a product of cycles, every permutation is a product of transpositions: $S_n = \langle T \rangle$ where $T = \{(i\ j) \mid 1 \leq i < j \leq n\}$. The parity (even/odd number) of transpositions in any such product is an invariant of the permutation (Dummit & Foote, pp. 107-108).

# Prerequisites
- **Cycle decomposition** — transpositions are the simplest cycles

# Key Properties
1. Every transposition has order 2
2. Every permutation is a product of transpositions
3. Transpositions are odd permutations: $\epsilon((i\ j)) = -1$
4. The parity of the number of transpositions is well defined
5. $S_n = \langle (1\ 2), (1\ 2\ 3\ \ldots\ n) \rangle$ (two generators suffice)

# Context & Application
Transpositions generate $S_n$ and define the sign homomorphism, which in turn defines the alternating group. They are the "elementary moves" for permutations.

# Examples
**Example 1** (p. 108): $(1\ 2\ 3\ 4\ 5) = (1\ 5)(1\ 4)(1\ 3)(1\ 2)$.
**Example 2** (p. 108): $(1\ 12\ 8\ 10\ 4)(2\ 13)(5\ 11\ 7)(6\ 9) = (1\ 4)(1\ 10)(1\ 8)(1\ 12)(2\ 13)(5\ 7)(5\ 11)(6\ 9)$.
