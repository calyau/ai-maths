# Quick Definition
In a PID, a nonzero element is irreducible if and only if it is prime. This follows because in a PID, the ideal $(p)$ generated by an irreducible p is maximal, hence prime.

# Core Definition
(Proposition 11) In a Principal Ideal Domain, a nonzero element is prime if and only if it is irreducible. Proof: irreducible p generates an ideal $(p)$ that is maximal (since any larger ideal $(m) \supseteq (p)$ forces $m \mid p$, so m is a unit or an associate of p). Maximal implies prime (Dummit & Foote, p. 284).

# Prerequisites
- **Principal ideal domain** — This equivalence requires R to be a PID
- **Irreducible element** — One direction of the equivalence
- **Prime element** — The other direction

# Key Properties
1. In a PID: irreducible = prime = generates a maximal ideal
2. In a general integral domain: prime $\Rightarrow$ irreducible, but NOT conversely
3. Example of failure: 3 is irreducible but not prime in $\mathbb{Z}[\sqrt{-5}]$ (which is not a PID)
4. This equivalence is also true in UFDs (Proposition 12)
