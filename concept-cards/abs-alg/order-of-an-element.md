# Quick Definition
The order of an element x in a group G is the smallest positive integer n such that $x^n = 1$, denoted |x|. If no such n exists, x has infinite order.

# Core Definition
For G a group and $x \in G$, the *order* of x is the smallest positive integer n such that $x^n = 1$, denoted |x|. If no positive power of x equals the identity, x has *infinite order*. The order of x equals the cardinality of the cyclic subgroup $\langle x \rangle$ generated by x (Dummit & Foote, p. 20).

# Prerequisites
- **Group** — order is defined for elements of groups

# Key Properties
1. $|x| = 1$ iff $x = 1$ (the identity)
2. $|x| = |\langle x \rangle|$ (order of element = order of cyclic subgroup it generates)
3. $|x|$ divides $|G|$ when G is finite (Lagrange's Theorem)
4. $x^{|G|} = 1$ for all x in a finite group G
5. If $x^m = 1$ then $|x|$ divides m (Proposition 2.3)
6. $|x| = |x^{-1}|$

# Construction / Recognition
## To Construct/Create:
1. Compute successive powers $x, x^2, x^3, \ldots$
2. The first n with $x^n = 1$ is $|x|$

## To Identify/Recognize:
1. In additive notation, the order of x is the smallest positive n with $nx = 0$

# Context & Application
Element orders are a primary invariant for understanding group structure. They constrain which groups can be isomorphic and, via Lagrange's Theorem, must divide the group order.

# Examples
**Example 1** (p. 20): In additive $\mathbb{Z}/9\mathbb{Z}$, $|\bar{6}| = 3$ since $3 \cdot 6 = 18 \equiv 0 \pmod{9}$.
**Example 2** (p. 20): Every nonzero element of $\mathbb{Z}, \mathbb{Q}, \mathbb{R}$ (under addition) has infinite order.

# Common Errors
- **Error**: Confusing |x| (element order) with |x| (absolute value) when $G \subseteq \mathbb{R}$. **Correction**: Context determines the meaning; the text is careful to distinguish.

# Common Confusions
- **Confusion**: Thinking $|x| = n$ means $x^n$ is the first power equaling 1. **Clarification**: It is the *smallest positive* n with $x^n = 1$; lower powers may also equal 1 is impossible by minimality.
