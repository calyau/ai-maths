# Quick Definition

The reflection $s_\alpha\colon \mathfrak{h}^* \to \mathfrak{h}^*$ defined by $s_\alpha(\lambda) = \lambda - \langle h_\alpha, \lambda \rangle \alpha$ preserves the root system: if $\beta$ is a root, so is $s_\alpha(\beta)$. These reflections generate the Weyl group.

# Core Definition

**Theorem 7.21, part (4)** (Kirillov, p. 87): For $\alpha \in R$, define

$$s_\alpha(\lambda) = \lambda - \langle h_\alpha, \lambda \rangle \alpha = \lambda - \frac{2(\alpha, \lambda)}{(\alpha, \alpha)} \alpha.$$

Then for any $\alpha, \beta \in R$, $s_\alpha(\beta) \in R$. In particular, $-\alpha = s_\alpha(\alpha) \in R$.

# Prerequisites

- **Root** — Reflections act on the root system
- **Coroot of Lie algebra** — The formula uses $\langle h_\alpha, \lambda \rangle$

# Key Properties

1. $s_\alpha$ is a reflection in the hyperplane perpendicular to $\alpha$
2. $s_\alpha(\alpha) = -\alpha$
3. $s_\alpha^2 = \operatorname{id}$
4. Proof uses $\mathfrak{sl}(2)$ representation theory: $f_\alpha^n$ maps $\mathfrak{g}_\beta$ to $\mathfrak{g}_{s_\alpha(\beta)}$

# Context & Application

Root reflections are the fundamental symmetries of the root system. They generate the Weyl group, which plays a central role in representation theory and the classification of root systems.
