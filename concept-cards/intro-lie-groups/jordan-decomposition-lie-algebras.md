# Quick Definition

In a semisimple complex Lie algebra, every element $x$ has a unique decomposition $x = x_s + x_n$ where $x_s$ is semisimple, $x_n$ is nilpotent, and $[x_s, x_n] = 0$. Moreover, $\operatorname{ad} x_s$ is a polynomial in $\operatorname{ad} x$.

# Core Definition

**Theorem 7.2** (Kirillov, p. 82): If $\mathfrak{g}$ is a semisimple complex Lie algebra, then any $x \in \mathfrak{g}$ can be uniquely written as $x = x_s + x_n$ where $x_s$ is semisimple, $x_n$ is nilpotent, and $[x_s, x_n] = 0$. Moreover, $\operatorname{ad} x_s = P(\operatorname{ad} x)$ for some polynomial $P \in t\mathbb{C}[t]$.

# Prerequisites

- **Semisimple Lie algebra** — The algebra must be semisimple (uses zero center, Der = ad)
- **Semisimple element** — The semisimple part $x_s$
- **Nilpotent element** — The nilpotent part $x_n$
- **Inner derivation identity** — Used to show $(\operatorname{ad} x)_s = \operatorname{ad} x_s$ for some $x_s \in \mathfrak{g}$

# Key Properties

1. Uniqueness: uses injectivity of $\operatorname{ad}$ (since $\mathfrak{z}(\mathfrak{g}) = 0$)
2. Existence: uses Jordan decomposition of $\operatorname{ad} x$ and the fact that $(\operatorname{ad} x)_s$ is a derivation, hence inner (Proposition 6.47)
3. $\operatorname{ad} x_s = P(\operatorname{ad} x)$ implies $x_s$ is in the Lie algebra generated by $x$
4. This is an intrinsic property of the abstract algebra, not dependent on a representation

# Context & Application

The abstract Jordan decomposition is the foundation for the theory of semisimple elements and Cartan subalgebras. It ensures that the notions of "semisimple" and "nilpotent" are intrinsic to the algebra, not dependent on how it is realized as matrices.

# Examples

**Example**: In $\mathfrak{sl}(2, \mathbb{C})$, $x = e + h = \begin{pmatrix} 1 & 1 \\ 0 & -1 \end{pmatrix}$. Then $x_s = h$ and $x_n = e$ (after diagonalization of $\operatorname{ad} x$).

- **Confusion**: Thinking the Jordan decomposition requires a matrix representation
