# Quick Definition
$P_+$ is the set of all dominant integral weights, i.e., elements $\lambda \in P$ satisfying $\langle \lambda, \alpha_i^\vee \rangle \in \mathbb{Z}_+$ for all simple roots $\alpha_i$. It parametrizes the irreducible finite-dimensional representations of $\mathfrak{g}$.

# Core Definition
**Definition 9.16** (Kirillov, p. 114): The set of all integral dominant weights is denoted by $P_+$. In the fundamental weight basis $\omega_1, \ldots, \omega_r$, we have $P_+ = \{\sum m_i \omega_i \mid m_i \in \mathbb{Z}_+\}$.

# Prerequisites
- **Dominant integral weight** -- elements of $P_+$
- **Weight lattice** -- $P_+ \subset P$

# Key Properties
1. $P_+$ is a free commutative monoid generated by the fundamental weights $\omega_1, \ldots, \omega_r$
2. $|P_+|$ is countably infinite (for $\mathfrak{g} \neq 0$)
3. There is a bijection $P_+ \leftrightarrow \{\text{irreducible finite-dimensional representations}\}$ via $\lambda \mapsto L_\lambda$
4. $P_+$ is the set of lattice points in the closed positive Weyl chamber

# Context & Application
$P_+$ is the indexing set for the complete classification of irreducible finite-dimensional representations (Corollary 9.18). Every finite-dimensional representation decomposes uniquely as $V = \bigoplus_{\lambda \in P_+} n_\lambda L_\lambda$ with $n_\lambda \in \mathbb{Z}_+$.

# Examples
**Example** (p. 123-129): Explicit descriptions of $P_+$ for each classical type:
- $A_n$: $\{(\lambda_1, \ldots, \lambda_n, 0) \mid \lambda_1 \geq \cdots \geq \lambda_n \geq 0, \lambda_i \in \mathbb{Z}\}$
- $B_n$: $\{(\lambda_1, \ldots, \lambda_n) \mid \lambda_1 \geq \cdots \geq \lambda_n \geq 0, \lambda_i \in \frac{1}{2}\mathbb{Z}, \lambda_i - \lambda_j \in \mathbb{Z}\}$
- $C_n$: $\{(\lambda_1, \ldots, \lambda_n) \mid \lambda_1 \geq \cdots \geq \lambda_n \geq 0, \lambda_i \in \mathbb{Z}\}$
- $D_n$: $\{(\lambda_1, \ldots, \lambda_n) \mid \lambda_1 \geq \cdots \geq \lambda_n, \lambda_{n-1} + \lambda_n \geq 0, \lambda_i \in \frac{1}{2}\mathbb{Z}, \lambda_i - \lambda_j \in \mathbb{Z}\}$
