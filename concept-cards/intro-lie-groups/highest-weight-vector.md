# Quick Definition
A highest-weight vector is a non-zero vector $v_\lambda \in V[\lambda]$ that is annihilated by all positive root vectors: $e_\alpha \cdot v_\lambda = 0$ for all $\alpha \in R_+$. It generates a highest-weight representation.

# Core Definition
**Definition 9.4** (Kirillov, p. 111-112): In a highest-weight representation, the highest-weight vector $v_\lambda \in V[\lambda]$ satisfies:

$$hv_\lambda = \langle h, \lambda \rangle v_\lambda \quad \forall h \in \mathfrak{h}$$
$$xv_\lambda = 0 \quad \forall x \in \mathfrak{n}_+ = \bigoplus_{R_+} \mathfrak{g}_\alpha$$

# Prerequisites
- **Weight of a representation** -- the highest-weight vector has a specific weight
- **Weight space** -- lives in $V[\lambda]$
- **Root action on weights** -- the annihilation condition uses positive root vectors

# Key Properties
1. Unique up to scalar multiple in any highest-weight module (Corollary 9.12)
2. Annihilated by all of $\mathfrak{n}_+$
3. An eigenvector for all of $\mathfrak{h}$ with eigenvalue $\lambda$
4. Generates the entire representation under $U\mathfrak{g}$ (equivalently, under $U\mathfrak{n}_-$)
5. The weight space $V[\lambda]$ is one-dimensional (Theorem 9.10(3))

# Construction / Recognition
## To find the highest-weight vector in a finite-dimensional irreducible representation
1. Compute the weight decomposition
2. Find the weight $\lambda$ that is maximal (no $\lambda + \alpha_i$ is a weight for any simple root $\alpha_i$)
3. Any non-zero vector in $V[\lambda]$ is a highest-weight vector

# Context & Application
The highest-weight vector is the "seed" from which the entire representation grows. By applying elements of $U\mathfrak{n}_-$ (products of negative root vectors) to $v_\lambda$, one obtains all of $V$. This is the key insight: the representation is completely determined by the single datum of the highest weight $\lambda$.

# Examples
**Example** (p. 112): For $\mathfrak{sl}(2, \mathbb{C})$, the highest-weight vector in $V_n$ (the $(n+1)$-dimensional irreducible) is $v_0$ satisfying $ev_0 = 0$ and $hv_0 = nv_0$.

# Relationships
## Builds Upon
- **Weight space** -- the highest-weight vector lives in a weight space
- **Root action on weights** -- the "highest" condition means no further raising is possible

## Enables
- **Highest-weight representation** -- generated by the highest-weight vector
- **Verma module** -- the universal module generated by a highest-weight vector with no other relations

# Common Errors
- **Error**: Looking for multiple linearly independent highest-weight vectors in an irreducible representation
  **Correction**: The highest-weight vector is unique up to scalar (Corollary 9.12)

# Common Confusions
- **Confusion**: Thinking a highest-weight vector is simply a vector in the highest weight space of any representation
  **Clarification**: It must also be annihilated by all positive root vectors; in a reducible representation, there may be vectors in $V[\lambda]$ that do not satisfy this
