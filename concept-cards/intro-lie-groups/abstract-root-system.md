# Quick Definition
An abstract root system is a finite set $R$ of nonzero vectors in a real inner product space $E$ satisfying axioms that ensure $R$ spans $E$, integrality of certain projections, and closure under reflections. It captures the combinatorial structure underlying semisimple Lie algebras.

# Core Definition
An abstract root system is a finite set $R \subset E \setminus \{0\}$, where $E$ is a real vector space with a positive definite inner product, such that: (R1) $R$ generates $E$ as a vector space; (R2) for any $\alpha, \beta \in R$, the number $n_{\alpha\beta} = 2(\alpha,\beta)/(\beta,\beta)$ is an integer; (R3) for any $\alpha, \beta \in R$, the reflection $s_\alpha(\beta) \in R$. The number $r = \dim E$ is called the rank of $R$ (Definition 8.1, p. 90).

# Prerequisites
This is a foundational concept. Understanding benefits from familiarity with:
- **Euclidean vector spaces** -- the ambient space $E$ with positive definite inner product
- **Reflections** -- the geometric operation underlying axiom (R3)

# Key Properties
1. $R$ is a finite subset of $E \setminus \{0\}$
2. $R$ spans $E$ (axiom R1)
3. The numbers $n_{\alpha\beta} = 2(\alpha,\beta)/(\beta,\beta)$ are integers for all $\alpha,\beta \in R$ (axiom R2)
4. $R$ is closed under reflections $s_\alpha$ for every $\alpha \in R$ (axiom R3)
5. The rank of $R$ equals $\dim E$
6. From (R1)--(R3) one can deduce that if $\alpha, c\alpha \in R$ then $c \in \{\pm 1, \pm 2, \pm 1/2\}$ (Remark 8.2)

# Construction / Recognition
## To verify a set $R$ is a root system
1. Check $R$ is finite and does not contain $0$
2. Check $R$ spans $E$ (axiom R1)
3. For every pair $\alpha,\beta \in R$, compute $n_{\alpha\beta} = 2(\alpha,\beta)/(\beta,\beta)$ and verify it is an integer (axiom R2)
4. For every pair $\alpha,\beta \in R$, compute $s_\alpha(\beta) = \beta - n_{\beta\alpha}\,\alpha$ and verify the result is in $R$ (axiom R3)

# Context & Application
The concept of abstract root system extracts the combinatorial-geometric structure of the roots of a semisimple Lie algebra (as identified in Section 7.3) into a self-contained axiomatic framework. This abstraction allows the classification of root systems -- and hence semisimple Lie algebras -- by purely combinatorial means. Root systems also appear independently in other areas of mathematics.

**Theorem 8.3** (p. 91): The set of roots $R$ of a semisimple complex Lie algebra is a reduced root system.

# Examples
**Example 8.4** (p. 91): Let $e_i$ be the standard basis of $\mathbb{R}^n$ with $(e_i,e_j) = \delta_{ij}$. Let $E = \{(\lambda_1,\ldots,\lambda_n) \mid \sum \lambda_i = 0\}$ and $R = \{e_i - e_j \mid i \neq j\}$. Then $R$ is a reduced root system of rank $n-1$, called the root system of type $A_{n-1}$. The reflection $s_{e_i - e_j}$ acts by transposing the $i$-th and $j$-th entries. This is the root system of $\mathfrak{sl}(n,\mathbb{C})$.

# Common Errors
- **Error**: Forgetting that $R$ must not contain $0$
  **Correction**: The definition explicitly requires $R \subset E \setminus \{0\}$

- **Error**: Confusing $n_{\alpha\beta}$ with $n_{\beta\alpha}$; these are generally different
  **Correction**: $n_{\alpha\beta} = 2(\alpha,\beta)/(\beta,\beta)$ depends on which root appears in the denominator

# Common Confusions
- **Confusion**: Believing axioms R1--R3 force $R$ to be reduced (no $\alpha$ and $2\alpha$ simultaneously)
  **Clarification**: Non-reduced root systems exist (e.g., $BC_n$); the reduced property requires the additional axiom R4

- **Confusion**: Thinking the inner product is part of the abstract structure up to isomorphism
  **Clarification**: Root system isomorphisms preserve the integers $n_{\alpha\beta}$ but need not preserve the inner product itself (Definition 8.5)
