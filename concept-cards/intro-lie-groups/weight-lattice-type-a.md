# Quick Definition
For type $A_n$, the weight lattice is $P = \{(\lambda_1, \ldots, \lambda_n, 0) \mid \lambda_i \in \mathbb{Z}\}$ and the root lattice is $Q = \{(\lambda_1, \ldots, \lambda_{n+1}) \mid \lambda_i \in \mathbb{Z}, \sum \lambda_i = 0\}$, with $P/Q \cong \mathbb{Z}/(n+1)\mathbb{Z}$.

# Core Definition
(Kirillov, p. 123):

$$P = \{(\lambda_1, \ldots, \lambda_{n+1}) \mid \lambda_i - \lambda_j \in \mathbb{Z}\}/\mathbb{R}(1, \ldots, 1) = \{(\lambda_1, \ldots, \lambda_n, 0) \mid \lambda_i \in \mathbb{Z}\}$$

$$Q = \{(\lambda_1, \ldots, \lambda_{n+1}) \mid \lambda_i \in \mathbb{Z}, \sum \lambda_i = 0\}$$

$$P/Q \cong \mathbb{Z}/(n+1)\mathbb{Z}$$

# Key Properties
1. $P/Q \cong \mathbb{Z}/(n+1)\mathbb{Z}$ -- the quotient group has order $n+1$
2. Dominant weights: $P_+ = \{(\lambda_1, \ldots, \lambda_n, 0) \mid \lambda_1 \geq \cdots \geq \lambda_n \geq 0, \lambda_i \in \mathbb{Z}\}$
3. Positive Weyl chamber: $C_+ = \{\lambda_1 > \lambda_2 > \cdots > \lambda_n > 0\}$
4. The weight lattice is generated by fundamental weights $\omega_k = e_1 + \cdots + e_k = (1, \ldots, 1, 0, \ldots, 0)$

# Examples
**Example** (p. 123): For $A_1$: $P = \mathbb{Z}$, $Q = 2\mathbb{Z}$, $P/Q = \mathbb{Z}/2\mathbb{Z}$.

**Example** (p. 123): For $A_2$: $P = \{(\lambda_1, \lambda_2, 0) \mid \lambda_i \in \mathbb{Z}\}$, $P/Q \cong \mathbb{Z}/3\mathbb{Z}$.
