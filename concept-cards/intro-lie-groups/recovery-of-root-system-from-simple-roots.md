# Quick Definition
The full root system $R$ can be recovered from the simple roots $\Pi$: generate the Weyl group $W$ from simple reflections $s_i = s_{\alpha_i}$, then $R = W(\Pi)$ (every root is a $W$-image of a simple root).

# Core Definition
**Corollary 8.33** (p. 101): The root system $R$ can be recovered from the set of simple roots $\Pi$.

**Proof**: Given $\Pi$, recover $W$ as the group generated by $s_i$; then $R = W(\Pi)$ (by Theorem 8.30(2)).

# Prerequisites
- **simple-roots** -- the starting data
- **simple-reflections** -- generate the Weyl group
- **weyl-group** -- applies to $\Pi$ to produce $R$

# Key Properties
1. $R = W(\Pi)$: every root is $w(\alpha_i)$ for some $w \in W$, $\alpha_i \in \Pi$
2. This justifies classifying root systems by classifying possible sets $\Pi$ (equivalently, Dynkin diagrams)
3. The procedure is constructive: enumerate $W$-orbits of simple roots

# Construction / Recognition
## To recover $R$ from $\Pi = \{\alpha_1,\ldots,\alpha_r\}$
1. Compute simple reflections $s_i(\lambda) = \lambda - \langle\lambda,\alpha_i^\vee\rangle\alpha_i$
2. Starting from each $\alpha_i$, repeatedly apply all $s_j$
3. Continue until no new roots are produced
4. The resulting set is $R$

# Context & Application
This result is the theoretical justification for the entire classification program: since $R$ is determined by $\Pi$, and $\Pi$ is encoded by the Dynkin diagram, classifying diagrams classifies root systems.

# Examples
**Example**: For $A_2$ with $\Pi = \{\alpha_1, \alpha_2\}$: applying $s_1$ to $\alpha_2$ gives $\alpha_1 + \alpha_2$, and the negatives give all of $R = \{\pm\alpha_1, \pm\alpha_2, \pm(\alpha_1+\alpha_2)\}$.
