# Quick Definition

The center $Z\mathfrak{g} = Z(U\mathfrak{g})$ consists of elements that commute with all of $U\mathfrak{g}$, equivalently the $\mathrm{ad}$-invariant elements $(U\mathfrak{g})^{\mathrm{ad}\,\mathfrak{g}}$. By Schur's lemma, each central element acts as a scalar on every irreducible representation.

# Core Definition

**Proposition 4.56(2)** (Kirillov, p. 52): The center $Z\mathfrak{g} = Z(U\mathfrak{g})$ coincides with the space of invariants under the adjoint action: $Z\mathfrak{g} = (U\mathfrak{g})^{\mathrm{ad}\,\mathfrak{g}}$.

An element $C \in U\mathfrak{g}$ is central iff $\mathrm{ad}\, x \cdot C = xC - Cx = 0$ for all $x \in \mathfrak{g}$.

# Prerequisites

# Key Properties

1. Central elements act as scalars on irreducible representations (Schur)
2. Eigenspaces of central elements are subrepresentations
3. The Casimir operator is in $Z\mathfrak{g}$
4. For $\mathfrak{sl}(2,\mathbb{C})$: $Z\mathfrak{g}$ is the polynomial algebra $\mathbb{C}[C]$ (generated by the Casimir)
