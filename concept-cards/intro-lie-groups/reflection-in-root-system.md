# Quick Definition
The reflection $s_\alpha$ associated to a root $\alpha$ is the orthogonal reflection of $E$ across the hyperplane $L_\alpha = \{\lambda \in E \mid (\alpha,\lambda) = 0\}$. It sends $\alpha$ to $-\alpha$ and fixes every vector orthogonal to $\alpha$.

# Core Definition
For a root $\alpha \in R$, the reflection $s_\alpha : E \to E$ is defined by (equation 8.2, p. 90):

$$s_\alpha(\lambda) = \lambda - \frac{2(\alpha,\lambda)}{(\alpha,\alpha)}\,\alpha$$

Equivalently, using coroots: $s_\alpha(\lambda) = \lambda - \langle\lambda, \alpha^\vee\rangle\,\alpha$ (equation 8.5, p. 91).

It is characterized by: $s_\alpha(\lambda) = \lambda$ if $(\alpha,\lambda) = 0$, and $s_\alpha(\alpha) = -\alpha$.

# Key Properties
1. $s_\alpha$ is an orthogonal transformation (it preserves the inner product)
2. $s_\alpha(\alpha) = -\alpha$
3. $s_\alpha$ fixes pointwise the hyperplane $L_\alpha = \{\lambda \mid (\alpha,\lambda) = 0\}$
4. $s_\alpha^2 = \mathrm{id}$ (reflections are involutions)
5. By axiom (R3), $s_\alpha(R) = R$ for every $\alpha \in R$
6. Conjugation formula: $s_{w(\alpha)} = w\,s_\alpha\,w^{-1}$ for $w \in W$ (Lemma 8.7)

# Construction / Recognition
## To compute $s_\alpha(\beta)$
1. Compute $n_{\beta\alpha} = 2(\alpha,\beta)/(\alpha,\alpha)$
2. Apply $s_\alpha(\beta) = \beta - n_{\beta\alpha}\,\alpha$

# Context & Application
Reflections are the building blocks of the Weyl group and are central to root system theory. Axiom (R3) requires root systems to be invariant under all their reflections, which constrains the geometry severely and enables the classification. The conjugation formula (Lemma 8.7) shows how the Weyl group permutes the reflections.

# Examples
**Example** (p. 91): For $A_{n-1}$, the reflection $s_{e_i - e_j}$ acts by transposing the $i$-th and $j$-th coordinates:
$$s_{e_i - e_j}(\ldots,\lambda_i,\ldots,\lambda_j,\ldots) = (\ldots,\lambda_j,\ldots,\lambda_i,\ldots)$$

# Common Errors
- **Error**: Computing $s_\alpha(\beta) = \beta - 2(\alpha,\beta)\alpha$ (forgetting to divide by $(\alpha,\alpha)$)
  **Correction**: The correct formula is $s_\alpha(\beta) = \beta - \frac{2(\alpha,\beta)}{(\alpha,\alpha)}\alpha$

# Common Confusions
- **Confusion**: Thinking $s_\alpha$ depends on the length of $\alpha$
  **Clarification**: The reflection $s_\alpha$ depends only on the direction of $\alpha$ (the hyperplane $L_\alpha$), not its magnitude; $s_\alpha = s_{c\alpha}$ for any $c \neq 0$
