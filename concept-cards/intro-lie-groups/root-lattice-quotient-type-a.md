# Quick Definition
For type $A_n$, the quotient of the weight lattice by the root lattice is $P/Q \cong \mathbb{Z}/(n+1)\mathbb{Z}$, a cyclic group of order $n+1$.

# Core Definition
(Kirillov, p. 123): $P/Q \cong \mathbb{Z}/(n+1)\mathbb{Z}$.

# Prerequisites
- **Weight lattice type A** -- provides $P$ and $Q$

# Key Properties
1. $|P/Q| = n+1 = \det A$ where $A$ is the Cartan matrix
2. $P/Q$ is isomorphic to the center of the simply-connected group $\mathrm{SL}(n+1, \mathbb{C})$
3. The non-trivial cosets correspond to representations that do not descend to $\mathrm{PGL}(n+1)$
4. Generated by the class of $\omega_1$

# Examples
**Example** (p. 123): For $A_1$: $P/Q \cong \mathbb{Z}/2\mathbb{Z}$. For $A_2$: $P/Q \cong \mathbb{Z}/3\mathbb{Z}$.
