# Quick Definition
The Verma module $M_\lambda$ is the universal highest-weight representation with highest weight $\lambda$: it is generated by a highest-weight vector $v_\lambda$ subject only to the relations $hv_\lambda = \langle \lambda, h \rangle v_\lambda$ and $\mathfrak{n}_+ v_\lambda = 0$. Every highest-weight module with highest weight $\lambda$ is a quotient of $M_\lambda$.

# Core Definition
(Kirillov, p. 112): The Verma module is defined as:

$$M_\lambda = U\mathfrak{g} / I_\lambda$$

where $I_\lambda$ is the left ideal in $U\mathfrak{g}$ generated by $(h - \langle h, \lambda \rangle)$ for $h \in \mathfrak{h}$ and $\mathfrak{n}_+$. Equivalently, as an induced representation:

$$M_\lambda = \mathrm{Ind}_{U\mathfrak{b}}^{U\mathfrak{g}} \mathbb{C}_\lambda = U\mathfrak{g} \otimes_{U\mathfrak{b}} \mathbb{C}_\lambda$$

where $\mathfrak{b} = \mathfrak{h} \oplus \mathfrak{n}_+$ is the Borel subalgebra and $\mathbb{C}_\lambda$ is the one-dimensional representation of $\mathfrak{b}$ defined by the highest-weight conditions.

# Prerequisites
- **Highest-weight representation** -- the Verma module is the universal one
- **Highest-weight vector** -- the generating vector $v_\lambda$
- **Borel subalgebra** -- used in the induction construction
- **Nil-negative subalgebra** -- provides the vector space structure
- **Universal enveloping algebra** -- $M_\lambda$ is a quotient of $U\mathfrak{g}$

# Key Properties
1. **Universality** (Lemma 9.8): Every highest-weight module with highest weight $\lambda$ is isomorphic to $M_\lambda / W$ for some submodule $W$
2. **Basis** (Theorem 9.9): The map $U\mathfrak{n}_- \to M_\lambda$, $u \mapsto uv_\lambda$ is an isomorphism of vector spaces
3. **Weight decomposition** (Theorem 9.9): $P(M_\lambda) = \lambda - Q_+$ where $Q_+ = \{\sum n_i \alpha_i \mid n_i \in \mathbb{Z}_+\}$
4. **One-dimensional top**: $\dim M_\lambda[\lambda] = 1$
5. **Typically infinite-dimensional**: $M_\lambda$ is infinite-dimensional (unless $\mathfrak{g} = 0$)
6. **Unique maximal proper submodule**: $J_\lambda \subset M_\lambda$ (proof of Theorem 9.13)

# Construction / Recognition
## To construct $M_\lambda$
1. Fix $\lambda \in \mathfrak{h}^*$
2. Define the one-dimensional $\mathfrak{b}$-module $\mathbb{C}_\lambda$ where $\mathfrak{h}$ acts by $\lambda$ and $\mathfrak{n}_+$ acts by 0
3. Induce: $M_\lambda = U\mathfrak{g} \otimes_{U\mathfrak{b}} \mathbb{C}_\lambda$
4. Equivalently, take the quotient $U\mathfrak{g} / I_\lambda$

## To recognize a Verma module
1. Check that it is a highest-weight module
2. Check that $U\mathfrak{n}_- \to V$, $u \mapsto uv_\lambda$ is injective (not just surjective)

# Context & Application
Verma modules are the key construction in the representation theory of semisimple Lie algebras. Though typically infinite-dimensional and reducible, they serve as the building blocks: every irreducible highest-weight module $L_\lambda$ is obtained as $M_\lambda / J_\lambda$ where $J_\lambda$ is the unique maximal proper submodule. The study of submodules of Verma modules leads to deep results including the BGG resolution and the Kazhdan-Lusztig conjecture.

# Examples
**Example 9.7** (p. 112): For $\mathfrak{g} = \mathfrak{sl}(2, \mathbb{C})$ with $\lambda \in \mathbb{C}$, the Verma module $M_\lambda$ has basis $\{f^k v_\lambda \mid k \geq 0\}$ with $hf^k v_\lambda = (\lambda - 2k)f^k v_\lambda$, as described in Lemma 5.5.

**Example 9.14** (p. 114): For $\mathfrak{sl}(2, \mathbb{C})$, $M_\lambda = L_\lambda$ (Verma module is irreducible) when $\lambda \notin \mathbb{Z}_+$.

## Contrasts With
- **Irreducible highest-weight module** -- $L_\lambda$ is the irreducible quotient; $M_\lambda$ is typically reducible and always infinite-dimensional

# Common Errors
- **Error**: Assuming the Verma module is finite-dimensional
  **Correction**: Verma modules are always infinite-dimensional (their weight set $\lambda - Q_+$ is infinite)

- **Error**: Assuming submodules of Verma modules are Verma modules
  **Correction**: Submodules generated by singular vectors are Verma modules (Lemma 9.20), but not all submodules have this form

# Common Confusions
- **Confusion**: Thinking $M_\lambda = L_\lambda$ always
  **Clarification**: $M_\lambda$ is irreducible (hence equals $L_\lambda$) only for "generic" $\lambda$; for $\lambda \in P_+$, $M_\lambda$ is strictly larger than $L_\lambda$

- **Confusion**: Confusing Verma modules with irreducible highest-weight modules
  **Clarification**: $M_\lambda$ is the universal (largest) highest-weight module; $L_\lambda$ is the irreducible (smallest non-zero) quotient. They coincide only when $M_\lambda$ has no proper submodules.
