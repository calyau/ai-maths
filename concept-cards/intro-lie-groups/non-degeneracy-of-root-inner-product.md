# Quick Definition

For every root $\alpha$, $(\alpha, \alpha) = (H_\alpha, H_\alpha) \neq 0$. This is essential for defining the coroot $h_\alpha = 2H_\alpha/(\alpha, \alpha)$ and the $\mathfrak{sl}(2)$ subalgebra.

# Core Definition

**Lemma 7.19, part (1)** (Kirillov, p. 87): Let $\alpha \in R$. Then $(\alpha, \alpha) = (H_\alpha, H_\alpha) \neq 0$.

Proof by contradiction: if $(\alpha, \alpha) = 0$, then $\langle H_\alpha, \alpha \rangle = 0$. Choosing $e \in \mathfrak{g}_\alpha, f \in \mathfrak{g}_{-\alpha}$ with $(e, f) \neq 0$, the subalgebra generated by $e, f, h = (e,f)H_\alpha$ is solvable (since $[h, e] = [h, f] = 0$). By Lie's theorem, $\operatorname{ad} h$ is nilpotent. But $h \in \mathfrak{h}$ is also semisimple, so $h = 0$, contradicting $(e, f) \neq 0$.

# Prerequisites

- **Root** — The root $\alpha$
- **Killing form on $\mathfrak{h}$** — Defines the inner product

# Key Properties

1. Proved by contradiction using the solvable/semisimple dichotomy
2. Essential for the normalization defining $h_\alpha$
3. Shows the Killing form is positive definite on the real span of roots (confirmed by Theorem 7.22)

# Relationships

## Enables
- **Coroot of Lie algebra** — The normalization $h_\alpha = 2H_\alpha/(\alpha, \alpha)$ requires $(\alpha, \alpha) \neq 0$
- **sl(2) subalgebra** — Cannot be constructed if $(\alpha, \alpha) = 0$
