# Quick Definition
The root lattice $Q$ is the abelian group generated by all roots $\alpha \in R$. Equivalently, $Q = \bigoplus_{i=1}^r \mathbb{Z}\alpha_i$, the integer span of the simple roots.

# Core Definition
The root lattice is defined by (equation 8.9, p. 96):

$$Q = \{\text{abelian group generated by } \alpha \in R\} \subset E$$

Since every root is an integer combination of simple roots (Corollary 8.18), one has $Q = \bigoplus \mathbb{Z}\alpha_i$ (equation 8.10, p. 97), confirming $Q$ is indeed a lattice.

# Prerequisites
- **abstract-root-system** -- the root lattice is built from roots

# Key Properties
1. $Q = \bigoplus_{i=1}^r \mathbb{Z}\alpha_i$ (simple roots form a $\mathbb{Z}$-basis)
2. $Q$ is a sublattice of the weight lattice: $Q \subset P$ (p. 98)
3. $Q$ has rank $r = \dim E$
4. The quotient $P/Q$ is a finite abelian group with $|P/Q| = |\det A|$ where $A$ is the Cartan matrix (Exercise 8.4)

# Construction / Recognition
1. Choose simple roots $\Pi = \{\alpha_1,\ldots,\alpha_r\}$
2. $Q = \mathbb{Z}\alpha_1 \oplus \cdots \oplus \mathbb{Z}\alpha_r$

# Context & Application
The root lattice is one of two fundamental lattices associated to a root system. In representation theory, the root lattice records which weights differ by roots (and hence are related by the action of raising/lowering operators). The quotient $P/Q$ classifies the center of the simply-connected group.

# Examples
**Example 8.20** (p. 97): For $A_1$, $Q = \mathbb{Z}\alpha$ where $(\alpha,\alpha) = 2$.

**Example 8.21** (p. 97): For $A_2$, $Q$ is the triangular lattice generated by $\alpha_1, \alpha_2$; shown as bold dots in Figure 8.3.

# Common Errors
- **Error**: Confusing the root lattice with the set of roots $R$
  **Correction**: $Q$ is the $\mathbb{Z}$-span of $R$; most elements of $Q$ are not roots
