# Quick Definition
The triangular decomposition is $\mathfrak{g} = \mathfrak{n}_- \oplus \mathfrak{h} \oplus \mathfrak{n}_+$, splitting the Lie algebra into negative root spaces, Cartan subalgebra, and positive root spaces. At the level of the universal enveloping algebra, this gives $U\mathfrak{g} \cong U\mathfrak{n}_- \otimes U\mathfrak{h} \otimes U\mathfrak{n}_+$.

# Core Definition
(Kirillov, p. 112, referencing Corollary 4.63): The semisimple Lie algebra decomposes as:

$$\mathfrak{g} = \mathfrak{n}_- \oplus \mathfrak{h} \oplus \mathfrak{n}_+$$

At the enveloping algebra level (PBW theorem corollary):

$$U\mathfrak{g} \cong U\mathfrak{n}_- \otimes U\mathfrak{h} \otimes U\mathfrak{n}_+$$

# Prerequisites
- **Cartan subalgebra** -- the middle term $\mathfrak{h}$
- **Nil-positive subalgebra** -- $\mathfrak{n}_+$
- **Nil-negative subalgebra** -- $\mathfrak{n}_-$
- **Root decomposition** -- provides the decomposition

# Key Properties
1. This is a vector space decomposition (not a Lie algebra decomposition)
2. The enveloping algebra factorization is used in the proof of Theorem 9.9 (Verma module basis)
3. Generalizes the decomposition of $\mathfrak{gl}(n)$ into lower-triangular + diagonal + upper-triangular
4. The Borel subalgebra is $\mathfrak{b} = \mathfrak{h} \oplus \mathfrak{n}_+$

# Context & Application
The triangular decomposition is the organizational principle behind the highest-weight theory. The highest-weight vector is annihilated by $\mathfrak{n}_+$, is an eigenvector for $\mathfrak{h}$, and the representation is generated by applying $\mathfrak{n}_-$.

# Examples
**Example**: For $\mathfrak{sl}(2, \mathbb{C})$: $\mathfrak{n}_- = \mathbb{C}f$, $\mathfrak{h} = \mathbb{C}h$, $\mathfrak{n}_+ = \mathbb{C}e$.
