# Quick Definition
Every highest-weight module with highest weight $\lambda$ is a quotient of the Verma module $M_\lambda$. This universal property means that the study of highest-weight representations reduces to studying submodules of Verma modules.

# Core Definition
**Lemma 9.8** (Kirillov, p. 112): If $V$ is a highest weight module with highest weight $\lambda$, then $V \cong M_\lambda / W$ for some submodule $W \subset M_\lambda$.

# Prerequisites
- **Verma module** -- the universal object
- **Highest-weight representation** -- the quotients

# Key Properties
1. $M_\lambda$ maps surjectively onto any highest-weight module with weight $\lambda$
2. The kernel $W$ determines the quotient completely
3. $L_\lambda = M_\lambda / J_\lambda$ where $J_\lambda$ is the unique maximal $W$

# Context & Application
The universal property is what makes Verma modules so useful: instead of studying the enormous class of all highest-weight representations, one studies the lattice of submodules within a single Verma module. Each submodule gives a quotient, and the maximal proper submodule gives the irreducible.

# Examples
**Example** (p. 112): For $\mathfrak{sl}(2, \mathbb{C})$ with $\lambda = n \in \mathbb{Z}_+$: the submodule generated by $f^{n+1}v_\lambda$ gives the quotient $L_n = M_n / \langle f^{n+1}v_\lambda \rangle$.
