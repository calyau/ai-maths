# Quick Definition

A filtered algebra is an algebra $A$ equipped with an increasing sequence of subspaces $F^0 A \subset F^1 A \subset \cdots$ with $A = \bigcup F^p A$ and $F^p A \cdot F^q A \subset F^{p+q} A$. The universal enveloping algebra $U\mathfrak{g}$ carries a natural filtration by degree.

# Core Definition

(Kirillov, pp. 52-53): The filtration on $U\mathfrak{g}$ is defined by $F^k U\mathfrak{g} = $ subspace generated by products $x_1 \cdots x_p$, $p \leq k$, where $x_i \in \mathfrak{g}$.

**Proposition 4.57**:
1. $U\mathfrak{g}$ is a filtered algebra: $F^p \cdot F^q \subset F^{p+q}$
2. If $x \in F^p$, $y \in F^q$, then $xy - yx \in F^{p+q-1}$ (the associated graded is commutative)
3. Ordered monomials $x_1^{k_1} \cdots x_n^{k_n}$ with $\sum k_i \leq p$ span $F^p U\mathfrak{g}$

# Prerequisites

- **Universal enveloping algebra** — The primary example

# Key Properties

1. Each $F^p U\mathfrak{g}$ is finite-dimensional (Corollary 4.58)
2. The associated graded $\mathrm{Gr}\, U\mathfrak{g} = \bigoplus_p F^p / F^{p-1}$ is commutative (Corollary 4.59)
3. $\mathrm{Gr}\, U\mathfrak{g} \cong S\mathfrak{g}$ by PBW

# Context & Application

The filtration is needed because $U\mathfrak{g}$ is not graded: the relation $xy - yx = [x,y]$ prevents defining a consistent degree. The filtration captures the "approximate" degree structure.

# Examples

**Example**: For $U\mathfrak{sl}(2,\mathbb{C})$: $F^0 = \mathbb{C}$, $F^1 = \mathbb{C} \oplus \mathrm{span}\{e, f, h\}$, $F^2$ also includes $e^2, ef, eh, f^2, fh, h^2$, etc.
