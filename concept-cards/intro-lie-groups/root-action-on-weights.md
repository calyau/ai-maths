# Quick Definition
If $x \in \mathfrak{g}_\alpha$ is a root vector and $V[\lambda]$ is a weight space, then $x$ maps $V[\lambda]$ into $V[\lambda + \alpha]$. Root vectors shift weights by the corresponding root.

# Core Definition
**Lemma 9.3** (Kirillov, p. 111): If $x \in \mathfrak{g}_\alpha$, then $x \cdot V[\lambda] \subset V[\lambda + \alpha]$.

# Prerequisites
- **Weight space** -- the domain and codomain of the root action
- **Weight decomposition theorem** -- ensures weight spaces exist
- **Root decomposition** -- provides the root spaces $\mathfrak{g}_\alpha$

# Key Properties
1. Positive root vectors ($\alpha \in R_+$) raise weights: they map $V[\lambda] \to V[\lambda + \alpha]$
2. Negative root vectors ($\alpha \in R_-$) lower weights: they map $V[\lambda] \to V[\lambda + \alpha]$ with $\alpha < 0$
3. If $\lambda + \alpha \notin P(V)$, then $x \cdot V[\lambda] = 0$ for $x \in \mathfrak{g}_\alpha$
4. This is the mechanism by which the representation "moves" between weight spaces

# Construction / Recognition
The proof is analogous to the $\mathfrak{sl}(2, \mathbb{C})$ case (Lemma 5.2): for $v \in V[\lambda]$ and $x \in \mathfrak{g}_\alpha$, compute $h \cdot (xv) = [h, x]v + x(hv) = \langle \alpha, h \rangle xv + \langle \lambda, h \rangle xv = \langle \lambda + \alpha, h \rangle xv$.

# Context & Application
This lemma is the key structural result connecting the root decomposition of $\mathfrak{g}$ to the weight decomposition of representations. It explains why the set of weights of a highest-weight module lies in $\lambda - Q_+$: starting from the highest weight, applying negative root vectors successively lowers the weight by positive root combinations. It also explains why a highest-weight vector (annihilated by all positive root vectors) generates the entire representation.

# Examples
**Example** (p. 111): For $\mathfrak{sl}(2, \mathbb{C})$, $e \in \mathfrak{g}_\alpha$ raises the weight by 2, and $f \in \mathfrak{g}_{-\alpha}$ lowers it by 2, recovering the familiar action from Section 5.1.

# Common Errors
- **Error**: Thinking root vectors always produce non-zero vectors in the target weight space
  **Correction**: If $\lambda + \alpha$ is not a weight, the action is zero

# Common Confusions
- **Confusion**: Confusing the root action on weights (shifting $\lambda \mapsto \lambda + \alpha$) with the Weyl group action on weights (reflecting)
  **Clarification**: Root vectors shift weights additively; the Weyl group acts by reflections
