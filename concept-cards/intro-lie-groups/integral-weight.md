# Quick Definition
An integral weight is an element $\lambda \in \mathfrak{h}^*$ that pairs to an integer with every coroot: $\langle \lambda, \alpha^\vee \rangle \in \mathbb{Z}$ for all $\alpha \in R$. Equivalently, it is an element of the weight lattice $P$.

# Core Definition
**Theorem 9.2** (Kirillov, p. 110): All weights of a finite-dimensional representation are integral: $P(V) \subset P$, where $P$ is the weight lattice defined in Section 8.5. An element $\lambda \in \mathfrak{h}^*$ is integral when $\langle \lambda, \alpha^\vee \rangle \in \mathbb{Z}$ for all roots $\alpha \in R$.

# Prerequisites
- **Weight of a representation** -- integral weights are a special class of weights
- **Weight lattice** -- integral weights form the weight lattice $P$
- **Coroot** -- the integrality condition is stated in terms of coroots

# Key Properties
1. The set of all integral weights forms a lattice $P \subset \mathfrak{h}^*$ (the weight lattice)
2. Every weight of every finite-dimensional representation is integral (Theorem 9.2)
3. The root lattice $Q \subset P$ is a sublattice of the weight lattice
4. $P$ has a basis of fundamental weights $\omega_1, \ldots, \omega_r$ dual to the simple coroots
5. The integrality condition follows from restricting to $\mathfrak{sl}(2, \mathbb{C})$ subalgebras (proof of Theorem 9.2)

# Construction / Recognition
## To check if $\lambda$ is integral
1. Compute $\langle \lambda, \alpha_i^\vee \rangle$ for each simple root $\alpha_i$
2. If all values are integers, then $\lambda \in P$
3. Equivalently, write $\lambda = \sum m_i \omega_i$ and check that all $m_i \in \mathbb{Z}$

# Context & Application
The integrality condition is the first constraint on weights of representations. It arises because restricting to any $\mathfrak{sl}(2, \mathbb{C})$ subalgebra (generated by $e_\alpha, f_\alpha, h_\alpha$), the eigenvalues of $h_\alpha$ must be integers. This is the Lie algebra analog of the requirement that representations of compact groups have integer characters.

# Examples
**Example** (p. 110-111): The proof uses the $\mathfrak{sl}(2, \mathbb{C})$ subalgebra for each root $\alpha$: the eigenvalue $\langle \lambda, h_\alpha \rangle$ must be an integer since weights of $\mathfrak{sl}(2, \mathbb{C})$ representations are integers.

**Example** (p. 123): For $A_n = \mathfrak{sl}(n+1, \mathbb{C})$, the weight lattice is $P = \{(\lambda_1, \ldots, \lambda_n, 0) \mid \lambda_i \in \mathbb{Z}\}$.

# Relationships
## Builds Upon
- **Weight lattice** -- integral weights are exactly the elements of $P$

## Enables
- **Dominant integral weight** -- adding the positivity condition to integrality
- **Classification of irreducible representations** -- only integral dominant weights yield finite-dimensional irreducibles

## Contrasts With
- **Dominant integral weight** -- dominant integral weights additionally satisfy $\langle \lambda, \alpha_i^\vee \rangle \geq 0$

# Common Errors
- **Error**: Checking integrality only against simple roots, not all roots
  **Correction**: It suffices to check against simple roots (since simple coroots form a basis for the coroot lattice), but this requires proof (Exercise 8.2)

# Common Confusions
- **Confusion**: Thinking integral weight means the coordinates are integers in some basis
  **Clarification**: Integrality is defined intrinsically via coroot pairings; in the fundamental weight basis, it does mean integer coordinates
