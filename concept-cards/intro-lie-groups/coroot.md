# Quick Definition
The coroot $\alpha^\vee$ of a root $\alpha$ is the element of $E^*$ defined by $\langle\alpha^\vee,\lambda\rangle = 2(\alpha,\lambda)/(\alpha,\alpha)$. It provides a convenient reformulation of the integrality condition and reflection formula.

# Core Definition
For every root $\alpha \in R$, the corresponding coroot $\alpha^\vee \in E^*$ is defined by (equation 8.4, p. 91):

$$\langle\alpha^\vee, \lambda\rangle = \frac{2(\alpha,\lambda)}{(\alpha,\alpha)}$$

For the root system of a semisimple Lie algebra, $\alpha^\vee$ coincides with the element $h_\alpha \in \mathfrak{h}$ defined by equation (7.4).

# Key Properties
1. $\langle\alpha^\vee, \alpha\rangle = 2$
2. $n_{\alpha\beta} = \langle\alpha, \beta^\vee\rangle$ (equation 8.5, p. 91)
3. $s_\alpha(\lambda) = \lambda - \langle\lambda, \alpha^\vee\rangle\,\alpha$ (equation 8.5, p. 91)
4. The set $R^\vee = \{\alpha^\vee \mid \alpha \in R\}$ is itself a root system, called the dual root system (Exercise 8.2)
5. The map $\alpha \mapsto \alpha^\vee$ is not linear (it involves dividing by $(\alpha,\alpha)$)

# Construction / Recognition
## To compute $\alpha^\vee$
1. Compute $(\alpha,\alpha)$
2. Define the linear functional $\alpha^\vee(\lambda) = 2(\alpha,\lambda)/(\alpha,\alpha)$
3. If identifying $E^* \simeq E$ via the inner product, then $\alpha^\vee = 2\alpha/(\alpha,\alpha)$

# Context & Application
Coroots provide a clean notation for root system computations. The integrality axiom (R2) becomes $\langle\alpha,\beta^\vee\rangle \in \mathbb{Z}$, and the reflection formula becomes $s_\alpha(\lambda) = \lambda - \langle\lambda,\alpha^\vee\rangle\alpha$. The weight lattice is defined as the dual lattice to the coroot lattice, making coroots central to representation theory.

# Examples
**Example** (p. 92): For $A_1$ with $(\alpha,\alpha) = 2$, identifying $E^* \simeq E$ gives $\alpha^\vee = \alpha$, so $Q^\vee \cong Q$.

**Example**: For any root $\alpha$ with $(\alpha,\alpha) = 2$ (as in $A_{n-1}$), one has $\alpha^\vee = \alpha$ under the identification $E^* \simeq E$.

# Common Errors
- **Error**: Treating $\alpha \mapsto \alpha^\vee$ as a linear map
  **Correction**: It is not linear; $(\alpha+\beta)^\vee \neq \alpha^\vee + \beta^\vee$ in general because $(\alpha,\alpha)$ appears in the denominator

# Common Confusions
- **Confusion**: Thinking $\alpha^\vee = \alpha$ always
  **Clarification**: $\alpha^\vee = 2\alpha/(\alpha,\alpha)$ under $E^* \simeq E$; this equals $\alpha$ only when $(\alpha,\alpha) = 2$
