# Quick Definition
Every irreducible finite-dimensional representation of a semisimple Lie algebra is a highest-weight representation. This fundamental theorem reduces the classification of irreducibles to the study of highest-weight modules.

# Core Definition
**Theorem 9.5** (Kirillov, p. 111): Every irreducible finite-dimensional representation of $\mathfrak{g}$ is a highest-weight representation.

**Proof sketch**: Among the finitely many weights $P(V)$, choose $\lambda$ maximal in the sense that $\lambda + \alpha \notin P(V)$ for all $\alpha \in R_+$. Any non-zero $v \in V[\lambda]$ satisfies $e_\alpha v = 0$ for all positive roots. The subrepresentation generated by $v$ is a highest-weight representation; by irreducibility, it equals $V$.

# Prerequisites
- **Weight decomposition theorem** -- ensures finitely many weights
- **Root action on weights** -- positive root vectors move weights up

# Key Properties
1. The proof is constructive: maximize a linear functional on the weight set to find the highest weight
2. Uses irreducibility only at the last step: the subrepresentation generated by $v$ must be all of $V$
3. Combined with Theorem 9.13 (uniqueness of $L_\lambda$), this gives: every irreducible finite-dimensional representation is $L_\lambda$ for some $\lambda$

# Context & Application
This is the bridge between the general representation theory (complete reducibility) and the specific classification via highest weights. It shows that studying highest-weight modules is sufficient for understanding all finite-dimensional representations.

# Examples
**Example** (p. 111): The proof constructs the highest weight by choosing $h \in \mathfrak{h}$ with $\langle h, \alpha \rangle > 0$ for all $\alpha \in R_+$ and maximizing $\langle h, \lambda \rangle$ over weights $\lambda$.
