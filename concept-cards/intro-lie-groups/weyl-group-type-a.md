# Quick Definition
The Weyl group of type $A_n$ is $W = S_{n+1}$, the symmetric group on $n+1$ letters, acting on $E$ by permutations of the coordinates $e_1, \ldots, e_{n+1}$. Simple reflections are transpositions $s_i = (i \;\; i+1)$.

# Core Definition
(Kirillov, p. 123): $W = S_{n+1}$, acting on $E$ by permutations. Simple reflections are $s_i = (i \;\; i+1)$.

# Prerequisites
- **Root system type A** -- the Weyl group is generated by reflections in the roots
- **Weyl group** -- general concept

# Key Properties
1. $|W| = (n+1)!$
2. Generated by simple reflections $s_i = (i \;\; i+1)$ for $i = 1, \ldots, n$
3. $s_i$ acts as the reflection $s_{\alpha_i}$ in the hyperplane perpendicular to $\alpha_i = e_i - e_{i+1}$
4. The Weyl group action permutes the weights of any finite-dimensional representation

# Examples
**Example** (p. 123): For $A_2$: $W = S_3$, the symmetric group of order 6, generated by $s_1 = (1\;\;2)$ and $s_2 = (2\;\;3)$.

# Relationships

## Contrasts With
- **Weyl group type B** -- $W = S_n \ltimes (\mathbb{Z}_2)^n$ (permutations and all sign changes)
- **Weyl group type D** -- permutations and even number of sign changes
