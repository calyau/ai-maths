# Quick Definition

A 2x2 rotation matrix is an orthogonal matrix with determinant 1, of the form [[cos theta, -sin theta], [sin theta, cos theta]], representing counterclockwise rotation of R^2 about the origin through angle theta.

# Core Definition

The orthogonal 2x2 matrices with determinant 1 are the matrices R = [[cos theta, -sin theta], [sin theta, cos theta]] for some angle theta. The matrix R represents counterclockwise rotation of the plane R^2 about the origin and through the angle theta (Artin, Theorem 5.1.17(a), p. 145).

# Prerequisites

- **Orthogonal matrix** — Rotation matrices are orthogonal
- **Special orthogonal group** — Rotation matrices form SO_2

# Key Properties

1. Determinant is 1
2. Columns are orthonormal
3. Eigenvalues are e^{i*theta} and e^{-i*theta} (complex)
4. Composition of rotations: R_alpha * R_beta = R_{alpha+beta}
5. The set of all rotation matrices forms the group SO_2

# Construction / Recognition

## To Construct:
1. Choose an angle theta
2. Form the matrix [[cos theta, -sin theta], [sin theta, cos theta]]

## To Recognize:
1. Check that the matrix is 2x2 orthogonal with determinant 1
2. Verify it has the form [[c, -s], [s, c]] with c^2 + s^2 = 1

# Context & Application

Rotation matrices are the most fundamental orientation-preserving isometries. Every orientation-preserving isometry of the plane can be written as a translation composed with a rotation about the origin. They generate the cyclic rotation groups C_n which appear in the classification of finite subgroups of O_2.

# Examples

**Example 1** (p. 145): The identity matrix is the rotation through angle 0.

**Example 2** (p. 145): R = [[0, -1], [1, 0]] is rotation through angle pi/2.

# Relationships

## Builds Upon
- **Orthogonal matrix** — A rotation matrix is a special orthogonal matrix

## Enables
- **Cyclic rotation group** — Generated by a single rotation matrix
- **Dihedral group** — Contains rotations as a subgroup

## Contrasts With
- **Reflection (linear)** — Reflections have determinant -1; rotations have determinant 1

# Common Errors

- **Error**: Using [[cos theta, sin theta], [-sin theta, cos theta]] for counterclockwise rotation
  **Correction**: Counterclockwise rotation uses [[cos theta, -sin theta], [sin theta, cos theta]]

# Common Confusions

- **Confusion**: Thinking rotation matrices are always real-diagonalizable
  **Clarification**: Rotation matrices (except for angles 0 and pi) have complex eigenvalues and are not diagonalizable over R
