# Quick Definition

An element of an extension field K/F is transcendental over F if it is not a root of any nonzero polynomial with coefficients in F. The field generated by a transcendental element is isomorphic to the field of rational functions.

# Core Definition

An element alpha is transcendental over F if it is not algebraic over F -- if it is not a root of any polynomial with coefficients in F (Artin, p. 454). Equivalently, the substitution homomorphism F[x] -> K sending x to alpha is injective. If alpha is transcendental, F(alpha) is isomorphic to the field F(x) of rational functions. All transcendental extensions F(alpha) are isomorphic (p. 457).

# Prerequisites

- **Field extension** -- Transcendental elements live in extension fields

# Key Properties

1. F(alpha) is isomorphic to F(x), the field of rational functions, if alpha is transcendental
2. All extensions generated by a single transcendental element are isomorphic
3. The extension F(alpha)/F has infinite degree when alpha is transcendental

# Context & Application

Transcendental extensions are relatively simple structurally (all look like rational function fields), but proving an element is transcendental is often very difficult (e.g., pi, e over Q).

# Examples

**Example 1** (p. 454): The variable t in the rational function field F(t) is transcendental over F.
