# Quick Definition

A presentation of a group G is a description G = <x_1, ..., x_n | r_1, ..., r_k> where the x_i are generators and the r_i are a complete set of defining relations. The group is the quotient of the free group on the generators by the normal subgroup generated by the relations.

# Core Definition

The notation <x_1, ..., x_n | r_1, ..., r_k> denotes the quotient F/N where F is the free group on {x_1, ..., x_n} and N is the normal subgroup generated by {r_1, ..., r_k} (Artin, (7.10.5), p. 225). A group G has this as a presentation if the canonical map psi: F/N -> G is an isomorphism, meaning R is a complete set of relations (Corollary 7.10.14).

# Prerequisites

- **Generators and relations** — A presentation specifies both
- **Free group** — The numerator of the quotient

# Key Properties

1. The same group can have many different presentations
2. A presentation is "complete" if every relation is a consequence of R
3. The Todd-Coxeter algorithm can verify completeness
4. A group is finitely presented if it has a presentation with finitely many generators and relations

# Examples

**Example 1** (p. 225): D_n = <x, y | x^n, y^2, xyxy>.

**Example 2** (p. 225): T = <x, y, z | x^3, y^3, z^2, xyz>, verified by Todd-Coxeter (Example 7.11.5).

**Example 3** (p. 219): The non-abelian group of order 21: <x, y | x^7, y^3, y^{-1}xy x^{-2}>.
