# Quick Definition

An element alpha of an extension field K/F is algebraic over F if it is a root of some nonzero polynomial with coefficients in F. The degree of the irreducible polynomial for alpha over F measures how "complex" alpha is relative to F.

# Core Definition

Let K be an extension of a field F, and let alpha be an element of K. The element alpha is algebraic over F if it is a root of a monic polynomial f(x) = x^n + a_{n-1}x^{n-1} + ... + a_0, with a_i in F, and f(alpha) = 0 (Artin, p. 454, equation 15.2.1). An element is transcendental over F if it is not algebraic over F.

# Prerequisites

- **Field extension** -- Algebraic elements live in extension fields

# Key Properties

1. Whether an element is algebraic depends on the base field F (p. 454)
2. If alpha is algebraic, F[alpha] = F(alpha) is a field (Proposition 15.2.6)
3. The degree [F(alpha):F] equals the degree of the irreducible polynomial for alpha (15.3.4)
4. The set of elements algebraic over F in an extension K forms a subfield (15.3.6(d))
5. Every element of a finite extension is algebraic (15.3.6(a))

# Context & Application

The algebraic/transcendental dichotomy is fundamental to field theory. Algebraic elements have a rich theory (minimal polynomials, degree, isomorphism criteria), while transcendental elements generate extensions isomorphic to rational function fields.

# Examples

**Example 1** (p. 454): The complex number 2*pi*i is algebraic over R (root of x + 2*pi*i... actually root of the real polynomial) but transcendental over Q.

**Example 2** (p. 454): Every element alpha of a field K is algebraic over K, since it is a root of x - alpha.

**Example 3** (p. 456): sqrt(2) is algebraic of degree 2 over Q, with irreducible polynomial x^2 - 2.

# Common Confusions

- **Confusion**: Thinking "algebraic" is an absolute property
