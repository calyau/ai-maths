# Quick Definition

The order of a group G, denoted |G|, is the number of elements it contains. A group is finite if its order is finite, and infinite otherwise.

# Core Definition

The order of a group G is the number of elements that it contains, denoted |G| (Artin, p. 52, formula 2.2.1). If the order is finite, G is a finite group; otherwise, G is an infinite group.

# Prerequisites

- **Group** — The group whose order is being counted

# Key Properties

1. |G| can be finite or infinite
2. |S_n| = n! (the symmetric group has order n factorial)
3. Lagrange's theorem: if H is a subgroup, |H| divides |G|

# Context & Application

The order of a group is a basic invariant. Lagrange's theorem constrains the possible orders of subgroups and elements, and the order is used to classify groups.

# Examples

**Example 1** (p. 53): |S_3| = 6 (there are 6 permutations of 3 elements).

**Example 2** (p. 53): There is essentially only one group of order 2.

# Common Errors

- **Error**: Confusing the order of a group with the order of an element
  **Correction**: |G| is the number of elements; the order of an element is the smallest n with a^n = 1

# Common Confusions

- **Confusion**: Thinking groups of the same order are isomorphic
  **Clarification**: There can be multiple non-isomorphic groups of the same order (e.g., two groups of order 4)
