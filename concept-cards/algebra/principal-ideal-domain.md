# Quick Definition

A principal ideal domain (PID) is an integral domain in which every ideal is principal (generated by a single element).

# Core Definition

An integral domain in which every ideal is principal is called a principal ideal domain (Artin, p. 372). Every Euclidean domain is a PID (Proposition 12.2.7), and every PID is a UFD (Proposition 12.2.14).

# Prerequisites

- **Integral domain** -- A PID is a domain
- **Principal ideal** -- Every ideal must be principal

# Key Properties

1. In a PID, every irreducible element is prime (Corollary 12.2.9)
2. GCDs always exist and can be written as linear combinations ra + sb (Proposition 12.2.8)
3. Maximal ideals are generated by irreducible elements (Corollary 12.2.9)
4. Every PID is a UFD, but not conversely (Z[x] is a UFD but not a PID)

# Examples

**Example 1** (p. 372): Z is a PID.

**Example 2** (p. 372): F[x] over a field is a PID.

**Example 3** (p. 374): Z[x] is NOT a PID (the ideal (2, x) is not principal), but it IS a UFD.

# Relationships

## Builds Upon
- **Integral domain** -- A PID is a special domain

## Enables
- **Unique factorization domain** -- Every PID is a UFD

## Contrasts With
- **Unique factorization domain** -- Z[x] is a UFD that is not a PID
