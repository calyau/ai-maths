# Quick Definition

The fundamental unit of a real quadratic field Q[sqrt(d)] is the smallest unit greater than 1 in the first quadrant of the (u,v)-embedding. All positive units are powers of this fundamental unit.

# Core Definition

The group of units in the ring of integers R of a real quadratic number field is infinite (Theorem 13.9.9). The set of units in the first quadrant forms an infinite cyclic subgroup (Exercise 9.5), generated by the fundamental unit (Artin, Section 13.9, pp. 417-422).

# Key Properties

1. The unit group is {+/- epsilon^n | n in Z} for a fundamental unit epsilon
2. N(epsilon) = +/-1
3. The fundamental unit can be found using continued fractions

# Examples

**Example 1** (p. 418): For d = 2, the fundamental unit is 1 + sqrt(2), with N(1+sqrt(2)) = -1.
