# Quick Definition

A cyclic R-module is a module generated by a single element v, isomorphic to R/I for some ideal I. Cyclic modules are the building blocks in the structure theorem for modules over PIDs.

# Core Definition

A cyclic R-module C is a module generated by a single element v. There is a surjective homomorphism R -> C sending r to rv, whose kernel is an ideal I. By the First Isomorphism Theorem, C is isomorphic to R/I (Artin, p. 450). When R = F[t], C is isomorphic to R/(d) for some polynomial d, and the module of relations is generated by d.

# Prerequisites

- **Module** -- Cyclic modules are the simplest modules

# Key Properties

1. Generated by one element
2. Isomorphic to R/I for some ideal I
3. Over Z: cyclic modules are Z/nZ (finite cyclic groups) or Z (infinite cyclic)
4. Over F[t]: cyclic modules are F[t]/(f) for a polynomial f
5. The structure theorem decomposes into direct sums of cyclic modules

# Context & Application

Cyclic modules are the atoms of the structure theorem. Every finitely generated module over a PID decomposes as a direct sum of cyclic modules and a free part.

# Examples

**Example 1** (p. 437): C_5 = Z/5Z is a cyclic Z-module.

**Example 2** (p. 450): F[t]/(t^3 - 3t^2 + 2t) is a cyclic F[t]-module that decomposes further as F[t]/(t) + F[t]/(t-1) + F[t]/(t-2).
