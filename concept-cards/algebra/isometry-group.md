# Quick Definition

The isometry group M_n is the group of all isometries of R^n under composition. There is a surjective homomorphism pi: M_n -> O_n whose kernel is the group of translations.

# Core Definition

The set of all isometries of R^n forms a group M_n, with composition of functions as its law of composition (Artin, Corollary 6.2.9, p. 166). The map pi: M_n -> O_n, defined by dropping the translation part (writing f = t_a * phi, set pi(f) = phi), is a surjective homomorphism whose kernel is the normal subgroup T of translations (Proposition 6.2.10).

# Key Properties

1. Every isometry decomposes uniquely as f = t_a * phi
2. The homomorphism pi: M_n -> O_n has kernel T (translations)
3. T is a normal subgroup of M_n
4. Composition rules: phi * t_a = t_{phi(a)} * phi and t_a * t_b = t_{a+b}

# Construction / Recognition

## To Compose:
1. Write each isometry as t_a * phi
2. Use rules: phi * t_v = t_{phi(v)} * phi; t_a * t_b = t_{a+b}

# Context & Application

M_n is the ambient group for studying symmetry. Every symmetry group is a subgroup of M_n. The short exact sequence T -> M_n -> O_n captures the structure: isometries are built from translations and orthogonal operators.

# Examples

**Example 1** (p. 166): In the plane, M = M_2 is generated by translations, rotations about the origin, and the reflection r about the e_1-axis.

# Common Errors

- **Error**: Composing isometries without tracking translation changes
  **Correction**: Use the rule phi * t_a = t_{phi(a)} * phi carefully

# Common Confusions

- **Confusion**: Thinking M_n = O_n
  **Clarification**: O_n is the quotient M_n / T; M_n also includes translations
