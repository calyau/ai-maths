# Quick Definition

A noncommutative ring satisfies all axioms of a ring except commutativity of multiplication. The set of real n x n matrices is the primary example. Artin uses "ring" exclusively for commutative rings.

# Core Definition

A noncommutative ring is a structure that satisfies all axioms of Definition 11.1.3 except the commutative law for multiplication. The set of all real n x n matrices is one example (Artin, p. 340).

# Prerequisites

- **Ring** -- A noncommutative ring relaxes one axiom

# Key Properties

1. Artin excludes these from his study, using "ring" to mean commutative ring
2. Matrix rings are the primary examples
3. The theory of ideals and quotients generalizes but with left/right distinctions

# Examples

**Example 1** (p. 340): The ring of real n x n matrices is noncommutative for n >= 2.
