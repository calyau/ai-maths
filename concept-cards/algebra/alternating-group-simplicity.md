# Quick Definition

The alternating group A_n is simple for every n >= 5. The proof shows that any nontrivial normal subgroup must contain a 3-cycle, and for n >= 5, all 3-cycles are conjugate in A_n, so the normal subgroup is all of A_n.

# Core Definition

For every n >= 5, the alternating group A_n is a simple group (Artin, Theorem 7.5.4, p. 214). The proof uses two key facts: (a) A_n is generated by 3-cycles (Lemma 7.5.5(a)), and (b) for n >= 5, the 3-cycles form a single conjugacy class in A_n (Lemma 7.5.5(b)). The case analysis shows any nontrivial normal subgroup contains a commutator that is a 3-cycle.

# Prerequisites

- **Simple group** — The theorem asserts simplicity

# Key Properties

1. A_n is simple for all n >= 5
2. A_2 is trivial, A_3 is cyclic of order 3
3. A_4 is NOT simple (has a normal subgroup of order 4)
4. For n >= 5, 3-cycles form one conjugacy class in A_n
5. The proof uses commutators gxg^{-1}x^{-1} to reduce to 3-cycles

# Context & Application

This is one of the most important results in elementary group theory. It shows that the pattern of solvability of polynomial equations by radicals breaks down at degree 5 (Galois theory, Chapter 16). A_5 = I (the icosahedral group) is the smallest non-abelian simple group.

# Examples

**Example 1** (p. 214): In S_5, the class equation is 120 = 1 + 10 + 15 + 20 + 20 + 30 + 24.

**Example 2** (p. 215): Case 1 of the proof: if a normal subgroup N of A_n contains a permutation with a cycle of length >= 5, computing the commutator with a suitable 3-cycle yields a 3-cycle in N.
