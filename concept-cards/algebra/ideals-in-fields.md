# Quick Definition

A field has exactly two ideals: the zero ideal (0) and the unit ideal (1). Conversely, a ring with exactly two ideals is a field.

# Core Definition

(a) The only ideals of a field are the zero ideal and the unit ideal. (b) A ring that has exactly two ideals is a field (Artin, Proposition 11.3.19, p. 349).

# Prerequisites

- **Ideal** -- Understanding ideal structure
- **Field** -- Fields are rings where every nonzero element is invertible

# Key Properties

1. If an ideal of a field contains a nonzero element, it contains 1 (since the element is invertible)
2. Every homomorphism from a field to a nonzero ring is injective (Corollary 11.3.20)

# Examples

**Example 1** (p. 349): R has only (0) and (1) as ideals, confirming it is a field.
