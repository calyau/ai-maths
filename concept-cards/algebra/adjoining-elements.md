# Quick Definition

To adjoin an element alpha satisfying a polynomial relation f(alpha) = 0 to a ring R, form the quotient R' = R[x]/(f). The residue of x becomes alpha. When f is monic of degree n, every element of R[alpha] is uniquely a linear combination of 1, alpha, ..., alpha^{n-1}.

# Core Definition

Given a ring R and a polynomial f(x) in R[x], the ring obtained by adjoining an element alpha satisfying f(alpha) = 0 is R' = R[x]/(f), where (f) is the principal ideal generated by f. The residue alpha = x-bar of x in R' satisfies f(alpha) = 0 (Artin, Section 11.5, pp. 355-357).

When f is monic of degree n, Proposition 11.5.5 states: (a) {1, alpha, ..., alpha^{n-1}} is a basis of R[alpha] over R; (b) addition is vector addition; (c) multiplication is done by polynomial multiplication followed by division by f.

# Prerequisites

- **Quotient ring** -- The construction uses R[x]/(f)
- **Polynomial ring** -- Domain of the construction
- **First Isomorphism Theorem (rings)** -- Identifies R[x]/(f) with the adjunction

# Key Properties

1. When f is monic of degree n, R[alpha] has a basis {1, alpha, ..., alpha^{n-1}}
2. The original ring R embeds in R[alpha] when f is monic
3. The construction generalizes building C from R (adjoin i with i^2 + 1 = 0)
4. One can also adjoin an inverse by using f(x) = ax - 1

# Examples

**Example 1** (p. 357): Z[gamma] where gamma = cube root of 2, has basis {1, gamma, gamma^2} over Z. It is isomorphic to Z[x]/(x^3 - 2).

**Example 2** (p. 357): Adjoining delta with delta^2 - 3 = 0 to F_5 produces a field of 25 elements.

**Example 3** (p. 357): Adjoining delta with delta^2 - 3 = 0 to F_11 does NOT produce a field, since F_11 already contains square roots of 3.

# Relationships

# Common Errors

- **Error**: Assuming R[alpha] is always a field
  **Correction**: R[alpha] is a field only when f is irreducible over R (and R is a field)
