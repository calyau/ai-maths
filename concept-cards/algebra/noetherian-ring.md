# Quick Definition

A ring R is Noetherian if every ideal of R is finitely generated. Equivalently, R satisfies the ascending chain condition on ideals: there is no infinite strictly increasing chain of ideals.

# Core Definition

A ring R is Noetherian if every ideal of R is finitely generated (Artin, Definition 14.6.2). This is equivalent to the ascending chain condition: there is no infinite strictly increasing chain I_1 < I_2 < ... of ideals of R (Corollary 14.6.3).

# Prerequisites

- **Ring** -- Noetherian is a property of rings
- **Ideal** -- Defined in terms of ideals being finitely generated

# Key Properties

1. Principal ideal domains are Noetherian (every ideal is generated by one element)
2. Z, Z[i], and F[x] (F a field) are Noetherian
3. Every submodule of a finitely generated module over a Noetherian ring is finitely generated (Theorem 14.6.5)
4. Every proper ideal is contained in a maximal ideal (Corollary 14.6.4)
5. Quotients of Noetherian rings are Noetherian (Proposition 14.6.8)

# Construction / Recognition

## To Recognize:
1. Check whether every ideal is finitely generated, OR
2. Verify the ascending chain condition on ideals

# Context & Application

The Noetherian property ensures that the module of relations is finitely generated, which is essential for the existence of presentation matrices. Combined with the Hilbert Basis Theorem, it shows that polynomial rings over Noetherian rings are Noetherian.

# Examples

**Example 1** (p. 449): Z is Noetherian (it is a PID).

**Example 2** (p. 449): F[x_1, ..., x_n] is Noetherian by iterated application of the Hilbert Basis Theorem.

# Common Confusions

- **Confusion**: Thinking Noetherian means every module is finitely generated
  **Clarification**: Noetherian means every ideal is finitely generated. As a consequence, submodules of finitely generated modules are finitely generated (14.6.5).
