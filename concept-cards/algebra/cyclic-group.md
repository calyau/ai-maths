# Quick Definition

A cyclic group is a group generated by a single element x: it consists of all powers of x. A cyclic group of order n has the form {1, x, x^2, ..., x^(n-1)} with x^n = 1.

# Core Definition

The cyclic subgroup <x> generated by an element x of a group G is the set of all powers of x: {..., x^(-2), x^(-1), 1, x, x^2, ...}. If x has order n, then <x> = {1, x, x^2, ..., x^(n-1)} is a cyclic group of order n. It is called cyclic because repeated multiplication by x cycles through all n elements (Artin, pp. 57-58).

# Prerequisites

- **Group** — A cyclic group is a special type of group
- **Subgroup** — <x> is a subgroup of the ambient group
- **Order of an element** — The order of x determines the size of <x>

# Key Properties

1. Generated by a single element
2. Always abelian
3. Every subgroup of a cyclic group is cyclic
4. A group of prime order p is cyclic (Corollary 2.8.11)
5. C_rs is isomorphic to C_r x C_s when gcd(r,s) = 1 (Proposition 2.11.3)

# Construction / Recognition

## To Construct:
1. Choose an element x in a group G
2. Compute all powers: 1, x, x^2, ... until they repeat

## To Recognize:
1. The group is generated by a single element
2. Every element is a power of some fixed element

# Context & Application

Cyclic groups are the simplest groups and serve as building blocks. They arise as subgroups generated by any single element, and the structure of any finite abelian group decomposes into products of cyclic groups.

# Examples

**Example 1** (p. 58): If x has order n, then <x> = {1, x, x^2, ..., x^(n-1)}.

**Example 2** (p. 58): In S_3, the element x = (123) generates a cyclic subgroup of order 3.

**Example 3** (p. 58): The matrix [[1,1],[0,1]] has infinite order in GL_2(R); [[1,1],[-1,0]] has order 6.

# Common Errors

- **Error**: Assuming every abelian group is cyclic
  **Correction**: The Klein four group V = C_2 x C_2 is abelian but not cyclic

# Common Confusions

- **Confusion**: Thinking a cyclic group must have a unique generator
  **Clarification**: A cyclic group of order n has phi(n) generators (where phi is Euler's totient)
