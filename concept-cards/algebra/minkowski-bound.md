# Quick Definition

The Minkowski bound mu for a quadratic integer ring bounds the norms of ideal class representatives: every class contains an ideal of norm at most mu. One need only examine primes p < mu to determine the class group.

# Core Definition

For the ring of integers in an imaginary quadratic field Q[sqrt(d)], the Minkowski bound is mu = 2*sqrt(|d|/3) if d = 2 or 3 mod 4, and mu = sqrt(|d|/3) if d = 1 mod 4 (Artin, formula 13.7.7, p. 414). Every ideal class contains an ideal A with N(A) <= mu (Theorem 13.7.10(a)). The class group is generated by classes of prime ideals whose norms are primes p < mu.

# Prerequisites

- **Class group** -- The bound is used to compute the class group
- **Norm of an ideal** -- N(A) <= mu for representatives

# Key Properties

1. Only finitely many primes need to be examined
2. If no primes less than mu split, the class group is trivial
3. The bound depends only on d

# Examples

**Example 1** (p. 413): For d = -163, mu ~ 8.28, so inspect primes 2, 3, 5, 7. All remain prime, so class number = 1.

**Example 2** (p. 413): For d = -5, mu ~ 2.58, so inspect only p = 2. Since 2 ramifies, the class group has order 2.
