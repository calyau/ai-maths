# Quick Definition

A linear operator T on R^2 is a reflection if it has orthogonal eigenvectors with eigenvalues 1 and -1, reflecting the plane about the one-dimensional eigenspace for eigenvalue 1.

# Core Definition

A linear operator T on R^2 is a reflection if it has orthogonal eigenvectors v_1 and v_2 with eigenvalues 1 and -1, respectively. It fixes v_1 and changes the sign of v_2, thus reflecting the plane about the line spanned by v_1 (Artin, p. 145). The orthogonal 2x2 matrices with determinant -1 have the form S = [[c, s], [s, -c]] = R * S_0, where S_0 = [[1, 0], [0, -1]] is the reflection about the e_1-axis (Theorem 5.1.17(b)).

# Prerequisites

- **Orthogonal matrix** — Reflection matrices are orthogonal with determinant -1

# Key Properties

1. Determinant is -1
2. Eigenvalues are 1 and -1 with orthogonal eigenvectors
3. Has order 2 (applying twice gives the identity)
4. The characteristic polynomial is t^2 - 1
5. The matrix S reflects about the line making angle theta/2 with the e_1-axis

# Construction / Recognition

## To Construct:
1. Choose a line through the origin (angle theta/2 with e_1-axis)
2. Form S = [[cos theta, sin theta], [sin theta, -cos theta]]

## To Recognize:
1. Check orthogonality and determinant -1
2. Verify eigenvalues are 1 and -1

# Context & Application

Reflections are the orientation-reversing orthogonal operators on R^2. Together with rotations, they generate the dihedral groups. In the plane isometry group M, reflections about lines (not necessarily through the origin) are one of the four types of isometries.

# Examples

**Example 1** (p. 145): S_0 = [[1, 0], [0, -1]] reflects about the e_1-axis.

**Example 2** (p. 145): The general reflection S = RS_0, where R is a rotation, reflects about a line through the origin at angle theta/2.

# Common Errors

- **Error**: Expecting a reflection matrix to be diagonal
  **Correction**: Only reflection about a coordinate axis gives a diagonal matrix

# Common Confusions

- **Confusion**: Thinking reflections in R^2 and R^3 work the same way
  **Clarification**: In R^3, a reflection in a plane (through the origin) has eigenvalue structure {1, 1, -1}, not {1, -1}
