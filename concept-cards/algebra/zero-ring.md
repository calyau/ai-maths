# Quick Definition

The zero ring is the ring containing just one element, 0, where 1 = 0. It is the unique ring in which the additive and multiplicative identities coincide.

# Core Definition

The zero ring is the ring that contains just one element, 0. A ring R in which 1 = 0 is the zero ring, because for any element a, a = 1a = 0a = 0 (Artin, Proposition 11.1.5, p. 339).

# Prerequisites

- **Ring** -- The zero ring is a special case of a ring

# Key Properties

1. Contains exactly one element: 0
2. 1 = 0 in this ring
3. It is the only ring with this property
4. Every equation a = 0 holds in the zero ring

# Context & Application

The zero ring is a degenerate case that must often be excluded. For example, a field requires 1 != 0, so the zero ring is not a field. It is the only ring with no maximal ideals (Corollary 11.9.3).

# Examples

**Example 1** (p. 339): {0} with 0 + 0 = 0 and 0 * 0 = 0 is the zero ring.
