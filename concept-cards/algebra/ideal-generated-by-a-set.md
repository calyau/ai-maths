# Quick Definition

The ideal generated by elements a_1, ..., a_n in a ring R is the smallest ideal containing those elements: (a_1, ..., a_n) = {r_1 a_1 + ... + r_n a_n | r_i in R}.

# Core Definition

The ideal I generated by a set of elements {a_1, ..., a_n} of a ring R is the smallest ideal that contains those elements. It can be described as the set of all linear combinations r_1 a_1 + ... + r_n a_n with coefficients r_i in the ring (Artin, formulas 11.3.17-11.3.18, p. 348).

# Prerequisites

- **Ideal** -- Understanding what an ideal is

# Key Properties

1. A principal ideal is the special case of one generator
2. The number of generators needed can be arbitrarily large
3. (a_1, ..., a_n) is the smallest ideal containing all the a_i

# Examples

**Example 1** (p. 348): The kernel of Z[x] -> F_p sending f(x) to f(0) mod p is (p, x).

**Example 2** (p. 348): The ideal (x_1, x_2 y, x y^2, y^3) of C[x, y] cannot be generated by fewer than four elements.
