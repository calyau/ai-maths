# Quick Definition

A module V is finitely generated if there exists a finite set of elements that span V -- every element can be written as a linear combination of these generators with coefficients in the ring.

# Core Definition

An ordered set (v_1, ..., v_k) of elements of a module V is said to generate V, or to span V, if every element v is a linear combination v = r_1 v_1 + ... + r_k v_k with coefficients in R (Artin, p. 434, equation 14.2.4). A module V is finitely generated if there exists a finite set of generators.

# Prerequisites

- **Module** -- Finitely generated is a property of modules

# Key Properties

1. A finitely generated module has a surjective homomorphism from some R^m to V
2. Over a Noetherian ring, submodules of finitely generated modules are finitely generated (14.6.5)
3. A finitely generated module over a PID has a presentation matrix
4. Not every finitely generated module is free

# Context & Application

Most modules studied in algebra are finitely generated. The assumption of finite generation is essential for the existence of presentation matrices and for the structure theorem.

# Examples

**Example 1** (p. 434): R^n is finitely generated by the standard basis (e_1, ..., e_n).

**Example 2** (p. 437): Z/4Z is finitely generated by a single element as a Z-module.
