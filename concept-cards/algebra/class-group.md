# Quick Definition

The class group C of the ring of integers R is the abelian group of ideal classes under multiplication: (A)(B) = (AB). It is always finite. Its order is the class number, and R is a UFD iff C is trivial.

# Core Definition

The ideal classes form an abelian group C, the class group of R, with multiplication (A)(B) = (AB). The identity is the class of principal ideals. The inverse of (A) is (A-bar), since A*A-bar = (n) is principal by the Main Lemma (Artin, Proposition 13.7.3, p. 411).

# Key Properties

1. C is abelian (ideal multiplication is commutative)
2. C is finite (Theorem 13.7.10)
3. Generated by classes of prime ideals with norm < mu
4. R is a UFD iff C is trivial (Theorem 13.5.6)
5. The class number quantifies how badly unique factorization fails

# Construction / Recognition

## To Compute:
1. Compute mu (the Minkowski bound)
2. Determine which primes p < mu split, ramify, or remain prime
3. For splitting primes, include one prime factor as a generator
4. Compute norms of small elements to find relations among generators

# Examples

**Example 1** (p. 413): For d = -5, C = C_2 (cyclic of order 2).

**Example 2** (p. 413): For d = -163, C is trivial (R is a UFD).

**Example 3** (p. 414): For d = -26, C = C_6 (cyclic of order 6).

**Example 4** (p. 415): For d = -74, C = C_{10} (cyclic of order 10).
