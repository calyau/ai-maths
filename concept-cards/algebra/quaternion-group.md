# Quick Definition

The quaternion group H is a non-abelian group of order 8 consisting of the matrices {+/-1, +/-i, +/-j, +/-k}, where i, j, k are specific 2x2 complex matrices satisfying i^2 = j^2 = k^2 = -1 and ij = k.

# Core Definition

The quaternion group H = {+/-1, +/-i, +/-j, +/-k} consists of eight 2x2 complex matrices (Artin, p. 59, formula 2.4.5). The defining relations are i^2 = j^2 = k^2 = -1, ij = -ji = k, jk = -kj = i, ki = -ik = j (formula 2.4.6). It is generated by i and j.

# Prerequisites

- **Group** — H is a group of order 8

# Key Properties

1. |H| = 8
2. Non-abelian (ij = -ji)
3. Every subgroup is normal
4. Not isomorphic to the dihedral group D_4 (despite both having order 8)

# Construction / Recognition

## To Construct:
1. Use the matrices: 1 = [[1,0],[0,1]], i = [[i,0],[0,-i]], j = [[0,1],[-1,0]], k = [[0,i],[i,0]]

## To Recognize:
1. A non-abelian group of order 8 where every element has order 1, 2, or 4

# Context & Application

The quaternion group is one of the smallest non-abelian groups and provides important examples in group theory. Its defining relations connect to the quaternion algebra discovered by Hamilton.

# Examples

**Example 1** (p. 59): i^2 = j^2 = k^2 = -1, ij = k, ji = -k.

# Common Errors

- **Error**: Confusing quaternion group H with the quaternion algebra (which has more elements)
  **Correction**: H has exactly 8 elements; the quaternion algebra is 4-dimensional

# Common Confusions

- **Confusion**: Thinking all groups of order 8 are isomorphic
  **Clarification**: There are 5 isomorphism classes of groups of order 8
