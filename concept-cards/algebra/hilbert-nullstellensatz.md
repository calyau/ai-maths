# Quick Definition

Hilbert's Nullstellensatz states that the maximal ideals of C[x_1, ..., x_n] are in bijective correspondence with points of C^n. The point a = (a_1, ..., a_n) corresponds to the kernel M_a of the substitution map sending x_i to a_i, generated by the linear polynomials x_i - a_i.

# Core Definition

The maximal ideals of the polynomial ring C[x_1, ..., x_n] are in bijective correspondence with points of complex n-dimensional space. A point a = (a_1, ..., a_n) of C^n corresponds to the kernel M_a of the substitution map s_a: C[x_1, ..., x_n] -> C that sends x_i to a_i. The kernel M_a is generated by the n linear polynomials x_i - a_i (Artin, Theorem 11.8.6, p. 365).

# Prerequisites

- **Maximal ideal** -- The theorem classifies maximal ideals
- **Polynomial ring in several variables** -- Domain is C[x_1, ..., x_n]

# Key Properties

1. Every maximal ideal has the form M_a = (x_1 - a_1, ..., x_n - a_n)
2. The proof uses a dimension argument: C(x) contains an uncountable independent set
3. The name is German for "zero places theorem"

# Context & Application

The Nullstellensatz is the fundamental bridge between algebra and geometry, connecting ideals in polynomial rings to geometric objects (varieties) in complex space.

# Examples

**Example 1** (p. 365): For n = 1, this reduces to the fact that maximal ideals of C[x] are (x - a).
