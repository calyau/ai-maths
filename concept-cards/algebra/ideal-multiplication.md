# Quick Definition

The product ideal AB consists of all finite sums of products alpha_i * beta_i with alpha_i in A and beta_i in B. If A = (a_1,...,a_m) and B = (b_1,...,b_n), then AB is generated by the mn products a_i b_j.

# Core Definition

Let A and B be ideals in a ring R. The product ideal AB consists of all finite sums of products sum alpha_i beta_i, with alpha_i in A and beta_i in B (Artin, formula 13.4.1, p. 401). Ideal multiplication is commutative, associative, and has the unit ideal R as identity: AB = BA, A(BC) = (AB)C, AR = A (formula 13.4.2).

# Prerequisites

- **Ideal** -- Products are defined for ideals
- **Ring of integers** -- Primary context is quadratic integer rings

# Key Properties

1. If A = (alpha) and B = (beta), then AB = (alpha*beta)
2. If A = (alpha) is principal, then AB = alpha*B
3. AB is generated by pairwise products of generators of A and B
4. Multiplication is commutative and associative

# Examples

**Example 1** (p. 402): In Z[sqrt(-5)], with A = (2, 1+delta), A-bar = (2, 1-delta): A*A-bar = (2).

**Example 2** (p. 402): AB = (1+delta), A*B-bar = (1-delta), B*B-bar = (3).

**Example 3** (p. 402): (6) = A*A-bar * B*B-bar, providing common refinement of 2*3 = (1+delta)(1-delta).
