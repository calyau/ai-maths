# Quick Definition

The word problem for a group G = <S | R> asks: given a word w in the free group F on S, does w represent the identity element of G? Equivalently, is w in the normal subgroup N generated by R? It is solvable for finite groups but not in general.

# Core Definition

Given a group G = F/N presented by generators and relations, the word problem is to decide when an element w of the free group F represents the identity element of G, i.e., when w is in N (Artin, p. 226). If the word problem is solvable, one can determine when two words represent the same group element (since w_1 = w_2 iff w_1^{-1}w_2 = 1).

# Key Properties

1. The word problem is solvable for every finite group
2. There exist finitely presented groups with unsolvable word problems
3. For specific groups (like dihedral groups), the word problem is easy
4. Solving the word problem enables computation in the group

# Context & Application

The word problem is a fundamental algorithmic question in group theory. Artin uses it to motivate the difficulty of working with presentations and to introduce the Todd-Coxeter algorithm as a practical tool.

# Examples
