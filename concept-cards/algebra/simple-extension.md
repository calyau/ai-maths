# Quick Definition

A simple extension is a field extension K/F generated by a single element alpha: K = F(alpha), the smallest subfield containing F and alpha.

# Core Definition

If alpha is an element of an extension field K, then F(alpha) denotes the smallest subfield of K containing F and alpha (Artin, p. 455, equation 15.2.4). If alpha is algebraic over F with irreducible polynomial f of degree n, then F(alpha) = F[alpha] is isomorphic to F[x]/(f), and (1, alpha, ..., alpha^{n-1}) is an F-basis (15.2.7). If alpha is transcendental, F(alpha) is isomorphic to the rational function field F(x).

# Prerequisites

- **Field extension** -- Simple extensions are a type of field extension
- **Algebraic element** -- The behavior differs for algebraic vs. transcendental generators

# Key Properties

1. If alpha is algebraic of degree n, then [F(alpha):F] = n (15.3.4)
2. F[alpha] = F(alpha) when alpha is algebraic (Proposition 15.2.6)
3. F(alpha) = F[x]/(f) where f is the minimal polynomial (15.2.6)
4. By the Primitive Element Theorem, every finite extension in characteristic zero is simple (15.8.1)

# Context & Application

Simple extensions are the building blocks of all finite extensions in characteristic zero. The Primitive Element Theorem reduces the study of general finite extensions to simple extensions.

# Examples

**Example 1** (p. 455): Q(sqrt(2)) is a simple extension of Q with basis (1, sqrt(2)) and degree 2.

**Example 2** (p. 456): Q(omega) where omega = e^{2*pi*i/3} is simple of degree 2 over Q.
