# Quick Definition
If G and G* are dual plane graphs with blocks B_1,...,B_n, then B_1*,...,B_n* are the blocks of G* (Exercise 31). Duality preserves the block structure.

# Core Definition
"Let G, G* be dual plane graphs. Let B_1,...,B_n be the blocks of G. Show that B_1*,...,B_n* are the blocks of G*" (Diestel, p. 108, Exercise 31).

# Prerequisites
- **Plane dual**

# Key Properties
1. Block structure is preserved under duality
2. If G is 2-connected, so is G*; if 3-connected, so is G*
3. 4-connected G does not imply 4-connected G* (Exercise 30(iii))
