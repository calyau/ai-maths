# Quick Definition
A 2-connected plane graph is bipartite if and only if every face is bounded by an even cycle (Exercise 23). This connects the algebraic property of bipartiteness to the geometric structure of face boundaries.

# Core Definition
"Show that a 2-connected plane graph is bipartite if and only if every face is bounded by an even cycle" (Diestel, p. 107, Exercise 23).

# Prerequisites
- **Plane graph**, **Face boundary**

# Key Properties
1. Bipartite iff all face boundaries are even cycles
2. All face boundaries even -> all cycles even -> bipartite
3. Provides a visual test for bipartiteness of plane graphs
