# Quick Definition
A circle in a topological space is a homeomorphic image of the unit circle S^1. In |G|, circles can incorporate both edges and ends, and the set of edges in a circle is called its circuit.

# Core Definition
A *circle* in a topological space is a homeomorphic image of the unit circle S^1 in R^2. In |G|, a closed standard subspace C is a circle if and only if C is connected, every vertex in C has exactly two incident edges in C, and every end in C has vertex-degree 2 in C. The set D of edges in a circle is its *circuit*; the circle is the closure of D and is uniquely determined by it (Diestel, pp. 240-242).

# Key Properties
1. Every circle is a standard subspace
2. A circle is uniquely determined by its circuit (edge set)
3. No circles consist only of ends
4. A circle can contain uncountably many ends (Figure 8.5.1)

# Examples
**Example 1** (p. 240): The two double rays of the 2-way infinite ladder, together with its two ends, form a circle.

**Example 2** (p. 241, Fig. 8.5.1): A circle containing uncountably many ends, built from the binary tree T_2 with added edges.
