# Quick Definition
If an infinite graph contains k disjoint rays for every k, it contains infinitely many disjoint rays (Theorem 8.2.5, Halin 1965).

# Core Definition
**Theorem 8.2.5** (Halin 1965): (i) If G contains k disjoint rays for every k in N, then G contains infinitely many disjoint rays. (ii) Same for edge-disjoint rays (Diestel, p. 216).
