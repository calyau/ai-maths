# Quick Definition
A normal ray of a normal spanning tree T is a ray in T starting at the root that increases in the tree-order. Each end contains exactly one normal ray (Lemma 8.2.3).

# Core Definition
A *normal ray* of a normal spanning tree T is a ray in T that starts at the root and whose vertices increase in the tree-order. By Lemma 8.2.3, if T is a normal spanning tree of G, then every end of G contains exactly one normal ray of T (Diestel, p. 214).
