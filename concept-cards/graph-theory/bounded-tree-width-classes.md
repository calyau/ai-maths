# Quick Definition
For each k, the class of graphs with tw < k is minor-closed and characterized by forbidden minors. Known: tw < 2 = Forb(K^3), tw < 3 = Forb(K^4), tw < 4 = Forb(K^5, octahedron, 5-prism, Wagner).

# Core Definition
For each k, the class {G : tw(G) < k} is minor-closed (by Proposition 12.3.6). Known characterizations: tw < 2: Forb(K^3) = forests; tw < 3: Forb(K^4) (Proposition 12.4.2); tw < 4: Forb(K^5, octahedron, 5-prism, Wagner graph) (Arnborg et al., 1990) (Diestel, pp. 337-338, notes).
