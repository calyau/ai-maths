# Quick Definition
A set is infinite if it is not finite, i.e., there is no natural number n with |A| = n. If A is infinite then |N| <= |A|; the natural numbers have the smallest infinite cardinality.

# Core Definition
A set A is infinite if it is not finite. If A is infinite then |N| <= |A|. Thus, |N| is the smallest infinite cardinality; it is denoted by aleph_0 (Diestel, p. 357).

# Prerequisites
- **Cardinality**, **finite-set** — infinite is defined as not finite

# Key Properties
1. Every infinite set contains a countably infinite subset
2. |N| <= |A| for every infinite set A
3. If A is infinite and its elements are countable sets, then |union of A| = |A|
