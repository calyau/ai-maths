# Quick Definition
An induced cycle is a cycle in G that forms an induced subgraph, meaning it has no chords. The induced cycles generate the entire cycle space (Proposition 1.9.1).

# Core Definition
An induced cycle in G, a cycle in G forming an induced subgraph, is one that has no chords (Diestel, p. 8).

Proposition 1.9.1: The induced cycles in G generate its entire cycle space (Diestel, p. 23).

# Prerequisites
- **Cycle**, **induced-subgraph**, **chord**

# Key Properties
1. An induced cycle has no chords
2. Every cycle with a chord can be decomposed into shorter cycles
3. The induced cycles generate C(G) (Proposition 1.9.1)
