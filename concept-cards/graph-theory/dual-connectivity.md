# Quick Definition
If G is a 2-connected plane graph, its dual G* is 2-connected; if G is 3-connected, G* is 3-connected. However, a 4-connected G may have a dual that is not 4-connected.

# Core Definition
Exercise 30: "(i) If G is 2-connected, then G* is 2-connected. (ii) If G is 3-connected, then G* is 3-connected. (iii) If G is 4-connected, then G* need not be 4-connected" (Diestel, p. 108).

# Prerequisites
- **Plane dual**

# Key Properties
1. 2-connectivity preserved
2. 3-connectivity preserved
3. 4-connectivity NOT preserved
4. Connected plane multigraphs always have connected duals (Exercise 26)
