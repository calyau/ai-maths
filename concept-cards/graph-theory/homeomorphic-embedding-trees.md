# Quick Definition
T <= T' for rooted trees if there is an isomorphism from a subdivision of T to a subtree of T' that preserves tree-order. This relation is used in Kruskal's proof.

# Core Definition
For rooted trees T, T' with roots r, r': T <= T' if there exists an isomorphism phi from some subdivision of T to a subtree T'' of T' that preserves the tree-order (Diestel, p. 328).
