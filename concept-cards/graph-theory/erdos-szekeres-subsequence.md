# Quick Definition
Using Ramsey's theorem: for any k, l, every sequence of n = kl + 1 distinct integers contains an increasing subsequence of length k+1 or a decreasing subsequence of length l+1.

# Core Definition
**Exercise 6**: Use Ramsey's theorem to show that for any k, l in N there is an n in N such that every sequence of n distinct integers contains an increasing subsequence of length k+1 or a decreasing subsequence of length l+1. The Erdos-Szekeres theorem shows n = kl + 1 suffices. (Diestel, p. 271)

# Prerequisites
- **Ramsey's theorem (finite)** — Applied to pairs of sequence elements

# Key Properties
1. A classical application of Ramsey theory to combinatorics
2. The 2-colouring is defined by relative order of pairs
3. n = kl + 1 is tight
