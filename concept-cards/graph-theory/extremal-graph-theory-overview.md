# Quick Definition
Extremal graph theory studies how global parameters (edge density, chromatic number) influence local substructures. It divides into dense problems (subgraph forcing) and sparse problems (minor forcing).

# Core Definition
Chapter 7 studies how global parameters of a graph can influence its local substructures:
1. **Subgraphs** (7.1): Turan's theorem, Erdos-Stone theorem — dense theory
2. **Minors** (7.2): Average degree forcing minors — sparse theory
3. **Hadwiger's conjecture** (7.3): chi(G) >= r => K^r minor
4. **Regularity lemma** (7.4): Szemeredi's powerful partition tool
5. **Applying the regularity lemma** (7.5): Embedding lemma, Erdos-Stone proof (Diestel, pp. 163-194)

# Prerequisites
- **Graph**, **Edge**, **Vertex** — Basic graph theory

# Key Properties
1. Dense vs sparse dichotomy: quadratic edges for subgraphs, linear for minors
2. Turan's theorem is THE foundational result
3. The regularity lemma is the key modern tool
4. Hadwiger's conjecture is "the most challenging open problem"
