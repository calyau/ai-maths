# Quick Definition
Richardson's theorem states that every directed graph without odd directed cycles has a kernel. This guarantees the kernels needed in Galvin's theorem proof.

# Core Definition
"Richardson's theorem: every directed graph without odd directed cycles has a kernel" (Diestel, p. 136, Exercise 31).

# Prerequisites
- **Kernel**

# Key Properties
1. Guarantees kernel existence when no odd directed cycles are present
2. Used indirectly in the proof of Galvin's theorem (via stable marriages)
