# Quick Definition
For constant p, any first-order sentence about graphs is either almost surely true or almost surely false. All such statements follow from P_{i,j} for suitable i, j.

# Core Definition
For constant p in (0,1), any statement about graphs expressed by quantifying over vertices only (first-order sentence) is either almost surely true or almost surely false. Such statements are consequences of P_{i,j} for suitable i, j (Diestel, p. 313).
