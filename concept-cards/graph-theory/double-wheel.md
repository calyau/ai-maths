# Quick Definition
A double wheel is a graph of the form C^n * K-bar^2 (n >= 4): a cycle with two non-adjacent hub vertices, each joined to all cycle vertices.

# Core Definition
A graph of the form C^n * K-bar^2 (n >= 4) is called a **double wheel**. It is one of the four types of unavoidable minors in large 4-connected graphs (Theorem 9.4.4). (Diestel, p. 269)

# Prerequisites
- **Graph** — Basic graph concepts

# Key Properties
1. Two hub vertices, each adjacent to all vertices of a cycle
2. The hubs are NOT adjacent to each other (K-bar^2)
3. One of four unavoidable types in 4-connected graphs
