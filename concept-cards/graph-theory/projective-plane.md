# Quick Definition
The projective plane is the surface obtained from the sphere by adding one crosscap. It has Euler genus 1 (Euler characteristic 1) and is non-orientable.

# Core Definition
The projective plane is S^2 with one crosscap. By Lemma B.3, epsilon = 0 + 1 = 1, so chi = 1.

# Prerequisites
- **Surface**, **Crosscap**, **Sphere**

# Key Properties
1. Euler genus epsilon = 1
2. Non-orientable (the simplest non-orientable surface)
3. Contains one-sided circles
4. K_6 can be embedded on the projective plane

# Relationships
## Builds Upon
- **Sphere** plus one **Crosscap**

## Related
- **Torus** -- Orientable surface with epsilon = 2
- **Klein bottle** -- Sphere plus 2 crosscaps
