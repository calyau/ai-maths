# Quick Definition
The empty graph is the graph with no vertices and no edges, written as the empty set. A graph of order 0 or 1 is called trivial.

# Core Definition
For the empty graph (the empty set, the empty set) we simply write the empty set. A graph of order 0 or 1 is called trivial (Diestel, p. 2).

# Prerequisites
- **Graph** — the empty graph is a specific graph

# Key Properties
1. The empty graph has no vertices and no edges
2. It has no components (since components must be non-empty)
3. Trivial graphs (order 0 or 1) are often excluded in theorems to avoid degenerate cases

# Context & Application
Diestel notes that trivial graphs sometimes form silly counterexamples and are often treated with "generous disregard" to avoid cluttering the text with non-triviality conditions.
