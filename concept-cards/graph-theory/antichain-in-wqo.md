# Quick Definition
In a quasi-ordered set, an antichain is a set of pairwise incomparable elements. A quasi-ordering is WQO iff it has no infinite antichain and no infinite strictly decreasing sequence.

# Core Definition
An *antichain* in a quasi-ordered set (X, <=) is a set of pairwise incomparable elements. By Proposition 12.1.1, a quasi-ordering is a WQO iff X contains neither an infinite antichain nor an infinite strictly decreasing sequence (Diestel, p. 326).
