# Quick Definition
Lemma 12.5.4: for every surface S there exists r such that no graph minimally not embeddable in S contains the hexagonal grid H^r as a topological minor. Key step in proving Corollary 12.5.3.

# Core Definition
**Lemma 12.5.4**: For every surface S there exists an integer r such that no graph minimal with the property of not being embeddable in S contains H^r as a topological minor. The proof combines two embeddings: sigma' in S' and sigma in S (Diestel, pp. 353-357).
