# Quick Definition
A set U of vertices in a connected graph G is dispersed if every ray in G can be separated from U by a finite set of vertices. Jung's theorem can be rephrased: G has a normal spanning tree iff V(G) is a countable union of dispersed sets (Exercise 30).
