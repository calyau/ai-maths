# Quick Definition
For constant p, almost every G(n,p) has chi(G) > (log(1/q))/(2+epsilon) * n/log(n). This is sharp up to the epsilon factor.

# Core Definition
**Proposition 11.3.4**: For constant p and epsilon > 0, almost every G in G(n,p) has chi(G) > (log(1/q))/(2+epsilon) * n/log(n). Sharp by a result of Bollobás (1988): replacing epsilon by -epsilon gives an upper bound (Diestel, p. 314).
