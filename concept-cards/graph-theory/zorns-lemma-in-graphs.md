# Quick Definition
Zorn's lemma guarantees a maximal element in any partially ordered set whose chains have upper bounds. In graph theory, it produces maximal trees, matchings, and other structures.

# Core Definition
Given a connected graph G, the set of all trees in G ordered by subgraph inclusion has the property that every chain has an upper bound (the union). By Zorn's lemma, a maximal tree exists, which must be spanning (Proposition 8.1.1, Diestel, p. 208).
