# Quick Definition
The four colour problem was posed by Francis Guthrie in 1852, presented by Cayley in 1878, incorrectly proved by Kempe in 1879, corrected to a five colour theorem by Heawood in 1890, and finally proved by Appel and Haken in 1977 using computer assistance.

# Core Definition
The four colour conjecture -- that every planar map can be coloured with 4 colours -- was open for 124 years. Appel and Haken's proof (1977) showed that every plane triangulation contains one of ~1500 "unavoidable configurations," each provably "reducible" (4-colourable by induction). A shorter proof by Robertson, Sanders, Seymour, and Thomas appeared in 1997 (Diestel, Notes, pp. 137-138).

# Key Properties
1. First posed 1852 (Guthrie)
2. First published discussion 1878 (Cayley)
3. Kempe's flawed proof 1879 -- introduced Kempe chains, which remain important
4. Heawood corrected to 5-colour theorem 1890
5. Appel-Haken proof 1977 -- first major proof using computer assistance
6. Robertson-Sanders-Seymour-Thomas shorter proof 1997

# Context & Application
The four colour problem is one of the most famous problems in mathematics. Its resolution marked a milestone in computer-assisted proofs. Kempe's incorrect proof introduced the Kempe chain technique, which remains fundamental in graph colouring theory.
