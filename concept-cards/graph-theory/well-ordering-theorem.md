# Quick Definition
The well-ordering theorem states that every set can be well-ordered. It is equivalent to the axiom of choice.

# Core Definition
Well-ordering theorem: Every set can be well-ordered. This is one of the many statements equivalent to the axiom of choice (which Diestel assumes throughout) (Diestel, p. 358).

# Prerequisites
- **Well-ordered set** — the theorem asserts well-orderings exist
- **Axiom of choice** — the theorem is equivalent to it

# Key Properties
1. Equivalent to the axiom of choice
2. Equivalent to Zorn's lemma
3. Guarantees that any set, including R, can be well-ordered
