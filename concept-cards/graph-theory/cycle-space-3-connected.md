# Quick Definition
The cycle space of a 3-connected graph is generated by its non-separating induced cycles.

# Core Definition
**Theorem 3.2.3 (Tutte 1963).** The cycle space of a 3-connected graph is generated by its non-separating induced cycles (Diestel, p. 69).

# Prerequisites
- **3-connected graph** — the theorem applies to 3-connected graphs
- **Cycle space** — the vector space of edge sets that are symmetric differences of cycles

# Key Properties
1. Non-separating induced cycles generate the entire cycle space
2. A separating cycle would disconnect the graph; these are not needed as generators
3. Fundamental triangles (triangles containing a fixed edge e) are non-separating
4. Proof uses induction on |G| via contractible edges (Lemma 3.2.1)
5. Plays an important role in Chapter 4.5

# Construction / Recognition
## Proof Strategy
1. Use induction on |G| with base case K4
2. For the induction step, choose a contractible edge e = xy
3. Contract e to get a smaller 3-connected graph G' = G/e
4. Lift basic cycles from G' back to G using the "similarity" relation
5. Fundamental triangles fill any gaps

# Context & Application
This theorem provides an algebraic description of the cycle structure of 3-connected graphs. It is used in planarity testing and in the study of cycle double covers.

# Examples
**Example** (p. 70): In K4, every cycle is a triangle or the symmetric difference of triangles. All triangles are induced and non-separating.
