# Quick Definition
Two rays are equivalent if for every finite set S of vertices, both have a tail in the same component of G - S. This defines the end equivalence relation.

# Core Definition
Two rays in G are equivalent if for every finite set S of V(G), both have a tail in the same component of G - S. Equivalently, two rays are equivalent iff they can be linked by infinitely many disjoint paths (Diestel, p. 213).
