# Quick Definition
A set A is countable if |A| <= |N|, and countably infinite if |A| = |N|. A bijection N -> A is an enumeration of A.

# Core Definition
A set A is countable if |A| <= |N|, and countably infinite if |A| = |N|. A bijection N -> A is an enumeration of A (Diestel, p. 357).

# Prerequisites
- **Cardinality**, **infinite-set** — countability is a cardinality condition

# Key Properties
1. Every finite set is countable
2. N is countably infinite
3. A countable union of countable sets is countable
4. If A is infinite and its elements are countable sets, then |union of A| = |A|
