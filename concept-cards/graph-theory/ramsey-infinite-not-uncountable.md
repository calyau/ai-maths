# Quick Definition
Ramsey's theorem does NOT extend to uncountable sets: there exists a graph on R (the reals) with neither a complete nor an edgeless induced subgraph of cardinality 2^{aleph_0}.

# Core Definition
**Exercise 4**: Construct a graph on R that has neither a complete nor an edgeless induced subgraph on |R| = 2^{aleph_0} vertices. This shows that the infinite Ramsey theorem (Theorem 9.1.2) does not extend to uncountable sets. (Diestel, p. 271)

# Prerequisites
- **Ramsey's theorem (infinite)** — The theorem that is NOT generalized

# Key Properties
1. The infinite Ramsey theorem holds for COUNTABLE sets
2. Fails for uncountable sets (2^{aleph_0} vertices)
3. The counterexample uses a well-ordering of R
