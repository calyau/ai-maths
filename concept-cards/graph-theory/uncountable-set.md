# Quick Definition
A set is uncountable if it is not countable. If |A| = |R| then A has continuum many elements. The smallest uncountable cardinality is aleph_1.

# Core Definition
There is also a smallest uncountable cardinality, denoted by aleph_1. If |A| = |R| then A is uncountable, and we say that A has continuum many elements. Whether |R| equals aleph_1 or is greater depends on the axioms of set theory assumed (Diestel, p. 357).

# Prerequisites
- **Cardinality**, **countable-set** — uncountable means not countable

# Key Properties
1. aleph_1 is the smallest uncountable cardinality
2. The reals have continuum many elements
3. There are continuum many infinite 0-1 sequences
4. The continuum hypothesis (whether |R| = aleph_1) is independent of ZFC
