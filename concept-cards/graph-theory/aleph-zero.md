# Quick Definition
Aleph-zero (aleph_0) is the cardinality of the natural numbers N, the smallest infinite cardinality.

# Core Definition
|N| is the smallest infinite cardinality; it is denoted by aleph_0 (Diestel, p. 357).

# Prerequisites
- **Cardinality**, **countable-set** — aleph_0 is the cardinality of N

# Key Properties
1. aleph_0 = |N|
2. It is the smallest infinite cardinality
3. Every infinite set has cardinality >= aleph_0
