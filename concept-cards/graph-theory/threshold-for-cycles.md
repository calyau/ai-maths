# Quick Definition
t(n) = n^{-1} is a threshold function for the property of containing a k-cycle, for any fixed k >= 3. Cycles of all constant lengths appear at about the same time.

# Core Definition
**Corollary 11.4.4**: If k >= 3, then t(n) = n^{-1} is a threshold function for containing a k-cycle. Interestingly, the threshold is independent of k: all cycle lengths appear at about the same time (Diestel, p. 322).
