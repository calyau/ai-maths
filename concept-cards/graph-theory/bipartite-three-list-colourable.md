# Quick Definition
Every bipartite planar graph is 3-list-colourable (Exercise 32). This follows from Richardson's theorem and Lemma 5.4.3.

# Core Definition
"Show that every bipartite planar graph is 3-list-colourable. (Hint. Apply [Richardson's theorem] and Lemma 5.4.3.)" (Diestel, p. 136, Exercise 32).

# Prerequisites
- **List colouring**, **Planar graph**

# Key Properties
1. ch(G) <= 3 for bipartite planar graphs
2. This is tight: some bipartite planar graphs require 3 colours even in the list setting
3. Uses orientation-based list colouring techniques
