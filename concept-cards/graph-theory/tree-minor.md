# Quick Definition
The minor relation restricted to trees. Kruskal's theorem shows finite trees are WQO by the topological minor relation, which is stronger.

# Core Definition
The *topological minor relation* on trees: T is a topological minor of T' if T' contains a subdivision of T. Kruskal's theorem (12.2.1) shows finite trees are WQO by this relation. The proof uses rooted trees and an embedding relation that preserves tree-order (Diestel, p. 328).
