# Quick Definition
Dickson's lemma: the natural numbers N are WQO under <=, and hence N^k (with componentwise <=) is WQO. This is a basic example motivating the WQO concept.

# Core Definition
N is well-quasi-ordered under <=. By iterated application of Higman's lemma (or directly), N^k with componentwise ordering is WQO. This is Dickson's lemma, a basic result in WQO theory (Diestel, notes to Ch. 12).
