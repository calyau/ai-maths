# Quick Definition
t(n) = n^{-k/(k-1)} is a threshold for containing a copy of a tree of order k. Unlike cycles, the threshold depends on tree order but not tree shape.

# Core Definition
**Corollary 11.4.5**: If T is a tree of order k >= 2, then t(n) = n^{-k/(k-1)} is a threshold for containing a copy of T. The threshold depends on order k but not on shape (Diestel, p. 322).
