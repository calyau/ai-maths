# Quick Definition
For every k, there exists n such that for every partition of {1,...,n} into k sets, at least one subset contains x, y, z with x + y = z. This follows from Ramsey's theorem.

# Core Definition
**Exercise 8** (Schur's theorem): For every k in N there is an n in N such that, for every partition of {1,...,n} into k sets, at least one of the subsets contains numbers x, y, z such that x + y = z. This is a Ramsey-type result about additive structure in partitions of integers. (Diestel, p. 271)

# Prerequisites
- **Ramsey theorem (general)** — Schur's theorem follows from Ramsey theory

# Key Properties
1. An early application of Ramsey theory to number theory
2. Proved using Ramsey's theorem on a suitable graph/hypergraph
3. Precursor to Rado's theorem and additive combinatorics
