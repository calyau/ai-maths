# Quick Definition

A finite reflection group is a finite group generated by reflections in $\mathbb{R}^n$. Every finite reflection group is a Coxeter group, and conversely every finite Coxeter group is a finite reflection group.

# Core Definition

"A *finite reflection group* is a finite group generated by reflections. For such a group $G$, it is possible to choose a set $S$ of generating reflections for which $(G, S)$ is a Coxeter system (Humphreys 1990, 1.9). Thus, the finite reflection groups are all Coxeter groups (in fact, they are precisely the finite Coxeter groups, ibid., 6.4)." (Milne, 2.14, p. 39)

# Prerequisites

- **Reflection** — the group is generated by reflections
- **Coxeter system** — every finite reflection group admits a Coxeter system structure

# Key Properties

1. Finite reflection groups = finite Coxeter groups (Humphreys 1990, 6.4)
2. For any such group, there exists a choice of generating reflections $S$ making $(G, S)$ a Coxeter system
3. Classified by Coxeter diagrams: types $A_n, B_n, D_n, E_6, E_7, E_8, F_4, H_3, H_4, I_2(m)$

# Context & Application

Finite reflection groups appear throughout mathematics: as Weyl groups in Lie theory, as symmetry groups of regular polytopes, and as fundamental objects in combinatorial geometry.

# Examples

**Example 1** (p. 39, Example 2.15a): $S_n$ acts on $\mathbb{R}^n$ by permuting coordinates. Transpositions are reflections, so $S_n$ is a finite reflection group (type $A_{n-1}$).

**Example 2** (p. 39, Example 2.15b): $D_n$ is generated by two reflections through adjacent diagonals of a regular $n$-gon.
