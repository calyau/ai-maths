# Quick Definition

A cyclic group is a group generated by a single element. Up to isomorphism, there is exactly one cyclic group C_n of order n for each n <= infinity.

# Core Definition

A group is said to be **cyclic** if it is generated by a single element, i.e., if G = <r> for some r in G. If r has finite order n, then G = {e, r, r^2, ..., r^{n-1}} ≈ C_n, and G can be thought of as the group of rotational symmetries about the centre of a regular n-sided polygon. If r has infinite order, then G = {..., r^{-i}, ..., r^{-1}, e, r, ..., r^i, ...} ≈ C_infinity. Up to isomorphism, there is exactly one cyclic group of order n for each n <= infinity (1.16, p. 12).

Milne states: "In future, we shall loosely use C_n to denote any cyclic group of order n (not necessarily Z/nZ or Z)."

# Prerequisites

- **Group** — a cyclic group is a group
- **Subgroup generated by a set** — cyclic means generated by one element
- **Order of an element** — determines whether C_n or C_infinity

# Key Properties

1. Generated by a single element r
2. Every cyclic group is abelian
3. Up to isomorphism, exactly one cyclic group of each order n (including n = infinity)
4. C_n ≈ Z/nZ for finite n; C_infinity ≈ Z
5. Every subgroup of a cyclic group is cyclic
6. C_n can be thought of as the rotational symmetries of a regular n-gon

# Construction / Recognition

## To Construct:
1. Take an element r of order n (or infinite order)
2. G = <r> = {r^k | k in Z}

## To Recognize:
1. A group is cyclic if there exists an element whose powers give every element of the group
2. Equivalently, a group of order n is cyclic if it contains an element of order n

# Context & Application

Cyclic groups are the simplest groups and are completely classified. Every group of prime order p is cyclic (C_p), as stated in the classification of groups of small order (1.28, referenced p. 14). Cyclic groups serve as building blocks: adding a "tick" to each vertex of a regular polygon reduces its symmetry group from D_n to C_n (p. 13).

# Examples

**Example 1** (p. 8, 1.3): C_infinity = (Z, +) and C_m = (Z/mZ, +) for m >= 1.

**Example 2** (p. 12, 1.16): C_n = {e, r, r^2, ..., r^{n-1}} with r^i <-> i mod n.

**Example 3** (p. 14, table): C_4 is one of two groups of order 4; C_6 is one of two groups of order 6.

# Common Errors

- **Error**: Assuming every abelian group is cyclic
  **Correction**: C_2 x C_2 is abelian but not cyclic (no element has order 4)

# Common Confusions

- **Confusion**: Thinking the generator of a cyclic group is unique
  **Clarification**: C_n may have multiple generators; for example, C_6 is generated by either r or r^5
