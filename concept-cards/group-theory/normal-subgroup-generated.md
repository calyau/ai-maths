# Quick Definition

The normal subgroup generated by a subset $X$ of $G$ is the smallest normal subgroup containing $X$. It equals $\langle \bigcup_{g \in G} gXg^{-1} \rangle$ -- the subgroup generated by all conjugates of elements of $X$.

# Core Definition

"The normal subgroup generated by a subset $X$ of $G$ is $\langle \bigcup_{g \in G} gXg^{-1} \rangle$." (Milne, Proposition 1.40, p. 20)

This is obtained by: (1) forming the smallest normal set containing $X$, namely $\bigcup_{g \in G} gXg^{-1}$ (Lemma 1.39), then (2) taking the subgroup it generates, which is automatically normal (Lemma 1.38).

# Prerequisites

- **Normal subgroup** — the result is a normal subgroup
- **Conjugation** — construction involves conjugates

# Key Properties

1. It is the intersection of all normal subgroups containing $X$
2. $\bigcup_{g \in G} gXg^{-1}$ is the smallest normal set containing $X$ (Lemma 1.39)
3. If $X$ is already closed under conjugation, then $\langle X \rangle$ is normal (Lemma 1.38)

# Context & Application

This construction is essential for defining group presentations. In $\langle X \mid R \rangle$, the quotient is by the normal subgroup generated by $R$ in the free group $FX$.

# Examples

**Example 1** (p. 35): In a group presentation $\langle X \mid R \rangle$, the group is $FX / N$ where $N$ is the normal subgroup of $FX$ generated by (the images of) $R$.
