# Quick Definition

The dihedral group D_n is the group of symmetries of a regular n-sided polygon. It has order 2n and is generated by a rotation r of order n and a reflection s of order 2 satisfying srs = r^{-1}.

# Core Definition

For n >= 3, D_n is the group of symmetries of a regular polygon with n sides. Let r be the rotation through 2pi/n about the centre (so i -> i+1 mod n), and s be the reflection through the line from vertex 1 to the centre (so i -> n+2-i mod n). The defining relations are:

r^n = e, s^2 = e, srs = r^{-1} (equivalently, sr = r^{n-1}s).

These imply D_n = {e, r, ..., r^{n-1}, s, rs, ..., r^{n-1}s}, so |D_n| = 2n.

Special cases: D_1 = C_2 and D_2 = C_2 x C_2 (the Klein four-group). D_3 = S_3 is the full permutation group on {1, 2, 3} and the smallest noncommutative group (1.17, pp. 12-13).

Alternatively, D_n = <s, t> where s and t are reflections with s^2 = e, t^2 = e, (ts)^n = e (p. 13).

More formally, D_n can be defined as the subgroup of S_n generated by r: i -> i+1 (mod n) and s: i -> n+2-i (mod n) (footnote 4, p. 12).

# Prerequisites

- **Group** — D_n is a group
- **Cyclic group** — D_n contains the cyclic subgroup <r> of rotations
- **Symmetric group** — D_n is a subgroup of S_n
- **Order of an element** — r has order n, s has order 2

# Key Properties

1. |D_n| = 2n
2. Generated by r (rotation, order n) and s (reflection, order 2)
3. Key relation: srs = r^{-1}, equivalently sr = r^{n-1}s
4. Contains n rotations {e, r, ..., r^{n-1}} and n reflections {s, rs, ..., r^{n-1}s}
5. The rotations form a cyclic normal subgroup of index 2
6. D_n is noncommutative for n >= 3
7. D_3 = S_3 (the symmetric group on 3 letters)
8. Also generated by two reflections s, t with s^2 = t^2 = (ts)^n = e

# Construction / Recognition

## To Construct:
1. Start with a regular n-gon, number the vertices 1 through n
2. Let r be the rotation by 2pi/n
3. Let s be a reflection through a vertex
4. D_n = <r, s> with r^n = s^2 = e and srs = r^{-1}

## To Recognize:
1. A group of order 2n
2. Generated by elements r, s with r^n = s^2 = e and srs = r^{-1}
3. Contains an index-2 cyclic subgroup

# Context & Application

Dihedral groups are among the most fundamental examples of non-abelian groups. They arise as symmetry groups of regular polygons and appear throughout the classification of groups of small order. By adding a tick at each vertex, the symmetry group reduces from D_n to C_n; by adding a line from centre to vertex 1, it reduces to <s>. Physicists say the symmetry has been "broken" (p. 13).

# Examples

**Example 1** (p. 13, 1.17): D_3 is the group of symmetries of an equilateral triangle. It has 6 elements and equals S_3.

**Example 2** (p. 13, 1.17): D_4 is the group of symmetries of a square, with 8 elements. It is one of the five groups of order 8.

**Example 3** (p. 13): D_2 = C_2 x C_2 = {1, r, s, rs}, the Klein four-group.

**Example 4** (p. 14, table): D_5, D_7, D_9, D_{10}, D_{11} appear in the classification of groups of orders 10, 14, 18, 20, 22.

# Common Errors

- **Error**: Confusing D_n notation — some authors write D_{2n} for the same group
  **Correction**: Milne uses D_n for the symmetry group of an n-gon (order 2n), noting the notational ambiguity (footnote 3, p. 12)

# Common Confusions

- **Confusion**: Thinking D_n has order n
  **Clarification**: |D_n| = 2n (n rotations + n reflections)

- **Confusion**: Thinking all dihedral groups are non-abelian
  **Clarification**: D_1 = C_2 and D_2 = C_2 x C_2 are abelian
