# Quick Definition

The image of a homomorphism $\alpha: G \to G'$ is the set $\alpha(G) = \{\alpha(g) \mid g \in G\}$. It is always a subgroup of $G'$.

# Core Definition

"Recall that the image of a map $\alpha: S \to T$ is $\alpha(S) = \{\alpha(s) \mid s \in S\}$." The image $I$ of $\alpha$ is a subgroup of $G'$, and $\alpha$ factors as $G \twoheadrightarrow G/N \xrightarrow{\sim} I \hookrightarrow G'$. (Milne, Theorem 1.45, p. 21)

# Prerequisites

- **Homomorphism** — The image is defined for a homomorphism

# Key Properties

1. The image $\mathrm{Im}(\alpha)$ is a subgroup of $G'$ (Theorem 1.45)
2. $\alpha$ is surjective $\iff$ $\mathrm{Im}(\alpha) = G'$
3. The factorization theorem gives $G/\mathrm{Ker}(\alpha) \cong \mathrm{Im}(\alpha)$
4. If $b = \alpha(a)$ and $b' = \alpha(a')$, then $bb' = \alpha(aa')$ and $b^{-1} = \alpha(a^{-1})$

# Construction / Recognition

## To Compute the Image:
1. Evaluate $\alpha$ on generators of $G$
2. The image is the subgroup generated by those values in $G'$

# Context & Application

The image, together with the kernel, completely describes a homomorphism up to the factorization theorem. The first isomorphism theorem states $G/\mathrm{Ker}(\alpha) \cong \mathrm{Im}(\alpha)$.

# Examples

**Example 1** (p. 21): In the factorization $G \to G/N \xrightarrow{\sim} I \hookrightarrow G'$, the image $I$ is isomorphic to $G/N$.

**Example 2** (p. 20): The image of $\det: \mathrm{GL}_n(F) \to F^{\times}$ is all of $F^{\times}$ (since any scalar can appear as a determinant).

# Common Errors

- **Error**: Assuming the image is normal in $G'$
  **Correction**: The image is a subgroup but not necessarily normal

# Common Confusions

- **Confusion**: Confusing the image with the codomain
  **Clarification**: The image is a subgroup of the codomain; a homomorphism is surjective only when the image equals the full codomain
