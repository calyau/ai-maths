# Quick Definition

The quaternion group Q is the non-abelian group of order 8 consisting of {+/-1, +/-i, +/-j, +/-k} under quaternion multiplication. It can also be realized as a subgroup of GL_2(C).

# Core Definition

The quaternion group Q is defined as a subgroup of GL_2(C) generated by matrices a = ((0, sqrt(-1)), (sqrt(-1), 0)) and b = ((0, 1), (-1, 0)). The relations are a^4 = e, a^2 = b^2, bab^{-1} = a^3 (so ba = a^3 b). The group is:

Q = {e, a, a^2, a^3, b, ab, a^2 b, a^3 b}

Equivalently, Q can be described as the subset {+/-1, +/-i, +/-j, +/-k} of the quaternion algebra H = R1 + Ri + Rj + Rk with multiplication determined by i^2 = -1 = j^2, ij = k = -ji. The map i -> a, j -> b gives an isomorphism <i, j> -> <a, b> (1.18, pp. 13-14).

# Prerequisites

- **Group** — Q is a group
- **General linear group** — Q is realized as a subgroup of GL_2(C)
- **Order of an element** — a has order 4, a^2 has order 2

# Key Properties

1. |Q| = 8
2. Q is non-abelian
3. Q has a unique element of order 2 (namely a^2 = b^2 = -1)
4. Every proper subgroup of Q is cyclic
5. Key relations: a^4 = e, a^2 = b^2, bab^{-1} = a^3
6. Q = {+/-1, +/-i, +/-j, +/-k} in quaternion notation
7. Q is one of the two non-abelian groups of order 8 (the other is D_4)

# Construction / Recognition

## To Construct:
1. In GL_2(C): take the matrices a and b as defined above, then Q = <a, b>
2. In H: take Q = {+/-1, +/-i, +/-j, +/-k} under quaternion multiplication

## To Recognize:
1. A non-abelian group of order 8
2. With a unique element of order 2
3. (D_4, the other non-abelian group of order 8, has more than one element of order 2)

# Context & Application

The quaternion group is one of the five groups of order 8 and one of only two non-abelian ones. It is structurally different from D_4 despite having the same order. Its connection to the quaternion algebra H makes it important in algebra and geometry.

# Examples

**Example 1** (p. 13, 1.18): Q as a subgroup of GL_2(C) generated by the matrices a and b.

**Example 2** (p. 13, 1.18): Q = {+/-1, +/-i, +/-j, +/-k} with i^2 = j^2 = -1, ij = k = -ji.

**Example 3** (p. 14, table): Q is one of the five groups of order 8 (along with C_8, C_2 x C_4, C_2 x C_2 x C_2, and D_4).

## Contrasts With
- **abelian-group** — Q is non-abelian

# Common Errors

- **Error**: Confusing Q with D_4 since both are non-abelian of order 8
  **Correction**: Q has a unique element of order 2 (the element -1); D_4 has multiple elements of order 2

# Common Confusions

- **Confusion**: Thinking the quaternion group is related to the quaternion number system only abstractly
  **Clarification**: Q is literally a subgroup of the unit quaternions in H
