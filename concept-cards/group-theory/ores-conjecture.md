# Quick Definition

Not every element of the commutator subgroup is a single commutator (smallest counterexample at order 96). However, Ore proved that every element of A_n (n >= 5) is a commutator and conjectured the same for all finite simple groups. This conjecture was proved in 2008.

# Core Definition

**Aside 6.11.** Not every element of the commutator subgroup of a group is itself a commutator, but the smallest groups where this occurs have order 96. This was shown by a computer search through the libraries of small groups.

In 1951 Ore proved that every element of A_n is a commutator when n >= 5 and conjectured that the same is true of all finite simple groups. This was proved in 2008.

(Milne, Aside 6.11, p. 92)

# Key Properties

1. The commutator subgroup is *generated* by commutators, but not every element is a commutator
2. The smallest counterexample has order 96
3. For finite simple groups, every element of the group IS a commutator (Ore's conjecture, now theorem)

# Context & Application

This subtlety is important when working with commutator subgroups: one cannot assume that elements of G' have a clean "commutator form." However, for simple groups (the building blocks of all finite groups), this distinction vanishes.
