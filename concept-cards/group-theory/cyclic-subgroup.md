# Quick Definition

The cyclic subgroup generated by an element a in a group G is <a> = {a^n | n in Z}, the set of all powers of a. Its order equals the order of a.

# Core Definition

For an element a of a group G, the subgroup <a> generated by the singleton set {a} consists of all integer powers of a. The **order of an element** a of a group is the order of the subgroup <a> it generates (p. 12).

If a has finite order m, then <a> = {e, a, a^2, ..., a^{m-1}} has m elements. If a has infinite order, then <a> = {..., a^{-2}, a^{-1}, e, a, a^2, ...} is infinite.

# Prerequisites

- **Subgroup generated by a set** — <a> is the special case X = {a}
- **Order of an element** — determines the size of <a>

# Key Properties

1. <a> = {a^n | n in Z}
2. |<a>| = order of a
3. <a> is always abelian (cyclic groups are commutative)
4. If a has order m, then <a> ≈ C_m
5. If a has infinite order, then <a> ≈ C_infinity ≈ Z

# Construction / Recognition

## To Construct:
1. Take an element a in G
2. Compute all powers: e = a^0, a, a^2, a^3, ...
3. Include all negative powers: a^{-1}, a^{-2}, ...
4. The resulting set is <a>

# Context & Application

Cyclic subgroups are the simplest subgroups of any group. Every element generates a cyclic subgroup, and the structure of these subgroups (determined by element orders) provides fundamental information about the group.

# Examples

**Example 1** (p. 12, 1.16): If r has order n, then <r> = {e, r, r^2, ..., r^{n-1}} ≈ C_n.

**Example 2** (p. 13, 1.17): In D_n, <r> = {e, r, r^2, ..., r^{n-1}} is a cyclic subgroup of order n, and <s> = {e, s} is a cyclic subgroup of order 2.
