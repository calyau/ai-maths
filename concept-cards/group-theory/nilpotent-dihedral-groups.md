# Quick Definition

The dihedral group D_{2^n} is nilpotent of class n. The dihedral group D_n is nilpotent if and only if n is a power of 2. This follows from the characterization of finite nilpotent groups as direct products of their Sylow subgroups.

# Core Definition

**Example 6.12(c).** The dihedral group D_{2^n} is nilpotent of class n. This can be proved by induction, using that Z(D_{2^n}) has order 2 and D_{2^n}/Z(D_{2^n}) is isomorphic to D_{2^{n-1}}.

If n is not a power of 2, then D_n is not nilpotent (use Theorem 6.18: D_n would need to be a direct product of its Sylow subgroups, but it is not).

(Milne, Example 6.12c, p. 93)

# Key Properties

1. Z(D_{2^n}) has order 2 (generated by the rotation by pi)
2. D_{2^n}/Z(D_{2^n}) = D_{2^{n-1}}, giving the inductive step
3. D_4 = D_{2^2} has class 2 (metabelian), D_8 = D_{2^3} has class 3, etc.
4. D_6 (order 12) is not nilpotent: its Sylow 3-subgroup is not normal
