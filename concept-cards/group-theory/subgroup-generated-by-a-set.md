# Quick Definition

The subgroup generated by a subset X of a group G, denoted <X>, is the smallest subgroup of G containing X. It consists of all finite products of elements of X and their inverses.

# Core Definition

For any subset X of a group G, there is a smallest subgroup of G containing X. It is the intersection of all subgroups of G containing X, and it consists of all finite products of elements of X and their inverses (repetitions allowed). This subgroup is denoted **<X>** and called the **subgroup generated by** X (Proposition 1.15, p. 12).

Special cases: <emptyset> = {e}. If every element of X has finite order (e.g., if G is finite), then the set of all finite products of elements of X (without needing explicit inverses) already equals <X>.

# Prerequisites

- **Subgroup** — <X> is a subgroup
- **Intersection of subgroups** — <X> is defined as an intersection
- **Inverse element** — the construction uses inverses of elements of X

# Key Properties

1. <X> is the smallest subgroup of G containing X
2. <X> equals the intersection of all subgroups containing X
3. <X> consists of all finite products of elements of X and their inverses
4. <emptyset> = {e}
5. If all elements of X have finite order, inverses are not needed in the products

# Construction / Recognition

## To Construct:
1. Start with the elements of X and their inverses
2. Form all finite products (including the empty product e)
3. The resulting set is <X>

## To Recognize:
1. <X> is the unique smallest subgroup containing X
2. Every element of <X> can be written as a product of elements from X and X^{-1}

# Context & Application

The subgroup generated by a set is fundamental to describing group structure. It allows one to express a group in terms of generators: saying G = <X> means every element of G can be written as a finite product of elements from X and their inverses.

# Examples

**Example 1** (p. 12): <emptyset> = {e}, the trivial subgroup.

**Example 2** (p. 12, 1.16): <r> in a group where r has order n gives the cyclic group {e, r, r^2, ..., r^{n-1}}.

**Example 3** (p. 13, 1.17): D_n = <s, t> where s and t are reflections with s^2 = e, t^2 = e, (ts)^n = e.

# Common Errors

- **Error**: Forgetting to include inverses of elements of X when forming products
  **Correction**: Unless all elements of X have finite order, you must include both x and x^{-1} for each x in X

# Common Confusions

- **Confusion**: Thinking <X> is just the set X itself
  **Clarification**: <X> contains all finite products of elements of X and their inverses, which is generally much larger than X
