# Quick Definition

The Burnside problem asks: must a finitely generated group of finite exponent be finite? The answer is no (Adjan-Novikov, 1968). The restricted Burnside problem asks if there are only finitely many finite quotients, which was proved affirmatively by Zelmanov (1989).

# Core Definition

"Does there exist [an infinite finitely generated group of finite exponent]? (Burnside problem). In 1968, Adjan and Novikov showed the answer is yes: there do exist infinite finitely generated groups of finite exponent." (Milne, p. 37)

The Burnside group $B(n, e)$ is the quotient of the free group on $n$ generators by the subgroup generated by all $e$th powers. The restricted Burnside problem asks whether $B(n, e)$ has only finitely many finite quotients, proved by Zelmanov. (p. 38)

# Prerequisites

- **Finitely presented group** — the Burnside group is defined by a presentation
- **Exponent of a group** — the problem concerns groups of finite exponent

# Key Properties

1. Burnside problem (answered negatively): infinite finitely generated groups of finite exponent exist
2. The Burnside group $B(n, e)$ is infinite for large enough $e$ and $n \ge 2$
3. Restricted Burnside problem (answered positively by Zelmanov): $B(n, e)$ has only finitely many finite quotients
4. The classification of finite simple groups reduced the restricted problem to prime power exponents

# Context & Application

The Burnside problem is one of the most famous problems in group theory, illustrating the gap between finite generation + finite exponent and finiteness. It connects to deep results in combinatorics and Lie theory.

# Examples

**Example 1** (p. 37): There exist infinite groups generated by finitely many elements of finite order, even with a global exponent bound.
