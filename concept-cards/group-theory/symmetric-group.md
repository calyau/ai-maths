# Quick Definition

The symmetric group S_n is the group of all bijections (permutations) of the set {1, 2, ..., n} under composition. It has order n! and is noncommutative for n >= 3.

# Core Definition

Let S be a set and let Sym(S) be the set of bijections S -> S. The product of two elements is their composite: alpha * beta = alpha composed with beta, meaning (alpha * beta)(s) = alpha(beta(s)) for all s in S. Associativity holds, the identity map is the neutral element, and inverses exist because elements are bijections. Therefore Sym(S) is a group, called the **group of symmetries** of S. The **permutation group on n letters** S_n is defined to be Sym({1, ..., n}); it has order n! (Example 1.4, p. 8).

# Key Properties

1. |S_n| = n!
2. The group operation is composition of functions
3. S_n is noncommutative for n >= 3
4. S_n is generated by transpositions (1.19, p. 14)
5. S_3 is the smallest noncommutative group and equals D_3 (1.17, p. 13)
6. Every group embeds into some symmetric group (Cayley's theorem, 1.22)

# Construction / Recognition

## To Construct:
1. Take the set {1, 2, ..., n}
2. List all bijections from this set to itself
3. Define multiplication as composition of bijections

## To Recognize:
1. A group of order n! whose elements are permutations of n objects
2. The full group of all permutations of an n-element set

# Context & Application

The symmetric group is central to group theory because every group can be embedded in a symmetric group (Cayley's theorem). Permutation groups serve as concrete representations of abstract groups. S_n also appears prominently in combinatorics, representation theory, and the theory of equations.

# Examples

**Example 1** (p. 8, 1.4): S_3 = Sym({1, 2, 3}) has order 3! = 6. It equals D_3 (the symmetries of an equilateral triangle).

**Example 2** (p. 8, 1.4): Associativity of S_n is verified by the calculation ((alpha composed with beta) composed with gamma)(s) = alpha(beta(gamma(s))) = (alpha composed with (beta composed with gamma))(s).

# Common Errors

- **Error**: Composing permutations in the wrong order (left-to-right vs. right-to-left)
  **Correction**: Milne defines alpha * beta = alpha composed with beta, so beta is applied first: (alpha * beta)(s) = alpha(beta(s))

# Common Confusions

- **Confusion**: Confusing S_n (all permutations) with a general permutation group (a subgroup of S_n)
  **Clarification**: S_n is the *full* symmetric group; a permutation group is any subgroup of some S_n
