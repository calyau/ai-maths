# Quick Definition

The order of an element a in a group G is the smallest positive integer m such that a^m = e, or infinity if no such integer exists. It equals the order of the cyclic subgroup generated by a.

# Core Definition

The set {n in Z | a^n = e} is an ideal in Z, hence equals mZ for some integer m >= 0. When m = 0, the element a has **infinite order** (a^n != e for all n != 0). When m != 0, it is the smallest positive integer such that a^m = e, and a is said to have **finite order** m. In this case, a^{-1} = a^{m-1}, and a^n = e if and only if m divides n (p. 8).

The order of an element a equals the order of the subgroup <a> it generates (p. 12).

# Prerequisites

- **Group** — element order is defined within a group
- **Neutral element** — the order is defined by the equation a^m = e
- **Powers of an element** — the definition uses integer powers of a

# Key Properties

1. The order of a is the smallest positive integer m with a^m = e, or infinity
2. a^n = e if and only if the order of a divides n
3. a^{-1} = a^{m-1} when a has finite order m
4. The order of a equals |<a>|, the order of the cyclic subgroup generated by a
5. The identity element e has order 1
6. In a finite group, every element has finite order

# Construction / Recognition

## To Determine:
1. Compute a, a^2, a^3, ... until a^m = e
2. The first such m is the order of a
3. If no such m exists, the order is infinite

# Context & Application

Element order is fundamental to understanding group structure. It connects to the cyclic subgroup generated by the element and plays a key role in results like Lagrange's theorem (which constrains possible element orders in finite groups).

# Examples

**Example 1** (p. 12, 1.16): In C_n, the generator r has order n.

**Example 2** (p. 13, 1.17): In D_n, the rotation r has order n and the reflection s has order 2.

**Example 3** (p. 13, 1.18): In the quaternion group Q, the element a has order 4 and a^2 has order 2.

# Common Errors

- **Error**: Computing a^n = e and concluding the order is n without checking smaller values
  **Correction**: The order is the *smallest* positive integer m with a^m = e

# Common Confusions

- **Confusion**: Confusing order of an element with order of the group
  **Clarification**: Element order is a property of a single element; group order is the cardinality of the entire group
