# Quick Definition

A transposition is a permutation that interchanges two elements and leaves all others unchanged. The symmetric group S_n is generated by its transpositions.

# Core Definition

A **transposition** is a permutation that interchanges two elements and leaves all other elements unchanged. The symmetric group S_n is generated by transpositions (1.19, p. 14; a more precise statement is given in 4.26).

# Prerequisites

- **Symmetric group** — transpositions are elements of S_n

# Key Properties

1. A transposition swaps exactly two elements and fixes the rest
2. Every transposition has order 2
3. S_n is generated by its transpositions
4. A transposition of elements i and j is often written (i j)

# Construction / Recognition

## To Construct:
1. Choose two distinct elements i, j from {1, ..., n}
2. Define the permutation that sends i -> j, j -> i, and fixes all other elements

## To Recognize:
1. A permutation that moves exactly two elements

# Context & Application

Transpositions are the simplest non-trivial permutations. The fact that they generate S_n means any permutation can be decomposed into transpositions. This decomposition is foundational to defining the sign of a permutation and the alternating group (treated later in the text).

# Examples

**Example 1** (p. 14, 1.19): In S_n, the transposition (i j) swaps i and j while fixing all other elements.

# Common Confusions

- **Confusion**: Thinking the decomposition of a permutation into transpositions is unique
  **Clarification**: The decomposition is not unique, but the parity (even or odd number of transpositions) is well-defined
