# Quick Definition

The commutator of two elements $a, b$ in a group is $[a, b] = aba^{-1}b^{-1}$. Two elements commute if and only if their commutator is $e$.

# Core Definition

"Two elements $h_1$, $h_2$ of a group commute if and only if their commutator $[h_1, h_2] \stackrel{\mathrm{def}}{=} (h_1 h_2)(h_2 h_1)^{-1}$ is $e$." (Milne, p. 24)

Equivalently, $[h_1, h_2] = h_1 h_2 h_1^{-1} h_2^{-1}$.

In Chapter 2, the commutator is also written as $[a,b] = aba^{-1}b^{-1}$ (Example 2.7c, p. 35).

# Prerequisites

- **Group** — commutators are defined in any group

# Key Properties

1. $[a, b] = e \iff ab = ba$
2. $[a, b]^{-1} = [b, a]$
3. The commutator subgroup $[G, G]$ is the subgroup generated by all commutators
4. Used in the proof that normality of two factors forces commutativity (Proposition 1.51)

# Context & Application

Commutators quantify noncommutativity. The commutator subgroup $[G, G]$ is the smallest normal subgroup such that $G/[G,G]$ is abelian. Free abelian groups are defined by the relations $[a_i, a_j] = 1$ (Example 2.7c).

# Examples

**Example 1** (p. 24): If $H_1, H_2 \trianglelefteq G$ with $H_1 \cap H_2 = \{e\}$, then $[h_1, h_2] \in H_1 \cap H_2 = \{e\}$, so elements from different factors commute.

**Example 2** (p. 35): The free abelian group on $a_1, \ldots, a_n$ has relations $[a_i, a_j] = 1$ for $i \neq j$.
