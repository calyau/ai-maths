# Quick Definition
Two links are ambient isotopic (equivalent) if there exists an orientation-preserving homeomorphism of $\mathbb{R}^3$ mapping one to the other. For diagrams, this is generated by planar isotopy and all three Reidemeister moves.

# Core Definition
"Two links $L_1$ and $L_2$ are equivalent if there exists an orientation-preserving homeomorphism $h: \mathbb{R}^3 \to \mathbb{R}^3$ such that $h(L_1) = L_2$" (Bollobás, p. 364). By Reidemeister's theorem: two links are ambient isotopic iff their diagrams can be transformed into each other by planar isotopy and all three Reidemeister moves.

# Prerequisites
- **Link** — Ambient isotopy is an equivalence relation on links

# Key Properties
1. Generated by all three Reidemeister moves (plus planar isotopy)
2. Stronger than regular isotopy (which omits Type I moves)
3. The Jones polynomial is an ambient isotopy invariant
4. Preserves number of components
