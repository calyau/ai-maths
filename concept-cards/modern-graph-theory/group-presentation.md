# Quick Definition

A group presentation $A = \langle a, b, \ldots \mid R_\mu, R_\nu, \ldots \rangle$ defines a group as a quotient of the free group on generators $a, b, \ldots$ by the normal subgroup generated by relators $R_\mu, R_\nu, \ldots$.

# Core Definition

Let $F = \langle a, b, c, \ldots \rangle$ be the free group on generators $a, b, c, \ldots$ and let $K$ be the normal subgroup of $F$ generated by words $R_\mu, R_\nu, \ldots$. Then $A = F/K = \langle a, b, c, \ldots \mid R_\mu, R_\nu, \ldots \rangle$ is the group with generators $a, b, \ldots$ and relators $R_\mu, R_\nu, \ldots$. Equivalently, presentations use defining relations: $\langle a, b \mid a^2 = b^3 \rangle$ denotes $\langle a, b \mid a^2 b^{-3} \rangle$ (Bollobas, p. 260).

# Prerequisites

- **Free group** -- The starting point for defining presentations

# Key Properties

1. Two words are equivalent in $A$ iff one can be obtained from the other by inserting/deleting $xx^{-1}$, $x^{-1}x$, and relators $R_\mu^{\pm 1}$
2. A group is finitely presented if it has finitely many generators and relations
3. Dehn's fundamental problems (word, conjugacy, isomorphism) concern group presentations
4. All three of Dehn's problems are undecidable in general

# Construction / Recognition

## Example
$A = \langle a, b \mid a^3b, b^3, a^4 \rangle$: Since $a^3b = 1$, we get $b = a^{-3}$; from $a^4 = 1$ and $b^3 = 1$: $a = aa^3b = a^4b = b$, so $1 = a^3b = a^4 = a = b$. Thus $A$ is trivial.

# Context & Application

Group presentations arise naturally in knot theory (fundamental groups of knot complements), topology, and geometry. The Cayley and Schreier diagram techniques provide practical tools for working with presented groups.

# Examples

**Example 1** (p. 260): $\langle a, b \mid a^3 = b^2 = (ab)^2 = 1\rangle$ is $S_3$.

**Example 2** (p. 261): The group of the trefoil knot is $\langle a, b, c \mid cb = ba = ac\rangle$.

# Common Errors

- **Error**: Assuming a finite presentation means a finite group
  **Correction**: Many finitely presented groups are infinite (e.g., the trefoil knot group)

# Common Confusions

- **Confusion**: Confusing relators with relations
  **Clarification**: A relator is a word $R$ set equal to 1; a relation is an equation $W_1 = W_2$ (equivalent to relator $W_1 W_2^{-1}$)
