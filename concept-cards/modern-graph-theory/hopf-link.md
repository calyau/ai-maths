# Quick Definition
The Hopf link is the simplest non-trivial link: two interlocking circles that cannot be separated without cutting. It has an alternating diagram and is shown in Fig. X.3.

# Core Definition
The Hopf link is a two-component link shown in Fig. X.3 (p. 363). It is alternating and has linking number $\pm 1$. The Jones polynomial can be computed to confirm the two circles are indeed linked (Exercise 37).

# Prerequisites
- **Link** — The Hopf link is a specific two-component link

# Key Properties
1. Two components (circles)
2. Cannot be split (components are linked)
3. Has an alternating diagram
4. Linking number $= \pm 1$ (depending on orientation)
