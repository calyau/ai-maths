# Quick Definition
The three Reidemeister moves are local transformations of link diagrams that, together with planar isotopy, generate ambient isotopy: Type I adds/removes a curl, Type II adds/removes two consecutive crossings, Type III slides a strand past a crossing.

# Core Definition
"Reidemeister proved in the 1920s that two links are ambient isotopic iff their diagrams can be transformed into each other by planar isotopy and the three Reidemeister moves" (Bollobás, p. 365). Type I: add/remove a curl. Type II: add/remove two consecutive under- or over-crossings. Type III (triangle move): change position of two crossings. Regular isotopy uses only Types II and III.

# Prerequisites
- **Link diagram** — Reidemeister moves transform link diagrams

# Key Properties
1. Type I: changes the number of crossings by 1
2. Type II: changes the number of crossings by 2
3. Type III: preserves the number of crossings
4. Together with planar isotopy, they generate ambient isotopy
5. Types II and III alone generate regular isotopy

# Context & Application
Reidemeister moves reduce topological equivalence to a combinatorial question about diagram transformations. To prove a quantity is a knot invariant, one need only show invariance under all three types.
