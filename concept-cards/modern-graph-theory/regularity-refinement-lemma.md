# Quick Definition
If a partition is not $\varepsilon$-uniform, it can be refined to a partition with square mean increased by at least $\varepsilon^5/2$, at the cost of a moderate increase in the number of classes and exceptional set.

# Core Definition
**Lemma 28**: Let $G$ have an equitable partition $\mathcal{P} = (C_i)_{i=0}^k$ with $|C_i| = c \ge 2^{3k+1}$. If $\mathcal{P}$ is not $\varepsilon$-uniform ($0 < \varepsilon < 1/2$, $2^{-k} \le \varepsilon^5/8$), then there is an equitable partition $\mathcal{P}'$ with $\ell = k(4^k - 2^{k-1})$ classes, $|C_0'| \le |C_0| + n/2^k$, and $q(\mathcal{P}') \ge q(\mathcal{P}) + \varepsilon^5/2$ (Bollobás, pp. 141-145).

# Prerequisites
- **Square mean** — The potential function being increased
- **ε-uniform pair** — Non-uniform pairs drive the refinement
- **Density approximation lemma** — Controls errors

# Key Properties
1. The square mean increases by at least $\varepsilon^5/2$ per refinement
2. The exceptional set grows by at most $n/2^k$
3. The number of classes grows multiplicatively: $k \to k(4^k - 2^{k-1})$
4. At most $\lfloor\varepsilon^{-5}\rfloor$ refinements needed (since $q < 1/2$)
5. The proof uses atoms of algebras to avoid cutting across witness sets

# Context & Application
"The cornerstone of the proof of the regularity lemma" (p. 141). This iterative refinement drives convergence to an $\varepsilon$-uniform partition.

# Examples
**Example** (pp. 141-145): Non-uniform pairs $(C_i, C_j)$ have witness sets $C_{ij}, C_{ji}$. The refinement subdivides each $C_i$ into $H = 4^k - 2^{k-1}$ equal parts respecting atoms of the algebra generated by all witness sets.
