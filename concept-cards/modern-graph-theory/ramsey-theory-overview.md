# Quick Definition
Ramsey theory is the mathematical study of the principle that complete disorder is impossible: any sufficiently large structure, partitioned into finitely many classes, must contain a large well-organized substructure in one class.

# Core Definition
Ramsey theory expresses the deep mathematical principle, vastly extending the pigeon-hole principle, that "no matter how we partition the objects of a 'large' structure into a 'few' classes, one of these classes contains a 'large' subsystem." While the pigeon-hole principle guarantees 'many' objects in one class without conditions on their relationships, Ramsey theory demands structured subsets: not just many red edges, but all edges of an infinite set being red (Bollobás, p. 187).

# Key Properties
1. Extends the pigeon-hole principle to structured settings
2. Originated with Ramsey's 1930 paper, but van der Waerden (1927) and Schur (1916) proved earlier results
3. Encompasses graph colourings, integer colourings, and higher structures
4. Results span finite and infinite settings
5. A great variety of techniques from many branches of mathematics are used

# Context & Application
Ramsey theory results are important not only in graph theory and combinatorics but in set theory, logic, analysis, algebra, and geometry. The chapter covers: fundamental Ramsey theorems, canonical extensions, graphical Ramsey numbers, integer Ramsey theory (van der Waerden, Hales-Jewett, Rado), and infinite subsequence theory.

# Examples
**Example** (p. 187): In a party of 6, there are always 3 mutual acquaintances or 3 mutual strangers (R(3,3) = 6).

**Example** (p. 187): If edges of K_ℕ are 2-coloured, some infinite set has all edges the same colour.

**Example** (p. 187): Given k and p, if W is large enough and [W] is k-coloured, some class contains a p-term AP (van der Waerden).
