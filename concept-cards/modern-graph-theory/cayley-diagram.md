# Quick Definition

The Cayley diagram of a group $A$ generated by $a, b, \ldots$ is a directed multigraph with colored edges: there is an edge from $x$ to $y$ colored $g$ iff $xg = y$.

# Core Definition

Let $A$ be a group generated by $a, b, \ldots$. The graph of $A$, also called its Cayley diagram, with respect to these generators is a directed multigraph whose edges are coloured with the generators: there is an edge from $x$ to $y$ coloured with a generator $g$ iff $xg = y$ (Bollobas, p. 259). Two elements expressed as products of generators are equal iff the corresponding walks starting at 1 end at the same vertex.

# Prerequisites

- **Group presentation** -- A Cayley diagram represents a group given by generators and relations

# Key Properties

1. Regular coloring: for each vertex $x$ and generator $g$, exactly one edge of color $g$ starts at $x$ and one ends at $x$
2. At most one edge goes from $x$ to another vertex $y$ (but edges of different colors may connect the same pair)
3. Two words in the generators represent the same group element iff corresponding walks from 1 end at the same vertex
4. Solves the word problem when fully constructed

# Construction / Recognition

## To Construct a Cayley Diagram
1. Start with vertex 1 (the identity)
2. For each generator $g$, create an edge from 1 to a (possibly new) vertex labeled $g$
3. Continue: from each vertex $x$, for each generator $g$, create an edge to $xg$
4. Apply relations: identify vertices when forced by defining relations
5. If the process terminates, you have the Cayley diagram

## Example Construction
For $A = \langle a, b \mid a^3 = b^2 = (ab)^2 = 1\rangle$:
1. Start with 1; triangle 123 for $a^3 = 1$
2. Add $b$-edges from 1, 2, 3 to vertices 4, 5, 6
3. Use $a^3 = 1$ to create triangle 678
4. Apply $(ab)^2 = 1$ to identify vertices: 8 = 4, 5 = 7
5. Result: the Cayley diagram of $S_3$

# Context & Application

Cayley diagrams visualize group structure and are essential tools for solving problems about groups given by presentations. They are connected to tessellations: the group $(p, q, r) = \langle a, b, c \mid a^p = b^q = c^r = abc = 1\rangle$ has Cayley diagram that tessellates the sphere (if $1/p + 1/q + 1/r > 1$), Euclidean plane (equality), or hyperbolic plane (inequality).

# Examples

**Example 1** (p. 259, Fig. VIII.1): Cayley diagrams of $C_4$, the Klein four-group, and $S_3$.

**Example 2** (p. 261): $A = \langle a, b \mid a^3 = b^2 = (ab)^2 = 1\rangle$ has Cayley diagram isomorphic to that of $S_3$.

**Example 3** (p. 261, Theorem 1): For group $(p,q,r)$, if $1/p + 1/q + 1/r > 1$ the group is finite with order $2s$ where $1/s = 1/p + 1/q + 1/r - 1$.

# Common Errors

- **Error**: Drawing undirected edges when the generator has order $> 2$
  **Correction**: Edges are directed unless the generator satisfies $g^2 = 1$, in which case they may be drawn undirected

# Common Confusions

- **Confusion**: Thinking the Cayley diagram depends only on the group, not the generators
  **Clarification**: Different generating sets produce different Cayley diagrams for the same group
