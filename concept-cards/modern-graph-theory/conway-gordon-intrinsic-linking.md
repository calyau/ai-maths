# Quick Definition
Conway and Gordon proved that every embedding of $K_6$ into $\mathbb{R}^3$ is intrinsically linked: there exist two triangles forming a non-trivial link.

# Core Definition
Exercise 45 (p. 381): "Prove the theorem of Conway and Gordon that every embedding of $K_6$ into $\mathbb{R}^3$ is intrinsically linked: there are two triangles forming a non-trivial link." The proof uses linking numbers and parity arguments.

# Prerequisites
- **Link** — The concept of linked triangles
- **Knot** — Context for spatial graph theory

# Key Properties
1. $K_6$ is intrinsically linked (every spatial embedding contains a non-trivial link)
2. The Petersen graph is also intrinsically linked (Exercise 47)
3. Uses linking number parity argument

# Context & Application
"Knot theory has come full circle" (p. 378): from Thomson's hopes to apply it to space, to applications in DNA and molecular synthesis.
