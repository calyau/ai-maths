# Domain Taxonomy for Introduction to Lie Groups and Lie Algebras

## Categories

- **lie-groups**: Lie groups, subgroups, cosets, classical groups, group actions, homogeneous spaces
- **lie-algebras**: Lie algebra definitions, subalgebras, ideals, center, commutator, exponential map
- **representations**: Representations of groups and algebras, irreducibility, operations on representations, characters
- **structure-theory**: Solvable/nilpotent algebras, semisimplicity, radical, Killing form, Cartan criterion
- **root-systems**: Roots, weights, Cartan subalgebra, root decomposition, Weyl group, Dynkin diagrams
- **classification**: Classification of simple Lie algebras, classical types (A/B/C/D), exceptional types
- **enveloping-algebras**: Universal enveloping algebra, PBW theorem, Casimir elements
- **applications**: Spherical harmonics, hydrogen atom, physical applications

## Tiers

- **foundational**: Basic definitions (Lie group, Lie algebra, representation, root, weight) — concepts with no prerequisites within this source
- **intermediate**: Requires foundational concepts (Cartan subalgebra, Killing form, weight decomposition, Weyl group, Haar measure)
- **advanced**: Requires intermediate concepts (PBW theorem, BGG resolution, Weyl character formula, Levi decomposition, Serre relations)

## Notation Conventions

### Lie Groups
- Matrix groups: GL(n,K), SL(n,K), O(n,K), SO(n,K), U(n), SU(n), Sp(2n,K)
- General Lie group: G, H (subgroup)
- Identity element: e or 1

### Lie Algebras
- Fraktur letters: g, h, b, n
- Specific algebras: gl(n,K), sl(n,K), so(n,K), su(n), sp(2n,K)
- Bracket: [x,y]
- Adjoint maps: Ad (group), ad (algebra)

### Representations
- Representation space: V, W
- Weight: lambda (λ)
- Weight space: V[λ]
- Highest weight: λ (or n for sl(2,C))
- Irreducible representation: V_n (sl(2,C)), L_λ (general)

### Root Systems
- Roots: α, β, γ
- Simple roots: α_1, ..., α_n or Π = {α_1, ..., α_n}
- Positive roots: R_+ (or Δ_+)
- Coroot: α^∨ = 2α/(α,α)
- Fundamental weights: ω_1, ..., ω_n
- Weyl group: W
- Simple reflections: s_i = s_{α_i}
- Cartan matrix: A = (a_{ij}), a_{ij} = <α_i, α_j^∨>

### Lattices
- Root lattice: Q
- Weight lattice: P
- Dominant integral weights: P_+

### Inner Products
- Killing form: K(x,y) or (x,y)
- Invariant bilinear form: B(x,y) or B^V(x,y)
