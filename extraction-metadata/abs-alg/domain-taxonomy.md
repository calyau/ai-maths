# Domain Taxonomy for "Abstract Algebra" (Dummit & Foote, 3rd ed.)

## Categories

- **foundations**: Set theory, functions, equivalence relations, integers, modular arithmetic (Ch 0)
- **group-theory**: Groups, subgroups, homomorphisms, group actions, Sylow theorems, products (Ch 1-6)
- **ring-theory**: Rings, ideals, quotient rings, ring homomorphisms, factorization domains (Ch 7-9)
- **module-theory**: Modules, module homomorphisms, free modules, tensor products, exact sequences (Ch 10, 12)
- **linear-algebra**: Vector spaces, linear transformations, dual spaces, determinants, canonical forms (Ch 11-12)
- **field-theory**: Field extensions, algebraic extensions, splitting fields, separability, cyclotomic extensions (Ch 13)
- **galois-theory**: Galois groups, Fundamental Theorem, solvability by radicals, computation of Galois groups (Ch 14)
- **commutative-algebra**: Noetherian rings, localization, primary decomposition, Artinian rings, Dedekind domains (Ch 15-16)
- **algebraic-geometry**: Affine algebraic sets, varieties, coordinate rings, Nullstellensatz, prime spectrum (Ch 15)
- **homological-algebra**: Ext, Tor, group cohomology, crossed homomorphisms, group extensions (Ch 17)
- **representation-theory**: Group representations, characters, Wedderburn's theorem, induced characters (Ch 18-19)
- **category-theory**: Categories, functors, natural transformations, equivalence of categories (Appendix II)

## Tiers

- **foundational**: No prerequisites within this source — basic definitions and axioms
  - Ch 0 (Preliminaries), Ch 1.1 (group axioms), Ch 7.1 (ring axioms), Appendix I
- **intermediate**: Requires foundational concepts
  - Ch 1-3, Ch 7-9, Ch 10-11, Ch 13
- **advanced**: Requires intermediate concepts
  - Ch 4-6, Ch 12, Ch 14, Ch 15-19, Appendix II

## Notation Conventions

### Groups
- Group operation: multiplicative by default, additive for abelian groups
- Subgroup: H ≤ G
- Normal subgroup: N ⊴ G
- Index: |G : H|
- Order: |G|, |x|
- Cyclic group: Z_n
- Dihedral group: D_{2n}
- Symmetric group: S_n
- Alternating group: A_n

### Rings
- Units: R×
- Polynomial ring: R[x]
- Ideal generated by elements: (a, b)
- Quotient ring: R/I

### Modules & Linear Algebra
- Hom: Hom_R(A, B)
- Tensor product: M ⊗_R N
- Direct sum: A ⊕ B

### Field Theory
- Extension: K/F
- Degree: [K : F]
- Galois group: Gal(K/F)
- Minimal polynomial: m_{α,F}(x)

### Algebraic Geometry
- Affine space: A^n
- Zero set: Z(I)
- Ideal of set: I(A)
- Coordinate ring: k[V]
- Spectrum: Spec R
