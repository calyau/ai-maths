# Competency Questions for Diestel's "Graph Theory" (3rd ed.)

## Definitional (What is X?)

2. What is a vertex and an edge?
3. What is a subgraph? An induced subgraph?
4. What is a path? A cycle?
5. What is a tree? A forest?
6. What is a bipartite graph?
7. What is a matching?
8. What is a vertex cover?
9. What is a planar graph?
10. What is a graph colouring?
11. What is the chromatic number of a graph?
12. What is a flow in a network?
13. What is a Hamilton cycle?
14. What is a random graph?
15. What is tree-width?
16. What is a graph minor?
17. What is a well-quasi-ordering?
18. What is an end of a graph?

## Relational (How does X relate to Y?)
1. How does connectivity relate to the existence of disjoint paths (Menger's theorem)?
2. How do matchings relate to vertex covers (König's theorem)?
3. How does planarity relate to forbidden minors (Kuratowski's theorem)?
4. How does the chromatic number relate to the clique number?
5. How do flows relate to colourings (flow-colouring duality)?
6. How does the max-flow relate to the min-cut?
7. How does edge density relate to the existence of subgraphs (Turán's theorem)?
8. How does tree-width relate to forbidden minors?
9. How do Ramsey numbers relate to graph colouring?
10. How does the degree sequence relate to Hamiltonicity?

9. How do I find an Euler tour in a graph?
