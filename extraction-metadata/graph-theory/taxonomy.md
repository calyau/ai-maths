# Domain Taxonomy for Diestel's "Graph Theory"

## Categories

- **graph-fundamentals**: Basic definitions of graphs, vertices, edges, degree, isomorphism
- **paths-and-cycles**: Paths, walks, cycles, connectivity, distance
- **trees-and-forests**: Trees, forests, spanning trees, arboricity
- **matching-and-covering**: Matchings, vertex covers, packings, factors
- **connectivity**: k-connectivity, separators, Menger's theorem, linkedness
- **planar-graphs**: Planarity, embeddings, Kuratowski's theorem, duality
- **graph-colouring**: Vertex/edge colouring, chromatic number, list colouring, perfect graphs
- **network-flows**: Flows, circulations, cuts, max-flow min-cut
- **extremal-graph-theory**: Turán-type problems, density, regularity lemma
- **infinite-graphs**: Rays, ends, compactness, infinite connectivity
- **ramsey-theory**: Ramsey numbers, monochromatic subgraphs, partition properties
- **hamiltonian-graph-theory**: Hamilton cycles/paths, degree conditions, toughness
- **random-graphs**: Random graph models, probabilistic method, threshold functions
- **graph-minors**: Minors, tree-decomposition, tree-width, well-quasi-ordering
- **algebraic-graph-theory**: Cycle space, incidence matrix, linear algebra methods
- **topological-graph-theory**: Surfaces, embeddings, genus, Euler characteristic
- **set-theory-foundations**: Cardinality, ordinals, well-ordering, Zorn's lemma

- **foundational**: Ch 1 (The Basics), Appendix A (Infinite Sets) — core definitions needed by all later chapters
