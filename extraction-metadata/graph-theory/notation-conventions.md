# Notation Conventions for Diestel's "Graph Theory"

## Graphs
- G = (V, E) for a graph with vertex set V and edge set E
- V(G) for vertex set, E(G) for edge set
- |G| for order (number of vertices), ||G|| for size (number of edges)
- e = xy for edge between vertices x and y

## Degree
- d(v) or d_G(v) for degree of vertex v
- delta(G) for minimum degree, Delta(G) for maximum degree
- d(G) for average degree, epsilon(G) for edge density ||G||/|G|

## Special Graphs
- K^n for complete graph on n vertices
- K_{m,n} for complete bipartite graph
- C^n for cycle on n vertices
- P^n for path on n vertices
- G^2 for the square of graph G
- G* for the dual graph

## Connectivity
- kappa(G) for vertex connectivity
- lambda(G) for edge connectivity
- k-connected means kappa(G) >= k

## Colouring
- chi(G) for chromatic number
- chi'(G) for chromatic index (edge chromatic number)
- ch(G) for list chromatic number (choosability)
- omega(G) for clique number
- alpha(G) for independence number

## Sets and Operations
- G - X for vertex deletion
- G - F for edge deletion
- G / e for edge contraction
- G[X] for induced subgraph on vertex set X
- N(v) for neighborhood of vertex v

## Flows
- f: E -> group for a flow function
- cap: E -> R for capacity function

## Infinite Graphs
- aleph_0 for countable infinity
- omega for the first infinite ordinal
